Hyperlinking to a URL

  1. Create an Analyzer report or open an existing one.
  1. Right-click a row label or column header and select Hyperlink. The Link on dialog box appears.
  1. Click Enable Link to activate the hyperlink feature. You can disable linking by clearing the Enable Link checkbox.
  1. In the Link To drop-down menu, select URL.
  1. In the URL field, enter the full web address you want the hyperlink to launch. For example, http://www.yahoo.com.
  1. Choose how the URL is displayed by clicking on the Open In drop-down menu and selecting New Tab, New Window, or Current Window.
  1. Enter a Tool Tip to be displayed when you hover over hyperlinks and click OK. The new hyperlinks appear in the Analyzer report.
  1. Click the links to ensure the website associated with them appears correctly and save the report.
