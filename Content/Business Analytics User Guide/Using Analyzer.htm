Using Analyzer

Analyzer has an easy-to-use, web-based, drag-and-drop design environment which can be used by anyone who wants to dynamically explore data and drill down to discover previously hidden details. You do not need any special expertise to use Analyzer. You can display Analyzer reports in a dashboard dashboard in the Report Editor.

Before you can use Analyzer, you must have access to a data source. The data source model enables you to choose which dimensions and measures you want to explore in your data.
