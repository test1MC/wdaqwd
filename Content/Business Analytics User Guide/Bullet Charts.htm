Bullet Charts

Bullet charts display data in a linear fashion, allowing for a high-level overview of selected data. The larger the value being displayed, the longer the line. Colored thresholds can be set to quickly identify data of interest. An individual bullet chart is created for each row in the data table, allowing for multiple charts to be displayed in the same report. Data displayed by this type of chart cannot be drilled down into.

Optional Properties for Bullet Charts

Definition

Thresholds

Defines the color of the chart values. Values equal to or less than the first number will be displayed using the first color. Values equal to or greater than the last number will be displayed using the last color. Values in between the two numbers will be displayed using the middle color.
