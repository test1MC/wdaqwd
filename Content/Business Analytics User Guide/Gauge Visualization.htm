Gauge Visualization

Gauge visualization display the data in a semi-circle with larger values filling up more of the circle (similar to a speedometer). This type of chart is well suited to displaying data that represents the performance of a system. Colored thresholds can be set to quickly identify data of interest. An individual gauge visualization is created for each row in the data table, allowing for multiple visualizations to be displayed in the same report. Data displayed by this type of visualization cannot be drilled down into.
