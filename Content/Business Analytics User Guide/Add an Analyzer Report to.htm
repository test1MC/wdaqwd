Add an Analyzer Report to a Dashboard

Use these steps to display an Analyzer report in a dashboard.

  1. Select a panel in the Dashboard Designer.
  1. Click the Insert Content icon and choose File.
  1. Locate the appropriate Analyzer report and click Select. The Analyzer Report appears inside the dashboard panel.
