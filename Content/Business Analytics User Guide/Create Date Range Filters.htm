Create Date Range Filters

Date range filters enable you to show only data that meets the conditions of the filter in a report. For example, you can create filters to display data between 2013 and 2016, or all data after 2013. After you have applied your filter, the Analyzer report shows only data specified by the date ranges that you selected.

  1. Log in to the Report Editor, then open an existing analysis report that contains a time dimension or choose Create New > Analysis Report.
  1. For a new report, select the data source that you want to use and click OK.
  1. Create the report with a date dimension, such as year, month, or day.
  1. Click the icon to Add A Filter, then drag the time dimension that you want to filter on to the Filters board. You can also right-click on the dimension and add it as a filter.
  1. Choose one of these radio buttons to create your filter:
  • Use Choose a commonly used time period to specify a time period; Filter on Quarters is used here as an example. Once you have your time period specified, click OK to apply the filter. This filter is dynamic and changes with the current date. The other date filters are static.

Figure 3‑16: Filter Dialog

  • Select from a list lets you choose values from a list. You can use the single right- or left- arrows to add or subtract one value at a time, or use the double right- or left- arrows to add or subtract all of the values in the list. Click OK to apply the filter to your report.
  • If you choose Select a range, then click the Select from date picker link, you will be able to use the date picker calendars to select beginning and ending dates for your report data. Click Apply, then OK, to apply the filter to your report. Note that there may be duplicates in the list unless a filter has been applied.

Figure 3‑17: Filter Dialog - Select From a List

  • Once you select a date, the data is validated to make sure that your date ranges actually contain that date. Analyzer searches for the nearest time period, up to plus or minus 30 time periods, if no date is found the first time. You will not be able to save the date filter if no dates are found. Note that date pickers are limited to the date field selected. For example, if on a month filter you select 2015-01-10 -> 2017-04-09, you only get a filter for January to April (year and day are not included).

Figure 3‑18: Filter Dialog - Select a Range Date Picker

Note: When you create a date range filter using the Between (and incl.) operator and parameterize it, as described in Add Query Parameters in Analyzer Reports (section 3.1.4.3 Add Query Parameters to Analyzer Reports ), you specify one parameter name, but two parameters are created. One parameter controls the start of the range, and another controls the end of the range. The start date parameter is<YourParameterName>_START, and the end date parameter is <YourParameterName>_END.
