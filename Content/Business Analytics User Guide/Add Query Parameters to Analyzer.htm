Add Query Parameters to Analyzer Reports

You must be logged in to the Report Editor. Open the Analyzer report you created in 3.1.4 Add Filters to an Analyzer Report.

You can parameterize a query in Analyzer.

  1. Drag and drop the dimension member you want to create a parameter for onto the Filters canvas.
  1. Select the level you want to set as the default parameter value, then click the right arrow to move it to the list on the right.
  1. Select the check box at the bottom of the window, then enter a name for the parameter in the Parameter Name field.
  1. Click OK.

Your parameter is a filter in Analyzer. Whenever this Analyzer report is run, users will have a selection of columns to filter by.
