Number Charts

Number charts display the column title and the data in a user-configurable text format, allowing for a high-level overview of selected data. Colored thresholds can be set to quickly identify data of interest. An individual number chart is created for each row in the data table, allowing for multiple charts to be displayed in the same report. Data displayed by this type of chart cannot be drilled down into.

Optional Properties for Number Charts

Property

Definition

Available In:

Thresholds

Defines the color of the chart values. Values equal to or less than the first number will be displayed using the first color. Values equal to or greater than the last number will be displayed using the last color. Values in between the two numbers will be displayed using the middle color.

Number chart

Charts Per Row

Defines how many charts are displayed on each row in the report.

Number chart

Orientation

Sets the location of the values in relation to the column headers.

Number chart

Format

Allows you to set the text format used to display the chart values.

Number chart

Title/Value Font Ratio

Adjusts the size of the Value font in relation to the Title’s font size.

Number chart
