Using Dashboard Designer

Creating a dashboard in Dashboard Designer is as simple as choosing a layout template, theme, and the content you want to display.

Dashboard Designer has dynamic filter controls, which enable dashboard viewers to change a dashboard's details by choosing different values from a drop-down list, and to control the content in one dashboard panel by changing the options in another.
