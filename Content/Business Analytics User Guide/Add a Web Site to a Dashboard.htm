Add a Website to a Dashboard

Use these steps to display contents of a website in a dashboard panel.

  1. Select a panel in the Dashboard Designer.
  1. Click the Insert icon () and choose URL. The Enter Web site dialog box appears.
  1. Enter the website URL in the text box and click OK.
  1. If applicable, click the Edit icon () to make changes.
  1. Save your dashboard when you are done.
