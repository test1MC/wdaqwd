Using Dynamic Dashboard Titles

While in Dashboard, each panel has the ability to show a title with current parameters.

  1. Click the Edit () icon in the top toolbar. The Edit pane appears at the bottom of the screen
  1. Click the report that has the parameters you are going to change. You will need to go through this process for each report to which you want to add parameters.
  1. Click the Add Parameter to title button. The placed parameter appears after the title.
  1. Click Apply.
  1. Close the Edit pane by clicking the Edit (pencil) icon. The title will now show the parameters applied to it.
