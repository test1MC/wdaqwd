Home

The first thing you see after you log in to the console is the Home page, which serves as a starting place for the Report Editor and all of the tasks you do with it.

Figure 2‑2: Home Screen

Item

Name

Function

Home Indicator

Indicates the current Report Editor perspective that you are using. The Home menu lets you flip easily from page to page, or return to your Home page.

Current User

Shows the name of the person currently logged in to the Report Editor. Clicking the arrow next to the name lets you log out of the Report Editor.

Browse Files

Brings you to the Browse Files window, where you can locate your files using the Browsing and Files panes, and manage them using the Actions pane. See section 2.2.3 Browse Files .

Create New

Gives you the option to create a new Analyzer Report or Dashboard.

Documentation

Opens the SmartSuite Business Analytics user guide.

Recents

Shows a list of your most recently opened files. Clicking on the star next to a recently opened file adds it to your Favorites list.


Favorites

Shows a list of your favorite files for quick access.
