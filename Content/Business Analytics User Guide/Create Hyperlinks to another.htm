Create Hyperlinks to Another Report

  1. Create an Analyzer report or open an existing one.
  1. Right-click a row label or column header and select Hyperlink. The Link on dialog box appears.
  1. Click Enable Link to activate the hyperlink feature. You can disable linking by clearing the Enable Link checkbox.
  1. In the Link To drop-down menu, select File.
  1. Click Browse to locate a report, chart, or dashboard and click Open.
  • If the destination report has parameters, they automatically appear in the Destination Parameter list on the left. Map parameters to related row labels or column headers by selecting the checkbox for each parameter you want to use to constrain the resulting data. Enter the related names of the row labels or column headers within curly brackets.
  • If the destination report does not have parameters, the Destination Parameter list does not appear. Go to the next step.
  1. Specify how hyperlink content displays by clicking the Open in: drop-down menu and selecting New Tab, New Window, or Current Window.
  1. Enter a Tool Tip to be displayed when you hover over hyperlinks and click OK. Hyperlinks appear in the Analyzer report.
  1. Click the links to ensure the content associated with them appears correctly and save the report.
