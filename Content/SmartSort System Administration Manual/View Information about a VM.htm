View Information about a VM

To view information about a particular VM:

  1. On the leftmost navigation panel, click Hosts and Clusters to view VM information. A list of VMs display in the navigation bar.
  1. Click the VM name in the navigation bar. That VM's information displays.
  1. From this screen, the user can view all information about the VM, including recent tasks, alarms, and work in progress.
