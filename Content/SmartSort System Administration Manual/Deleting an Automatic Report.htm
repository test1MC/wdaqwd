Deleting an Automatic Report Job

Occasionally, Automatic Report Jobs need to be deleted.

To delete an existing Automatic Report Job:

  1. Select the checkbox for the appropriate job(s) and then click Delete Selected Report Jobs.
  1. A confirmation dialog displays.
  1. Click OK to delete the selected job(s). Click Cancel to abort the deletion.
