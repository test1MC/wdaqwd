Bag Records

The SAC stores bag records that contain the current status of all bags in the system, as well as bags that have already been processed by the BHS. The completeness of this data depends on what information is provided in the BSM and what happens to the bag in the BHS. This data may include:

  • Bag Tag Number
  • Flight Information (for Outbound and Inbound Flights)
  • Carrier
  • Flight Number
  • Flight Date
  • Destination
  • Passenger Class
  • Passenger Name
  • Transfer Type (Online, Interline, None)
  • Authorization to Load Status (Yes, No)
  • Standby Status (Yes, No)
  • Selectee Status (Yes, No)
  • Security Status and Level
  • Corporate or Group Name
  • Automated PNR Address
  • Frequent Traveler Number
  • Baggage Pieces and Weight Data
  • Tag Printer ID
  • Scanned Location and Timestamp
  • Sorted Location and Timestamp
  • Sort Method (Early Makeup unit, On Time Makeup unit, Late Makeup unit, Missed Makeup unit, etc.)
  • Reject Reason (Conflicting Barcodes, Flight not in Flight Schedule, etc.)
  • Sorted Makeup unit (Logical and Actual)
  • MEC Location and Timestamp
  • Security Information (Screening Machine, Timestamp, Security ID, and Status)
  • Recirculation Count
