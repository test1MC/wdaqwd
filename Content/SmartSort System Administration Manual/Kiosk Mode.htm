Kiosk Mode

Certain workstations can be configured as Kiosk displays, allowing users to access and view specific SmartSuite Enterprise Web Client screens without requiring user authentication. By default, the Kiosk Mode profile features the following four screens, though more can be included as required by the system administrator:

  • Activity Log
  • BHS Map
  • Bags En Route Dashboard
  • Active Flight Scheduler

These screens can be grouped together to create kiosk profiles; workstations can then be configured to only access certain profiles, thereby providing system administrators the ability to restrict and control Web Client access across the system.
