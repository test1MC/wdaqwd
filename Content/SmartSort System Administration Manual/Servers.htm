Servers

The Domain Administrator log on is primarily used for access, either locally or remotely.

SQL Server is also running on the SAC and DHS servers. This application is configured to run with mixed–mode authentication to allow both Windows and SQL users to connect. By default, all SmartSort services and application accesses use an SQL server user name. Technical support personnel connect to SQL using Integrated Security.

Remote Desktop Connections are configured in addition to Tight VNC so that remote connections can be made to workstations for troubleshooting purposes.
