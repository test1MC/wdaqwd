Automatic Reports Service

The Automatic Reports Service creates reports on a scheduled basis by printing them to a system printer or saving them to the file system. This service can be configured through the Automatic Reports setup screen within the Web Client application described in section (see 2.3.2 Generating Automatic Reports). New jobs can be added with any report customization required using this screen.

The service polls for new report jobs every few minutes, so new information can be changed on the fly. Different report jobs can be configured on the Web Client and can be set to output to saved files, e-mailed to individuals, or printed. Reports are available in PDF, Excel, or CSV formats.

The Automatic Reports service needs to run to generate the system reports expected by maintenance personnel.
