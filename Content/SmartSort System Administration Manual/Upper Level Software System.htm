Upper Level Software System

At the heart of the SmartSort BHS software are the databases that store all the configuration, sortation, and historical information. The primary database is on the SAC server and its data is copied to the DHS for storage. The databases are designed to be self-sufficient. No user interaction with tables or indexes should be necessary. Data is cleaned up and backups are taken on an automatic basis as described in this document.
