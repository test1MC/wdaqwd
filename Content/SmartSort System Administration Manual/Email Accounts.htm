Email Accounts

Email Groups allows multiple Email Recipients to be added to functional groups (like email distribution lists). The Email Accounts setup screen is used to establish these groups. Automatically generated reports can be emailed to configured groups (see section 2.3.2 Generating Automatic Reports for details). Additionally, email groups can be notified when certain alarms are detected (see section 2.1.5 Alarm Distribution Configuration for details).

The Email Groups screen displays a list of all existing email groups, and can be used to view members of this group. Email groups can be added, modified, and deleted.

Figure 2‑3: Email Group Membership

To add a new email group:

  1. Click Add Group. The Edit Email Group dialog displays.
  1. Enter a group name in the Description field.
  1. Click Add. A confirmation dialog displays.
  • Click OK to add the new group. Click Cancel to abort the email group creation.

To edit an existing email group:

  1. Select the desired Email Group from the Group List filter, then click Edit Group. The Edit Email Group dialog displays.
  1. Edit the name in the Description field.
  1. Click Save.

To delete an existing email group:

  1. Select the desired Email Group from the Group List filter, then click Edit Group. The Edit Email Group dialog displays.
  1. Click Delete. A confirmation dialog displays.
  • Click OK to delete the selected group. Click Cancel to abort the deletion.
