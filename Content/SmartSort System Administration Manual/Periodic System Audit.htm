Periodic System Audit

Periodically, computer hard drives should be checked for free space. The HMI indicates when free space available is less than 20% on the System servers by activating a Resources Alarm. Check that the SmartSort log file archiving is working correctly. The system archives files in a sustainable manner subject to various retention periods. Review Operating System event logs for potential problems. Determine if updates to the system are required (windows updates, antivirus updates).
