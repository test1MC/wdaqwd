Editing an Automatic Report Job

Clicking the Edit icon in the leftmost column on the Automatic Reports screen displays the Automatic Reports Edit screen. This screen is identical to the one used to create new Automatic Report Jobs, except that the appropriate fields and checkboxes will already be filled out and selected.

To edit an existing Automatic Report Job:

  1. Click the Edit icon () for the desired Automatic Report Job. The Automatic Reports Edit screen displays.
  1. Use the fields and checkboxes on the Automatic Reports Edit screen to change the job settings as necessary (e.g. add/remove reports from the job, change output settings, change report frequency, etc.).
  1. Click Save Report Job Changes to save the changes to the Automatic Report Job. Clicking Return to Report Jobs without clicking Save Report Job Changes returns to the Automatic Reports screen without changing the job's settings.
