Virtual Makeup Units

A Virtual Makeup unit is the virtual location to which a bag will be sorted. Virtual makeup units are mapped to a physical makeup unit (the bag's true destination) using the Virtual to Physical Mapping screen in the Web Client. The Virtual Makeup Units screen allows the operator to administer the various virtual makeup units that will be used in the system and accessible on the Virtual to Physical Mapping screen.

Virtual Makeup units can be added, edited, or deleted using the Virtual Makeup Units screen. They can also be configured so that an associated physical makeup unit destination is required. If so configured, the user will not be able to leave the virtual makeup unit unassigned on the Virtual to Physical Mapping screen.

Figure 2‑21: Virtual Makeup Units Screen

To add a new virtual makeup unit:

  1. Click Add New Virtual Makeup Unit. The Virtual Makeup Unit Details section on the right side of the screen is populated with blank fields, drop-down lists, and checkboxes.
  1. Using the fields, drop-down lists, and checkboxes in the Virtual Makeup Unit Details section, enter the appropriate Virtual Makeup unit information (the zone, makeup unit number, category, and description).
  1. If the virtual makeup unit must be mapped to a physical makeup unit at all times (i.e. cannot be unassigned), select the Must be Mapped to Physical Makeup Unit checkbox.
  1. The Alias information is used to identify the virtual makeup units for different host systems (if applicable).
  1. Click Add New at the bottom of the screen to save the new Virtual Makeup unit. Clicking Cancel cancels the Virtual Makeup unit creation.

To edit an existing virtual makeup unit:

  1. Select the desired Virtual Makeup unit from the Virtual Makeup Unit List. Note that this list can be filtered by typing in the Filter field at the top of the list. Clicking the Virtual Makeup unit in the list causes that Virtual Makeup unit's details to be displayed in the Virtual Makeup Unit Details section.
  1. Using the fields in the Virtual Makeup Unit Details section, edit the Virtual Makeup unit's settings as required.
  1. Click Save Changes to save the changes to the Virtual Makeup unit. A confirmation dialog is displayed.
  1. Click OK to save the changes. Clicking Cancel aborts the save.

To delete an existing virtual makeup unit:

  1. Click the Delete checkbox(es) for the appropriate Virtual Makeup Unit(s) in the Virtual Makeup Unit List.
  1. Click Delete Selected Virtual Makeup Units at the top of the screen. A confirmation dialog is displayed.
  1. Click OK to delete the Virtual Makeup unit(s). Clicking Cancel aborts the deletion.
