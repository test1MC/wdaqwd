SmartVision HMI Servers

There are two HMI servers active in the system running the FactoryTalk View SE platform. These virtual machines (HMI01 and HMI02) provide the graphics and alarm capabilities for the HMI clients described in SmartSort User Guide. The VMs use RSLinx Enterprise software for communications with the system PLCs.

The pair of HMI servers function as a redundant set in an Active-Standby fashion. The Active server in the set performs all functions unless or until it fails, at which point the secondary server takes over. The status of the HMI servers is shown on the HMI Connection Status screen. If a server is rebooted or when a failed server is brought back online, it may cause a brief interruption in graphics or alarms as the FactoryTalk system resynchronizes. Most HMI clients also see the server status pop-up as a warning that the server state is changing.
