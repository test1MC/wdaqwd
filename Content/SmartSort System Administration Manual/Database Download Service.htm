Database Download Service

The Database Download Service is used to periodically transfer information from the SAC database to the PLC. The service queries the current database data and writes the results to specified arrays in the PLC. This ensures that the PLC data is kept periodically up to date in case of a server failure.

Two groups of information are transferred by this service: the current virtual to physical mappings and the Carrier default makeup units. These two arrays are used for backup sortation by the PLC in the unlikely event that the SAM service is not responding to scanner messages.

The Database Download Service needs to remain running to keep the PLC information up to date after changes are made through the Web Client Application. Without this service running, the latest information might not be available for backup sortation.
