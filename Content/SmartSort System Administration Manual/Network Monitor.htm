Network Monitor

The purpose of this service is to drive the Network Status Report in the Web Client. It periodically pings devices on the network to ensure that they are reachable and to keep the report up to date. Certain devices in the report are associated with HMI alarms (those that don't otherwise have an alarm) that this service will trigger if the device is not network reachable.

This service should remain running to keep the HMI indicators and Network Status report up to date.
