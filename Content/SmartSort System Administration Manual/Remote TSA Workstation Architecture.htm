Remote TSA Workstation Architecture

Five TSA Remote TSA Workstations are configured to run the HMI application and SmartSuite Web Client.

Each of the two new workstations is configured as follows:

  • Contec BX-S959SD
  • Atom E3845 Processor
  • 4 GB RAM
  • Dual Video Adapter
  • One 55" NEC C551 Display
  • Windows 10 IOT
  • Adobe Reader
  • FactoryTalk View SE Client
