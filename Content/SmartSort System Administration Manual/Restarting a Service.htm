Restarting a Service

In the event of a problem, SmartSort services can be restarted from the SmartSort Service Manager shown above which is accessible from the Quick Launch bar or in the Windows Control Panel.

  1. Log into the appropriate Server (SAC or DHS) and open the service control application from the desktop or taskbar shortcut.
  1. Select the Service to be restarted and right-click on it. Multiple services can be selected if desired.
  1. Click Stop. The service is given a Stop command and begin its shutdown procedure.
  1. Wait for the Status to change to Stopped.
  1. Right-click on the service again and click Start.
  1. Wait for the Status to change to Running.
  1. Use the Restart menu option as an alternative.

If the service does not stop in a reasonable time (30 seconds) when given the Stop command, it can be killed via the context menu and then started again.

The HMI indicates that a particular service has stopped functioning correctly by a Heartbeat alarm and Comm Screen indicator. In this case, an administrator should log into the appropriate server and restart it if necessary.
