Network Attached Storage (NAS)

A Network Attached Storage device (NAS) is located in the server rack and provided to complement the Maintenance Server. This device allows files to be shared with all computers on the BHS network. Files ware stored on the NAS to avoid filling up the server hard drives.  Examples of these files include but are not limited to commonly-used files, database backups (will also store on local VM drives), PLC programs and software installations and computer backups.
