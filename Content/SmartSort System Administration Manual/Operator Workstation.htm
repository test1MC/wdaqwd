Operator Workstation

Operator workstations are identical in configuration but work independently of each other. The SmartVision HMI will be installed on these Workstations. The workstations will also be able to access the Web Client.

Each workstation will be connected to a set of 2 displays with a resolution of 1920x1080. One screen shows the SmartVision HMI, the other will show either a second SmartVision HMI screen or the Web Client application. The Operator can navigate to any screen within the SmartVision HMI application.
