System Hardware Administration

IT Infrastructure

The general architecture of the system is described in the sections that follow. For more detail, including all network connections, please refer to the Network Drawings provided with this project. Screenshots are for illustrative purposes. Some screens may vary from the actual installation on site.

The basic architecture of the system is to have virtual machines for all server computers that run the BHS. These servers are hosted by two physical machines located in the server rack. All physical disk storage for these VMs is provided by a Storage Area Network (SAN), also in the server rack. Redundancy is provided by the VMware vSphere software.
