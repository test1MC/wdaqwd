Server Hardware

Each of the physical host servers are comprised of hardware and off-the-shelf software components as follows:

  • Dell R640
  • 128 GB RDIMM RAM
  • CompactFlash Local Storage (no on-board drives)
  • Network Interfaces
  • VMware ESXi 6.5 Operating System

All servers and other components in the server rack are connected to one of two UPSs located in an adjacent rack. The equipment is thus protected from brief power outages, surges, and other power anomalies.

In addition to the hardware components already mentioned, the SAC and DHS systems also include the following off-the-shelf software:

  • Microsoft SQL Server 2016
  • Kepware KepServerEx (only on the DHS)
  • Carbon Black Endpoint Protection
  • TrendMicro Anti-Virus (Manage and Provided by AA)
  • Adobe Reader
  • Ultra VNC

After the end of the commissioning/on-site support period, no operating system, driver, or off-the-shelf software updates will be performed. During the warranty period Brock Solutions will provide recommendations on such patches or updates as they become available.
