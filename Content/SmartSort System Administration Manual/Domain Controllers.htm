Domain Controllers

Two Domain Controllers (DCs) run as virtual servers on VMware physical hosts. These machines provide network services and centralized authentication for all computers on the BHS network. There is no explicit failover between servers for the Domain Controllers, but at least one of them must be running to keep the system operational.

vCenter requires a domain connection to operate, so it may be necessary to start up a domain controller before bringing the vCenter VM online.
