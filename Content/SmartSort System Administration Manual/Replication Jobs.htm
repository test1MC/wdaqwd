Replication Jobs

SQL Server replication is described in section 3.3.9 Replication. The jobs listed below are set up automatically when replication is configured and should not be changed except through the tools described in section the 3.3.9 Replication topic.

  • Agent History Cleanup
  • Distribution Cleanup
  • Expired Subscription Cleanup
  • Reinitialize Failed Subscriptions
  • Replication Agents Cleanup
  • Replication Monitoring
