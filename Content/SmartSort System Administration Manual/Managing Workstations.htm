Managing Workstations

Workstations are created and defined using the Workstations setup screen in the Web Client, accessed via the Settings menu. When this screen is loaded, all existing workstations in the BHS are displayed.

Figure 2‑5: Workstation Setup Screen

The Workstations setup screen allows an operator to do the following:

  • Add a new workstation
  • Edit an existing workstation's details
  • Delete a selected workstation

This screen displays the workstation name, group membership, and defined function/location (e.g. MEC-01) of each workstation in the system. Operators can also define the workstation's location details from this screen. Note that the list of workstations can be filtered by Name, Group, or Type.

To add a new workstation:

  1. Click Add New Workstation at the top of the screen. The empty Workstation fields are displayed in the Workstation Details section.
  1. Use the fields and drop-down lists in the Workstation Details section to configure the workstation as appropriate.
  • Certain parameters will be preselected based on the facility that has been chosen.
  1. Click Add New to save the new workstation.
  • Clicking Cancel or navigating away from the page cancels any unsaved changes.

To edit an existing workstation:

  1. Click the name of the desired workstation from the Workstation List. The group's details display in the Workstation Details section of the screen.
  1. Use the fields and drop-down lists to change the workstation's settings as necessary.
  • Note that the Name field cannot be edited.
  1. Click Save Changes to save the changes to the workstation. A confirmation dialog displays.
  • Click OK to save the changes. Click Cancel to abort saving the changes.

To delete an existing workstation:

  1. Select the checkbox(es) in the Delete column next to the name(s) of the desired workstation in the Workstations List.
  1. Click Delete Selected Workstations. A confirmation dialog displays.
  • Click OK to delete the selected workstation(s). Click Cancel to abort the deletion.

Swapping two computers in the field is as simple as changing the location for both workstations on this portion of the screen. Spares can be created using new workstation names. Linking the spare workstation to the location that has failed is enough for the new machine to take over when it boots up.
