Colors Not Appearing on HMI (Wireframe)

If this occurs on all HMI workstations, the OPC Server isn't communicating with the PLC controlling that section:

  1. Confirm that the PLC communication is OK by pinging the PLC Ethernet card.
  1. Contact Brock Solutions Support to restart the components on the main HMI server to try to restore connectivity.

If this only occurs on certain HMI workstations, individual HMIs may need to be restarted:

  • Restart individual HMI workstation.
