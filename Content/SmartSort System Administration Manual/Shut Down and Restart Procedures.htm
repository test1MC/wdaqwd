Shut-Down and Restart Procedures

The computer hardware used in the BHS requires some special procedures for shutting down and restarting. In all cases, the power button can be used as a last resort to shut off the system, but the following steps should be tried first.

Restarting computers should not be a normal occurrence in the BHS. The system has been designed to run without restarting for long periods of time. Given the potential for instability inherent in any operating system, it is possible that some computers (especially workstations) may require periodic restarting.

All computers (except the Manual Encodes) are protected by UPS power. If possible, all computers should be shut down gracefully rather than by abruptly cutting the power to the machine.
