Workstations

All client computers use Windows 10 or Windows 10 IoT (for field devices) as their operating system.

A general windows user account has been configured on the domain for all windows client machines. This account has credentials set up restricting access to features that could harm the workstation/client health or applications running on it. The type of computer determines which user is configured: workstations, remote stations, or FormsClient computers.

The users specified in the topics listed belowthis section of the document refer to Windows users. Application level security on the operator workstations for the HMI and Web Client are also provided as described in the SmartSort User Guide.
