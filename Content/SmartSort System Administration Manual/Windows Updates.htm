Windows Updates

The system is generally considered to be in a static environment with restricted access to the outside world. Therefore, updates to the system are only to be performed when critical to the operation.

Occasionally, Windows security updates (kernel updates) have been known to interfere with third-party software (industrial component drivers). Check third-party vendors for warning bulletins.
