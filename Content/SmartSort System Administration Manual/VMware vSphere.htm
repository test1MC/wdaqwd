VMware vSphere

Note: VMware vSphere is available as an installed client and as a web application. Brock Solutions recommends using the web application.

VMware vCenter is a centralized management web application that allows users to manage virtual machines and hosts. vCenter itself is actually a virtual machine (appliance) running on the same physical hosts as the BHS virtual servers. A vSphere client is used to access vCenter and ultimately manage servers.

vCenter allows users to view VM status, resources, performance, alarms, etc., as well as perform tasks such as view and clear alarms, power cycle VMs, perform maintenance, and migrate VMs between hosts.
