Restore a VMware Server

The VMware servers are backed up to the NAS using exported OVF templates. An OVF template is a full VM that has been archived into a compressed file.

Warning: Errors in the restore process could cause significant damage to the system. Do not attempt to restore a VMware server without first consulting with Brock Solutions.

To restore a VMware Server:

  1. In vSphere, click File > Deploy OVF Template…
  1. Follow the instructions displayed in the wizard to browse to the location of the backed-up OVF template file, then click Next.
  1. Review the summary and EULA pages. Click Next and Accept to proceed.
  1. Define the location and name for the VM.
  1. Select the appropriate ESX server and Datastore to host the VM, and then click Next.
  1. Define the Network Mapping.
  1. Review the configuration and click Finish to start the import. When complete, the VM will be restored to the state it was in at the most recent backup.
