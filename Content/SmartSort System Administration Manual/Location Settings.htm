Location Settings

The Location Settings screen allows the user to define location settings for workstations, such as the MEC stations. Note that all locations are added by Brock Solutions Administrators when the system is initially configured and installed.

The location settings available for MECs include:

  • Hot Makeup Unit Keys – The Hot Makeup Unit Keys that are displayed on the MECs are configured here. The text for each hot key can be customized and the associated virtual makeup unit can be selected from the drop-down list.
  • Show Flights Minutes Before Departure – When encoding by flight, the list of flights that are displayed is limited by the value entered here and in the Show Flights Minutes Since Departure field. For example, if 60 is entered, flights will only be displayed if they are departing in 60 minutes or less.
  • Show Flights Minutes After Departure – When encoding by flight, the list of flights that are displayed is limited by the value entered here and in the Show Flights Minutes Before Departure field. For example, if 30 is entered, departed flights will only be displayed if their departure time was 30 minutes ago or less.
  • Show Carrier Default Pop-up – If a bag tag is scanned or manually keyed and a BSM does not exist for that particular bag tag, the system can be configured to automatically release it to the Carrier Default Makeup unit for the bag, or the system can be configured to prompt the operator before automatically releasing the bag to the Carrier Default makeup unit. When this checkbox is selected, a dialog is displayed to the user with an option to sort the bag to the Carrier's Default Sort Makeup unit. Note: If Pseudo BSMs are enabled, the Carrier Default Operator Prompt is disabled. In this case, bags are automatically released to the Carrier's Default makeup unit if a Default Makeup unit has been configured.
  • Automatically Print Retag Labels – When this is enabled, when a bag needs to be retagged at the Manual Encode the print label will automatically print without the user having to press the print button. 
  • Enable Pseudo BSM Generation – If a bag tag is scanned or manually keyed and a BSM does not exist for that particular bag tag, the system is capable of creating a pseudo BSM (a BSM created by SmartSort rather than one created by the Airport/Carrier). The purpose of creating this Pseudo BSM is to provide traceability for a particular bag to a sort destination. In addition, if a bag is subsequently seen at an ATR, it can be automatically sorted based on the Pseudo BSM instead of returning to MEC again.
  • Enable Pseudo BSM Strict Mode – There are two Pseudo BSMs modes: normal mode and strict mode. In Strict mode, the operator can only encode the bag by Flight or by selecting a Hot Makeup unit. In Normal mode, the operator is prompted to encode the bag by flight, but is able to select another encoding method (Makeup unit, Carrier Default Makeup unit, or Hot Makeup unit).
