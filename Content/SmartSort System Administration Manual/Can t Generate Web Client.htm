Can't Generate Web Client Reports

Report or search had too large a time span or criteria was too vague:

  • The operation timed out.
  • Reduce the scope of your query or report and try it again.

A server error has occurred:

A message of this nature appears on the screen after a report has been requested.

  • Try the same request from a new browser session or from a different computer.
