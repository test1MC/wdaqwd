Large Screen Display Architecture

Two Large Screen Display Workstations are configured to run the HMI application on four large screen displays.

The HMI application is installed on these computers.

Each of the two new workstations is configured as follows:

  • Dell Precision Tower 3420
  • Intel i5-6500 3.6 GHz processor
  • 8 GB of RAM
  • DVD+/-RW Drive
  • Dual Video Adapter
  • Four 55" NEC C551 Displays
  • Windows 10 Professional (64-bit)
  • Adobe Reader
  • FactoryTalk View SE Client
