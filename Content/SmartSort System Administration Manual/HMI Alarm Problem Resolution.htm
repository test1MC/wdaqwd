2.4.11 HMI Alarm Problem Resolution

Problem Resolution messages are accessed from the Alarm Summary screen or from the Alarm Banner in the HMI, and display information about resolving alarms.

The HMI Alarm Problem Resolutions screen is accessed via the Settings menu in the Web Client and allows the user to configure HMI Alarm Problem Resolution Details. For each type of alarm, the resolution text should include a Problem Description, a Primary Action, and Additional Actions.

Problem Resolution messages can be added or edited.

Figure 2‑23: HMI Alarm Problem Resolutions Screen

To add a new problem resolution message:

  1. Click Add a Problem Resolution. The Problem Resolution Details section is populated with blank fields.
  1. Using the fields in the Identification section, edit the Title, Problem Description, Primary Action, and Additional Action fields as needed.
  1. Click Add New.

To edit a problem resolution message:

  1. Select the desired problem resolution from the HMI Alarm Type table. Note that this list can be filtered by typing in the Filter field at the top of the list. Clicking on a problem resolution in the list causes that problem resolution's details to be displayed in the Problem Resolution Details section.
  1. Using the fields in the Identification section, edit the Title, Problem Description, Primary Action, and Additional Action fields as needed.
  1. Click Save Changes to save the changes to the laser head. A confirmation dialog is displays.
  1. Click OK to save the changes. Click Cancel to abort the changes.
