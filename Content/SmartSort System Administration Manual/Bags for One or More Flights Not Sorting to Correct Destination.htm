Bags for One or More Flights Not Sorting to Correct Destination

The Carrier not sending BSMs:

  • Request that the Carrier enable BSM messaging.

Flight information not entered correctly:

  • Correct the flight information in the flight schedule.
  • Ensure the Carrier code and flight number are correct and match the BSM information found in the Web Client search screens.

Flight not assigned to the correct makeup unit or correct time:

  • Correct the makeup unit mapping.

For information about Virtual to Physical Makeup Unit Mapping, refer to section 2.4.10 Virtual to Physical Mapping Configuration or the SmartSort User Guide Chapter 4: Sortation Control.

ATR not sending data, or reading data incorrectly:

  1. Confirm that no alarms are present on the HMI (see SmartSort User Guide Chapter 7: Monitoring Real-Time System Performance). Fix any that may be causing the problem.
  1. Clean the ATR heads to improve the read rate.
  1. Cycle power on the active (green) ATR controller to cause a failover to the secondary controller.
