Contec Imaging Procedure

Each of the FormsClient machines in the BHS is created from the same image (located on the NAS at \\DFW-D-NAS\DFW\FormsClient\). For a fresh computer, or to repair an existing one, the process is described below.

Hardware requirements:

  • USB Drive with the Redo software on it (either supplied by Brock Solutions, or make your own using the instructions at www.redobackup.org/download.php).
  • Another USB Drive containing the Image from the NAS (no special instructions for this drive). You can also image from the NAS, but this is much slower and requires setting up the IP address within the Redo software.
  • ELO touch screen and USB keyboard. If no touchscreen is available, connect a USB mouse as well for convenience.

The Imaging Process

  1. Turn off the unit or do not connect power to it.
  1. Plug in the ELO monitor (DVI), USB key with Redo, USB drive with image, touchscreen USB, and a USB keyboard. Then connect power and the machine will boot. Leave the network cable unplugged.
  1. Press DEL as it boots to get to the BIOS as you need the five beeps on start up. This needs to be done quickly or it will boot into Windows. The default delay is only one second.
  1. Right arrow key over to the Boot Options Tab (4th) and navigate down to the last item (Hard Drive BBS Priorities).
  1. Change the Hard Disk Boot Priority to put the local Contec SATA SM: SanDisk drive last (or at least move the Redo USB higher in the list).
  1. Press F4 to save and exit. The machine will boot to Redo.
  1. Choose to run the Redo software within 10s (second option).
  1. After Redo loads, Restore an image (Red option). Follow the prompts to select the source and target drives.
  1. The machine will be imaged in about five minutes.
  1. Choose to reboot from within Redo. As the machine is powering off/on, remove the USB drives to avoid booting into Redo again.

Post-Image Customization

Make sure to connect a network cable for this process.

  1. As the freshly-imaged computer is booting, hold down the left Shift key to prevent automatic logon to the operator account (password is operator). If this doesn’t work, log out of the operator account after it loads.
  1. Log in as administrator (password is .brock86).
  1. Let Windows install whatever device drivers it has to (usually the monitor is rediscovered).
  1. Using an ELO monitor (each time a new physical monitor is connected to the machine this process may need to be followed):
  • Calibrate the Touch Screen using the ELO shortcut on the desktop. After Windows detects the hardware, the calibration will probably be off.
  • The resolution may need to be changed to 1024x768 for BSDs (15" screen) or 1280x1024 for MECs (17" screen) if it isn't set to that automatically.
  1. Run the Setup Computer Script from the Desktop shortcut (file is located at C:\SmartSuite\Utilities) to set the correct IP address and computer name for the station when prompted.
  1. If on the BHS network, reboot now and make sure the application launches correctly. If not, shut down and reboot when appropriate.
  1. Check the local Folder C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup and ensure that the formsclient shortcut is contained in that folder. If it is not, copy it into that folder from the desktop.
