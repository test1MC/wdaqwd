Can't View Web Client on Workstations

Browser window hidden behind HMI:

  1. Use Alt+Tab to return to the original session.
  1. Clicking on a shortcut may open a new browser window that requires new credentials, or it may open another browser tab in an existing window.
Can't Print Reports or Search Results

Printer is not configured or is not connected:

  • Make sure Windows is set up with the correct default printer and that it is ready to receive documents.

Report print-outs do not appear as expected:

  • Check with the Adobe Acrobat help for assistance with printing pages or particular views.
Can't Open Excel Documents on Workstations

Microsoft Excel (or its viewer) is not installed on the local computer:

  • Install Microsoft Excel or a substitute application.
  • Save the report or search results to a memory stick and transfer to a computer that has Excel installed.
Can't Access Web Client Menus

Web session has timed out incorrectly (rare):

  • Sign out and sign back in to the Web Client to start a new session.
  • Restart the browser or open a new window to achieve the same result.
Can't Access Web Client (404 Error)

The DHS server is not accessible:

  • Confirm that the workstation is connected to the DHS using ping.

The IIS web server on the DHS is not running:

  • Restart the DHS to restore access to the Web Client.
Can't Generate Web Client Reports

Report or search had too large a time span or criteria was too vague:

  • The operation timed out.
  • Reduce the scope of your query or report and try it again.

A server error has occurred:

A message of this nature appears on the screen after a report has been requested.

  • Try the same request from a new browser session or from a different computer.
Bag Data in Reports Not Up-To-Date

Database Replication broken:

Note: Some reports will still have valid data, others will not.

  • There may be a replication alarm on the HMI if this persists for multiple hours. If this is the case, call Brock Solutions Support.
