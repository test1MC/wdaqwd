Airline Settings

Each airline can be configured with unique default Open Times, Late Times, and Close times. These times are used to calculate the time-based makeup unit assignments in the flight schedule. Airlines can be added, deleted, and edited.

The following can be configured for each airline:

  • Identification – The identification details for each airline include the airline name, a checkbox that indicates whether or not Baggage Processed Messages (BPMs) should be sent to the airline, and a checkbox that indicates if passengers are permitted to tag their own bags. If the Receives BPMs checkbox is selected, BPMs will be sent by the SAC for the relevant bag transactions. If the Is Self-Tagging checkbox is selected, the airline permits passengers to tag their own bags, and BSMs sent by that airline will also indicate if the bag is Active (verified by an agent) or Inactive (not verified by an agent). The Is Self-Tagging checkbox is only available if there are Inactive Makeup units configured. This section also displays the IATA/ICAO Code and Accounting Code for the airline.
  • Default Makeup Units – The Default Makeup Units section allows the user to add and delete Default Makeup Units. When adding a new Default Makeup Unit, a Type and Zone must be selected, as well as the Makeup Unit. After a Makeup Unit has been selected, click Add Default Makeup Unit to add the make-up unit. To delete a default makeup unit, click the Delete button beside the default makeup unit's name. Different default makeup units can be configured for different zones (e.g. Concourse A/B, Terminal 1/2, etc.).
  • Sort Periods – The default Open, Close, and Late time periods for the airline must be defined in this section. For each time period, a value can be entered in minutes. This value defines when the specified time period starts relative to the scheduled departure time. For example, if the Default Open Period is configured to be 180 minutes, the default Open Period will begin 180 minutes before the flight's scheduled departure time.

To add a new airline:

  1. Click Add New Airline. The Airline Details section on the right side of the screen is populated with blank fields, drop-down lists, and checkboxes.
  1. Using the fields, drop-down lists, and checkboxes in the Airline Details section, enter the appropriate airline information.
  1. Click Add New at the bottom of the screen to save the new airline. Clicking Cancel cancels the airline creation.

To edit an existing airline:

  1. Select the desired airline from the Airline List. Clicking the airline in the list causes that airline's details to be displayed in the Airline Details section.
  1. Using the fields in the Airline Details section, edit the airline's settings as required.
  1. Click Save Changes to save the changes to the airline. A confirmation dialog is displays.
  1. Click OK to save the changes. Click Cancel to aborts the save.

To delete an existing airline:

  1. Select the desired airline from the Airline List. Clicking the airline in the list causes that airline's details to be displayed in the Airline Details section.
  1. Click Delete Selected Airlines at the top of the screen. A confirmation dialog displays.
  1. Click OK to delete the airline(s). Click Cancel to abort the deletion.
