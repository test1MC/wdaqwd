Sort Allocation Controller (SAC)

The Sort Allocation Controller acts as the central server of the system, and performs all real-time communication requirements. The SAC component provides the sortation operations for the system. The SAC runs on Windows Server operating system, and includes a Microsoft SQL Server Database. This database stores the following transactional information:

  • Baggage Information (short term storage)
  • Flight Schedule Information (short term storage)
  • System and User Setup Information

The SQL Server database only keeps data for a short term (approximately three days) to ensure that sortation is as quick as possible. Long term data storage is on the DHS. Old records in the database are automatically cleared out daily. For flight schedules, as many Sort Assignment Templates as are necessary can be created and maintained within the database.
