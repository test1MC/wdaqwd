Laser Array Rates

The Laser Array Rate Configuration screen allows the user to change the nominal read rates and alarm thresholds for all the heads on the ATRs in the system. The nominal rate can be set once the system has run enough for the normal read rate percentage to become apparent. The nominal rate is displayed on the Laser Array Report along with the differential so that maintenance personnel can detect problems more easily.

This screen can be used to change nominal rates and alarm thresholds as needed, but laser array head information cannot be changed, added to, or deleted from this screen.

The following ATR Thresholds can be configured:

  • Flight not in Schedule – Occurs when the configured number of bag tags for a particular flight cannot be found in the Active Flight Schedule.
  • No BSM – Occurs when the configured number of bag tags for a particular Carrier are seen at an ATR and have no BSM data.
  • No Flight Makeup Unit – Occurs when the configured number of bag tags for a particular flight are seen at an ATR and have no assigned flight makeup unit.
  • Makeup Unit Lookup Failed – Occurs when the configured number of bag tags for a particular flight are seen at an ATR and the makeup unit the bag was assigned to is not configured in the Virtual to Physical Mapping.
  • Repeated IATA Tags – Occurs when a configured number of IATA tags read at a specific ATR repeat (e.g. 5 out of 50 IATA tags). This alerts the operator to the possibility that a tag is stuck on the conveyor at the ATR.
  • Tag Conflict – Occurs when the configured number of bag tags read at an ATR have conflicts (e.g. multiple tags of the same type).

To edit a laser array head's nominal read rate or alarm threshold:

  1. Select the desired laser array head from the Head List. Note that this list can be filtered by typing in the Filter field at the top of the list. Clicking a head in the list causes that head's details to be displayed in the Laser Head Details section.
  1. Using the fields in the Laser Head Details section, edit the nominal read rate and the alarm threshold as needed.
  1. Click Save Changes to save the changes to the laser head. A confirmation dialog displays.
  1. Click OK to save the changes. Click Cancel to abort the save.
