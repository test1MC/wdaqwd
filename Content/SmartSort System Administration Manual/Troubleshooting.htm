Discovering System Problems

The VMware vSphere Client can be used to monitor the performance of VMs, hosts, and clusters, and to discover issues within the system.

For example, the performance of a host can be viewed by selecting a host from the list of hosts and clusters, and then selecting Monitor > Performance in the main section of the screen.

Figure 3‑7: Performance

If an issue is occurring on a VM, host, or cluster, first check the various Monitoring tabs for that component. The Monitor tab includes a list of issues for the component, events, and tasks that have occurred for that component, as well as other metrics and information. This is the best place to review the health of a VM, host, or cluster.
