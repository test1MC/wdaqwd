Airline Codes

The various Carriers utilized in the system can be configured by the System Administrator using the Airline Codes screen. Airline codes can be added and modified, but typically all airlines in the system are already displayed on this screen. Airlines cannot be deleted.

Figure 2‑11: Airline Codes Screen

The Airline List displays a list of all airlines configured in the system. Airlines can be reorganized by clicking on the headers (i.e. sorted by Description, Two-Letter Code, Three-Letter Code, or Accounting Code), and can be filtered by typing in the Filter field.

To add a new airline:

  1. Click Add Airline. The Airline Settings section is populated with blank fields.
  1. Use the fields and checkboxes in the Airline Settings section to enter the airline's information as needed.
  1. Click Save to save the new airline.
  • Clicking Cancel or navigating away from the page cancels any unsaved changes.

To modify an existing airline:

  1. Select the desired Carrier from the Airline List. The selected airline's details are displayed in the Airline Settings section.
  1. Modify the airline's information as needed.
  • For pre-existing airlines, the only piece of existing airline information that can be edited is the description; all other information is read-only.
  1. Click Save to save the changes to the airline.
  • Clicking Cancel or navigating away from the page cancels any unsaved changes.
