Automatic Reports

The various reports available in the Web Client can be automatically generated at particular times each day. This functionality is managed from the Automatic Reports screen, accessed via the Settings menu. The user can schedule recurring jobs that export specific reports to specific destinations (i.e. to a printer or to a file). Jobs can have one or more reports associated with them. Each report is configured using the Report Generation parameters as well as a set of output destinations that can be customized for each report.

Typically, a report will have only two parameters (Start Time and End Time), but some reports have additional parameters. The report parameters that can be entered on the Automatic Reports screen are identical to those that can be entered when generating the reports themselves via the Web Client.

Each Automatic Report Job is only executed once a day. Automatic Report Jobs can be added, edited, and deleted. Report outputs can also be added, edited, and deleted.

A default job is configured to export all reports each night to a folder on the DHS where they are stored.

Figure 2‑9: Automatic Reports Screen

The following information is included on the Automatic Reports screen:

  • Description – The name of the Automatic Report Job.
  • Next Run Time – The next time the job will run.
  • Enabled – Indicates whether or not the Automatic Report Job is enabled.
