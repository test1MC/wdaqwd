Virtual to Physical Mapping Configuration

Sortation Assignments are managed through a virtual to physical mapping table. Instead of assigning a flight to a physical sort destination, such as a make-up unit (which could become unavailable), flights are instead assigned to a virtual sort destination. Usually the virtual sort destinations are identical to the physical sort destination (i.e. the virtual sort destination"MU01" would be assigned to the physical sort destination "MU01"). However, in cases of equipment failure, flight cancellations, etc., all bags can be easily rerouted by "remapping" the "MU01" virtual sort destination to another physical make-up unit (e.g. MU02).

Virtual sort destination assignments are also used by the system to handle special cases such as No Read bags. These special virtual sort destinations cannot be assigned to a flight, but they can be assigned to route bags to a specific destination. For example, in a system that has a Manual Encoding Station, any "No Read" or "Reject" bags are normally sent to the MEC, but these locations could also be re-mapped elsewhere.

Special virtual destinations are configured by the system administrator (section 2.4.9 Virtual Makeup Unit Configuration) and may include the following:

  • Cancelled Flight – This is the virtual sort destination for any bags assigned to a flight that has been cancelled.
  • Problem Makeup Unit – This virtual sort destination may be used as a Quick Release destination for bags that cannot be encoded using the MEC.
  • No Read – This is the virtual sort destination for any bags that could not be read by the Automatic Tag Reader (ATR). It is usually mapped to the MEC.
  • Reject – This is the virtual sort destination for any bags that could not be automatically be sorted by the Automatic Tag Reader (ATR). Examples of such Logical Reject Reasons are: No BSM Data, Flight Not in Schedule, Flight Make-up Unit Not Assigned, etc. It is usually mapped to the MEC.
  • Global Early – This is a virtual sort destination that can be used to sort bags for all flights that are checked in during the "Early" time period.
  • Global Late – This is a virtual sort destination that can be used to sort bags for all flights that are checked in during the "Late" time period.
  • Global Missed – This is a virtual sort destination that can be used to sort bags for all flights that are checked in during the "Missed" time period.

The Virtual to Physical Mapping screen can be accessed through the Sortation menu.

Note that if a single virtual make-up unit is mapped to multiple physical make-up units, bags will alternate between the physical make-up units in round-robin style.

To configure virtual make-up unit assignments:

  1. Select the relevant Physical make-up unit from the Physical Makeup Units list. The details for that Physical make-up unit are displayed in the Physical Makeup Unit Details section of the screen.
  1. To add a Virtual Make-up Unit Mapping, select a make-up unit from the [Add Virtual Makeup unit] drop-down list. Note that the makeup unit is automatically added to the Virtual to Physical Mapping table, but the Virtual to Physical Mapping must be saved before the change takes effect in the sortation system.
  1. To remove a Virtual Makeup Unit Mapping, select the checkbox beside the desired makeup unit(s) and click Remove. Clicking the checkbox in the header row of the Virtual Makeup units table selects all virtual makeup units. Note that the makeup unit is automatically removed from the Virtual to Physical Mapping table, but the Virtual to Physical Mapping must be saved before the change takes effect in the sortation system. If this is the last copy of a Virtual makeup unit in the mapping, it appears as a No Makeup unit (Unassigned) mapping.
  1. To copy a virtual makeup unit to another physical makeup unit or to move a virtual makeup unit to another physical makeup unit, select the checkbox beside the desired makeup unit(s) and then select the physical makeup unit from the [Move/Copy To] drop-down list. From the Move/Copy dialog that is displayed, select the Copy checkbox or the Move button as desired. In normal circumstances, Move is the desired option. Copy creates a round-robin effect for the virtual makeup unit.
  1. Once the virtual to physical mapping is configured as desired, click Save to save the changes. The changes take effect within 30 seconds.
