System Settings

System configuration (i.e. system configuration that does not directly affect the various SmartSort applications) is performed on the System Settings screen, accessed via the Settings menu. Note that most of this screen's functionality is explained in the various tooltips on the screen (the blue question marks).

 

Figure 2‑8: System Setup Screen

The screen is divided into three sections:

System Configuration

The System Configuration section of the System Settings screen allows the user to do the following:

  • Set the Day Rollover Time (the time at which system rollover operations take place)
  • Set the Application Exit Password (the exit password for other applications)
  • Configure the maximum age limit for user passwords (i.e. how long a user can go before having to change his/her password). If user passwords should never expire, this value can be set to 0.

SmartSort Configuration

The SmartSort Configuration section of the System Settings screen allows the user to:

  • Configure Maximum Sort Recirculations
  • Reset Operational Report Statistics (immediately performs the statistical reset operation that is scheduled to run at the day rollover time)
  • Reactivate the Flight Schedule (reload the flight schedule from the SAC database)

Retention Periods

The Retention Periods section of the System Settings screen allows the user to determine how long (in days) various types of data are retained in the system. This data can be used in the various searches and reports available in the Web Client. Each type of retention period is explained in a tooltip.
