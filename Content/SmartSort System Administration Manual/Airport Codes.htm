Airport Codes

The Airport Codes screen allows the system administrator to configure airport information (specifically the airport codes that are used in the system). Airport codes can be added and modified, but the system is pre-loaded with all the current IATA Station information, so this screen is provided primarily as an information-only resource to allow the user to identify the airport name associated with the 3-digit airport code. Airports cannot be deleted.

Figure 2‑12: Airport Codes Screen

The Airport List displays a list of all airports configured in the system. Airports can be reorganized clicking on the headers (i.e. sorted by Description, IATA Code, City, or Country), and can be filtered by typing in the Filter field.

To add a new airport:

  1. Click Add Airport. The Airport Settings section on the right side of the screen is populated with blank fields.
  1. Enter the appropriate airport information.
  1. Click Save to save the new airport.
  • Clicking Cancel or navigating away from the page cancels any unsaved changes.

To modify an existing airport:

  1. Select the desired airport from the Airport List. The selected airport's details are displayed in the Airport Settings section.
  1. Modify the airport's information as needed.
  • For pre-existing airports, the only piece of existing airport information that can be edited is the description. All other information is read-only.
  1. Click Save to save the changes to the airport. A confirmation dialog displays.
  1. Click OK to save the changes. Click Cancel to abort the save.
