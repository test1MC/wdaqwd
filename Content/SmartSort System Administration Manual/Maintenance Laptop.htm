Maintenance Laptop

One maintenance laptop is provided, loaded/configured with the software programs necessary to interface with the Baggage Measuring Array as well as the ATR Scanner Arrays for the collection and processing of individual scanner head results. The laptop includes all applicable PLC program ladder logic required to maintain the BHS system and features:

  • Windows 10 Operating System
  • 4GB RAM
  • 500GB Hard Drive
