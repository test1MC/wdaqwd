BSM (Bag Source Messages)

The SAC will receive baggage and flight schedule information from the Carrier host systems. This communication occurs over the Ethernet Network provided.

The SAC will comply with the following BSM interfaces:

  • SABRE Bag Data Interface (for American Carriers BSMs)
  • Arinc Bag Link (for all other Carriers)
  • IATA recommended practices 1745 (35th Edition) BSM Version 1
