Bag Tag Printers

The Bag Tag Printers screen allows administrators to add printers and assign a unique code to each one, referenced in the .T field of the BSM. Administrators can also view the percentage of bags used by each printer on the Bag Tag Printer Report. Bag Tag Printers can be added, edited, and deleted.

Figure 2‑15: Bag Tag Printers

To add a new bag tag printer:

  1. Click Add New Bag Tag Printer. The Bag Tag Printer Details section on the right side of the screen is populated with empty fields.
  2. Using the applicable fields, enter a description of the bag tag printer and a unique code (this code will appear in the .T field of the BSM).
  3. Click Add New to save the new bag tag printer. Clicking Cancel cancels the creation of the bag tag printer.

To edit an existing bag tag printer:

  1. Select the desire printer from the Bag Tag Printer List. Note that this list can be filtered by typing in the Filter field at the top of the list. Clicking the printer in the list causes that printer's details to be displayed in the Bag Tag Printer Details section.
  1. Using the fields in the Bag Tag Printer Details section, edit the printer's settings as required.
  1. Click Save Changes to save the changes to the printer. A confirmation dialog is displayed.
  1. Click OK to the save the changes. Click Cancel to abort the save.

To delete an existing bag tag printer:

  1. Select the checkbox next to the appropriate printer located in the Bag Tag Printer List.
  1. Click Delete Bag Tag Printer at the top of the screen. A confirmation dialog displays.
  1. Click OK to delete the printer(s). Clicking Cancel aborts the deletion.
