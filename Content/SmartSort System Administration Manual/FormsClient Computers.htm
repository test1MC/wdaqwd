Hardened Computers

The embedded PCs used by MECs and BSDs are also standard Windows computers. They are not equipped with a keyboard, but VNC can be used to connect from a computer with a keyboard. The application can be closed by pressing Alt + F4 or by pressing on the computer name label located in the top right corner of the screen (a pop-up will display for confirmation). It is not necessary to shut down the application before rebooting the computer.

Each FormsClient application automatically logs on with a severely restricted user account. The only functions available to the Operator user are the FormsClient application and the ELO Touch Screen calibration utility. Both have shortcuts on the desktop.
