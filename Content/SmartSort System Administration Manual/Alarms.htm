Alarms

If a VM is running and has no alarms or warnings present, the VM icon will be overlaid with a green chevron. If a warning is present for VM, the VM icon will be yellow. If an alarm is present for VM, the VM icon will be red.

If a yellow warning icon appears for a VM, a description of the warning displays in a yellow box on the Summary tab.

If a red warning icon appears for a VM, click on the Alarms tab to view alarm details. Alarms can also be cleared from the Alarms tab.

Although VMs can be shut down for maintenance, all the BHS VMs should be running at all times.
