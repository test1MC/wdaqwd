Managing Users

System Administrators can add and modify user accounts using the Users and Groups screen. Individual users are listed beneath the configured groups in the Groups List section of the screen. users list appears below the Groups list in the Groups List section of the screen. Selecting a user from the Users list populates the User Details section with information about that user, including a list of the groups to which that user belongs. System Administrators can perform the following actions on this screen:

  • Add a new user account
  • Modify an existing user account
  • Delete a user account

To add a new user account:

  1. Click Add User in the Users list section of the screen. The User Details section displays, allowing the System Administrator to enter details for the new user.
  1. Use the fields and checkboxes in the User Details section to configure the user account as appropriate.
  • If the user account should be disabled on a specific date, check the Disable Account on Date checkbox and use the calendar function that is displayed to select the appropriate date.
  • If the user's password should expire after a specific amount of time, check the Password Expires checkbox. The Password Age Limit is configured on the System Settings screen (see section 2.3.1 System Management for details).
  1. Use the checkboxes in the Group Membership table to select the groups to which the new user should belong.
  • Users can belong to more than one group, depending on their required access privileges.
  • The list of groups can be filtered by typing in the Filter field.
  1. Click Save at the top of the screen to save the new user.
  • Clicking Cancel or navigating away from the page cancels any unsaved changes.

To modify an existing user account:

  1. Select the appropriate user from the Users list. The user's details display in the User Details section.
  1. Use the fields and checkboxes in the User Details section to modify the user account as necessary.
  • The user account can be temporarily disabled if the user should not have access during a specific time frame, but the account will be needed in the future.
  • To unlock an account that has become locked due to invalid password attempts, deselect the Account Locked checkbox.
  1. Click Save at the top of the screen to save the changes.
  • Clicking Cancel or navigating away from the page cancels any unsaved changes.

To delete an existing user account:

  1. Select the appropriate user from the Users list. The user's details display in the User Details section.
  1. Click Delete User.
  1. Confirm the deletion. A Delete Confirmation message displays. Click OK to delete the group. A message displays to indicate that the user was successfully deleted.
  • To cancel the Delete operation, click Cancel.
  • The currently signed-in user account cannot be deleted.
