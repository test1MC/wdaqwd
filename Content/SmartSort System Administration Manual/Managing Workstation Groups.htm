Managing Workstation Groups

Workstation groups are created and defined using the FormsClient Workstation Setup screen in the Web Client, accessed via the Settings menu. When this screen is loaded, a list of all existing groups is displayed.

Figure 2‑6: FormsClient Workstation Setup Screen

The FormsClient Workstation Setup screen allows a user to do the following:

  • Add a new group
  • Edit an existing group
  • Delete a selected group (note that a group cannot be deleted unless it has no workstations)
  • View all the workstations in a selected group

To add a new workstation group:

  1. Click Add Group at the top of the screen. The Workstation Group Edit dialog displays.
  1. Use the fields and checkbox to configure the following workstation group parameters:
  • Name – Enter the name of the new workstation group.
  • Description – Enter a brief description of the new workstation group.
  • Requires Authentication – Selecting this checkbox will require users to enter authentication credentials when accessing workstations within the group.
  • Inactivity Timeout (minutes) – The amount of time (in minutes) that a workstation must be inactive before it will time-out.
  1. Click Add. A confirmation dialog displays.
  • Click OK to create the workstation group. Click Cancel to abort.
  • Note that the new group will have no workstations until the Workstations Details screen is used to add workstations to the group.

To edit an existing group:

  1. Select the desired group from the Workstation Group filter, then click Edit Group. The Workstation Group Edit dialog displays
  1. Use the fields and checkbox to change the group's settings as necessary.
  1. Click Save to save the changes to the group. A confirmation dialog displays.
  • Click OK to save the changes. Click Cancel to abort the changes.

To delete an existing group:

  1. Select the desired group from the Workstation Group filter, then click Edit Group. The Workstation Group Edit dialog displays.
  1. Click Delete. A confirmation dialog is displayed.
  • Click OK to delete the selected group. Click Cancel to abort the deletion.

Note that a group cannot be deleted if it has any workstations, so the Workstations setup screen must first be used to remove any workstations from the group.
