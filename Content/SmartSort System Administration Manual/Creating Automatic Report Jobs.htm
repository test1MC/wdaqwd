Creating Automatic Report Jobs

Clicking Add New Report Job at the top of the Automatic Reports screen displays the Automatic Reports Edit screen.

Figure 2‑10: Automatic Reports Edit Screen

This screen is divided into two sections:

  • Job Details – Includes information about the specific report job, including when it will be run, the groups who will receive the reports, and the reports that will be included in the job.
  • Report Details – When a report is selected for inclusion in the Automatic Report Job, that report's details will be displayed in this section. The top half of this section includes the report's parameters, while the bottom half includes the available report output options.

To create a new Automatic Report Job:

  1. Click Add New Report Job on the Automatic Reports screen. The Automatic Reports Edit screen is displayed.
  1. Enter the desired job name in the Description field.
  1. Select the next time the report should be run using the calendar function in the Next Run Time field.
  1. Select the Enabled checkbox to enable the new job.
  1. Select the appropriate Interval Hours (i.e. how often the report should be run). By default, this is set to 24 (once a day), but can be set so that reports are generated more often if necessary.
  1. Use the Run Period drop-down list to select the days of the week that this report should be run. The available options are every day (Monday - Sunday), only on weekends (Saturday - Sunday), or only on weekdays (Monday - Friday).
  1. Optionally, the contents of the job can be sent to an email group (see section 2.1.4 Email Group Configuration). Any reports that should be emailed need to have their output type set to one of the Email options.
  1. The Zip Reports checkbox can be used for jobs that have an email group. If selected (and it is by default), all reports included in the job will be zipped prior to being attached to the email. If not, the reports will be attached directly. Note that SSI reports will be zipped (and password protected) regardless of this setting.
  1. Use the checkboxes in the Reports Included section to determine which reports will be included in the new job. Selecting a checkbox means that the selected report will be included in the list of reports in that report job. Note that the list of reports can be filtered by typing in the Filter Results field.
  1. If necessary, edit each report's parameters in the Report Details section. Clicking an included report in the Reports Included section displays that report's details on the right side of the screen (note that a report must be selected for inclusion in the job in order to have its details displayed). The report parameters can be filtered by typing in the Filter Results field.
  1. Using the checkboxes in the Send To column of the Report Output table for each selected report, determine the output formats for each report to be included in the automatic job. The following options are available:
  • Email – CSV ‑ Saves the report in CSV format to be emailed with any other reports in the job.
  • Email – PDF ‑ Saves the report in PDF format to be emailed with any other reports in the job.
  • Laser Printer – Sends a copy of the report to a selected printer.
  • Save to DHS – Excel – Saves an Excel file to a specified folder on the DHS.
  • Save to DHS – PDF – Saves a PDF of the report to a specified folder on the DHS.
  1. Click Save Report Job Changes to save the new report job. Note that clicking Return to Report Jobs without first saving the new job cancels the job creation.
