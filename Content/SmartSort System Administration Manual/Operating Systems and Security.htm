Operating Systems and Security

The BHS is designed to be run on a variety of Windows operating systems. Servers use a different class of operating system than client computers. The operating system for each server is as current as possible, but may be restricted by the constraints of the applications installed on it.

Computer security is managed using the built-in Windows functionality. All computers are members of the existing BHS domain. As such, all BHS computers have security settings managed from a common domain server.
