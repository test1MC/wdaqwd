Data Historian Server (DHS)

The DHS Virtual Machine acts as a second central server within the system, and controls all data collection and long-term data storage. The DHS runs on the Windows Server operating system, and also includes a Microsoft SQL Server Database. This database stores all report and search data for the system, including the following:

  • Baggage Information (long term storage)
  • Flight Schedule Information (long term storage)
  • Statistical and Report Data
  • System and User Setup (replicated for redundancy)

The SQL Server database can store up to three years of historical baggage data, bag transactions, and statistical data. All BHS reports are automatically archived to disk daily in both PDF and Excel format. Archived files are also deleted from the disk after three years.
