Retag Configuration

The Retag Configuration screen allows the administrator to enable different scenarios in which bags need to be retagged. It also allows the administrator to configure some settings for ATRs.

Figure 2‑19: Retag/Scanner Configuration Screen

The Global Retag Options located at the top of the screen are applied to each ATR in the system. If a scanned bag tag meets any of the retagging rules it will be sorted according to the Retag Sort setting.

Baggage can be flagged for retagging for any of the following reasons:

  • Retag for Flight Route Changes – If the flight route changes for a bag (e.g. the destination city changes or a flight leg is added), the bag can be sent for retagging.
  • Retag for Flight Number Changes – If the flight number changes, but the bag is still going to the same airport, the bag can be sent for retagging.
  • Retag from Host BSMs – If a flag is set in the BSM from SABRE indicating that the bag needs to be retagged (.E/RUSH).
  • Retag for Dynamic Reroute – If the bag is rerouted using dynamic reroute, the bag can be sent for retagging.

For each ATR, the destination for retagging bags can be set (e.g. Manual Encode, Retag Pier, or Disabled). The global retagging options will apply to all retagging scanners. The Save Configuration button must be clicked to save the global retagging options.

Each ATR also can be configured with a specific Late Pier. If this location is assigned, bags that are late and do not have a flight late pier will be sent to this location ahead of the Global Late Pier (if it is mapped). Manual Encodes will send late bags to the location configured at the closest ATR.

The Manual Encode destination assigns bags to the next available MEC location determined by the PLC. The Retag Pier destination assigns bags to the physical destination(s) mapped to the Retag virtual pier in the Virtual to Physical Mapping. The Disabled choice will not send retag bags at that ATR anywhere special, although they could be reassigned by an ATR downstream (if it is configured differently). Bags that arrive at an MEC will be processed there regardless of the Retag Sort setting for any associated ATRs.

To configure Retagging Logic for an ATR:

  1. Select an ATR from the Retag Scanner list.
  1. Select a location from the Retag Sort drop-down list.
  1. Click Save Changes.
