Power On/Off, Startup/Shutdown

VMs can be powered off and restarted in a variety of ways. Shutting down or restarting the Guest OS from within Windows (as usual) closes all applications and properly shuts down the VM.

Powering the VM off or resetting the VM immediately turns off the VM without allowing the guest operating system time to properly shut down. The latter action is similar to unplugging the power cord on a physical computer and should not be used unless a VM has stopped responding.

To perform a power action:

  1. In the VMware vSphere Client, right-click the VM in the navigation panel.
  1. Select Power, and then select a power option (Power On, Power Off, Shut Down, or Restart).
