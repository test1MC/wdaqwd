3.3.10 Database Backups

Each night the complete database is backed up to a file on the local hard drive. Database backups are retained for a two-week period. Scheduling and organization of these nightly backups is handled by the Microsoft SQL Server database software. Incremental backups are not performed.

Automated jobs run every morning at the time configured in the Web Client application, and generate a full backup of the SAC, DHS and DHS databases. The backup files are located in subfolders in the following directory:

  • D:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Backup\

The Database files themselves are located on a separate local disk to reduce the risk if there were a complete disk failure. Live data files are in the directory:

  • D:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data\

Some changes to the backup job schedules can be made through the Scheduled Jobs tab in the Facility Setup of the Web Client application.

Restoring the Database from Backup

Do not restore a backup over the existing DHS databases unless directed by Brock Solutions support.

The standard SQL Management Studio tools can be used to restore the database. Both the DHS and the SAC must be restored from the same day’s backups if a restore is necessary. Since the DHS database is much larger than the SAC database, the backups of each system will never be perfectly in sync. Replication must be reinitialized once the databases have been restored to clean up the duplicate records.
