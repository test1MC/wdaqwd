BHS Status Monitor

The BHS Map displays the operational status of the conveyor system as well as individual bags flowing through the system. The BHS Status Monitor is required to be running for the operational status of the BHS to be accurately depicted on the BHS Map.

This service records the conveyor and device status from the system PLCs using direct Ethernet/IP connections and notifies the BHS Map processes using Events handled by the Job Processor service.
