Manual Contents

The System Administration information contained in this manual is divided into three chapters as follows:

  • Web Client Administration – Contains information about the various settings screens available in the Web Client, and instruction on the essential configuration operations available in the system.
  • System Network Administration – Contains information on detailed system architecture, including the primary applications used in the upper-level software system (e.g. SAC, DHS, VMware, etc.). Information on the upper-level system functionality, networks, services, and installation is also included in this chapter.
  • System Maintenance and Troubleshooting – Contains computer maintenance and troubleshooting information for the various services and applications in the system.

For detailed information on a specific type of system administration, refer to the appropriate chapter.
