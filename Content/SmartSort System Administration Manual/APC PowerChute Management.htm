APC PowerChute Management

APC PowerChute Management is a VM appliance, meaning it is a pre-configured virtual machine provided by the UPS vendor and hosts the PowerChute Network Shutdown application. It is protected by the High Availability environment. PowerChute communicates over the network with the APC UPSs in the server rack (using the UPS Network Management Cards) to protect the VMware High Availability environment from power disruptions and/or extended outages. When availability is threatened, PowerChute gracefully shuts down the physical and virtual servers to protect data integrity.

For additional information regarding APC PowerChute Network Shutdown application, please refer to the PowerChute Network Shutdown v4.1 VMware User Guide.
