Windows/Domain

The following sections outline outlines the common Windows issues, and the probable causes and remedies for those issues.

Can't Log onto Computer

Password not accepted because Caps Lock and/or Num Lock is on:

  • Make sure that no unexpected keys are pressed that would change the password as it is typed.

Password expired or was changed:

  • Check with your system administrator for the latest authentication details configurable on the (see section2.1.2 Managing Users).
Time Synchronization

Domain is not accessible or time synchronization has failed:

Note that Windows time can drift up to a couple of minutes before it is fixed.

  1. Confirm that the domain controllers or time server are functioning correctly.
  1. Use w32tm /resync from a command prompt to force an update. If this update returns an error, further troubleshooting is required.
Automatic Login Not Functioning

Shift key held down during boot process:

  1. Log in using the correct password.
  1. Reboot the computer if the password is not known.

Automatic login has been disabled or is not configured:

  • Get a system administrator to update the registry or use the Sysinternals Autologon tool to set the automatic logon credentials.
Named Computers Not Responding When Pinged

DNS servers are not available:

  • Ensure the domain controllers are functional.
  • Check the Windows Event Logs for errors.

DNS is not properly configured on the client computer:

  • Make sure the correct IP addresses are used for the DNS servers when using a static IP configuration.
General Windows Problems
  1. Use the Windows Event Log or System Information as a troubleshooting tool.
  1. Contact Brock Solutions for further troubleshooting assistance.
