Managing Reroute Groups

Reroute groups can be added, deleted, and modified. Instructions for managing reroute groups are below:

To Add a New Reroute Group:

  1. Click Add New Reroute Group. The Group Details portion of the screen will display default values for all fields.
  1. Enter the description, select the reroute mode, configure the Search Window settings, configure the Flight Options, and the Reroute Type Settings.
  1. Click Add Reroute Group.

To Edit a Reroute Group:

  1. Select the desired reroute group from the Reroute Group List table. The reroute group details will be displayed.
  1. Configure the reroute settings as desired.
  1. Click Save Changes.

To Delete a Reroute Group:

  1. Select the checkbox beside the desired reroute group in the Reroute Group List table. Selecting the checkbox in the header row will select all reroute groups.
  1. Click Delete Selected Reroute Groups. A confirmation dialog will be displayed.
  1. Click OK to delete the reroute group.

For details about configuring reroute groups, see the SmartSort User Manual – Chapter 4: Sortation Control.
