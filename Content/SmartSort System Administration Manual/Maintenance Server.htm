Maintenance Server

The Maintenance Server is a separate physical server used to administer the other BHS computers and used by authorized users to access all computers and devices on the BHS Network to diagnose problems and perform support activities. The Maintenance Server provides a central location through which to access the PLC programs, modify the HMI application, or troubleshoot upper-level problems. This single, non-redundant server is in the server rack. It does not run any processes that are critical to system operation.

The Maintenance Server hosts:

  • Rockwell software licenses for Studio 5000 (PLC programming), RSNetwork (ControlNet maintenance), and FactoryTalk View Studio (HMI programming). These licenses can be used locally on the Maintenance Server or remotely by another computer where the software is installed.
  • Centralized virus definition server.
  • The SICK RDT software will also be installed on the Maintenance Server. It is a web based program that provides information on the various SICK ATRs installed in the system.
