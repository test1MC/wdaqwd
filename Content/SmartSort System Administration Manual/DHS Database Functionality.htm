DHS Database Functionality

The DHS is the long-term data storage server and stores data for the amount of time configured in the Web Client(see section 2.3.1 System Management). Maximum retention periods are described in section 3.3.8 Cleanup Utilities (SQL Jobs). It includes applications that collect and display non-sortation data used by the operators.
