Counter Collection Service (CoCoS)

The Counter Collection Service records statistical counts from the system PLCs. This service gathers this data every minute and stores the differences in the database for use in reports. A variety of reports available on the Web Client use this counter data.

The connection to the PLCs is made through the Kepware OPC server installed on the DHS server. It is possible that the service may continue to function even though it fails to read some data in the PLC arrays. This condition is apparent from unexpected zero data on the reports, but does not generate an alarm because the collection is still occurring. The cause of this is memory fragmentation within the PLC caused by extensive online editing. Restarting the CoCoS service, restarting Kepware, or cycling power on the PLC are all possible solutions.

The CoCoS services must be kept running at all times to ensure that accurate daily reports can be generated for the BHS.
