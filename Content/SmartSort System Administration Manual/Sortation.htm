Bags Not Sorted to Correct Locations

Note that this excludes the MEC.

Makeup units in the Virtual to Physical Mapping not mapped properly:

  • Check the Web Client Virtual to Physical Mapping screen or generate a report to confirm that all virtual makeup units are properly mapped.
  • Most virtual makeup units should be in a one-to-one mapping with the physical ones.
  • No virtual makeup units should be unmapped.

For information about Virtual to Physical Makeup Unit Mapping, refer to section 2.4.10 Virtual to Physical Mapping Configuration or the SmartSort User Guide Chapter 4: Sortation Control.

Makeup unit assignments for the flight incorrect in the flight schedule:

  • Confirm that the flight is located on the correct makeup unit in the Web Client flight schedule screens.

For information about flight schedules, refer to section 2.4.10 Virtual to Physical Mapping Configuration or SmartSort User Guide Chapter 4: Sortation Control..

High Number of Bags Sent to MEC (Bags Not Automatically Sorting)

BSM connectivity issue:

  1. Check the SmartConnect Message Broker Service service(s).
  1. Check connectivity to the BSM provider and contact them to investigate the problem.

BSMs not received for a subset of Carriers:

  1. Depending on BSM configuration, a subset of services could be down.
  1. If some BSMs are not being received, contact the BSM provider to investigate the problem. This may be a problem outside of the scope of the BHS control (e.g. problems at SITA or ARINC).
High Amount of Unknown Status Bags

Bad/misaligned photo eye or ATR misreads:

  • Check the 9.16 Equipment Operational Summary Report to isolate trouble areas/devices.
  • Fix or realign photo eye.
  • Check ATRs.
  • Search for impacted bag tags.
Bags for One or More Flights Not Sorting to the Correct Destination

The Carrier not sending BSMs:

  • Request that the Carrier enable BSM messaging.

Flight information not entered correctly:

  • Correct the flight information in the flight schedule.
  • Ensure the Carrier code and flight number are correct and match the BSM information found in the Web Client search screens.

Flight not assigned to the correct makeup unit or correct time:

  • Correct the makeup unit mapping.

For information about Virtual to Physical Makeup Unit Mapping, refer to section 2.4.10 Virtual to Physical Mapping Configuration or 4 Sortation Controlthe SmartSort User Guide Chapter 4: Sortation Control.

ATR not sending data, or reading data incorrectly:

  1. Confirm that no alarms are present on the HMI (see 7 Monitoring Real-Time System PerformanceSmartSort User Guide Chapter 7: Monitoring Real-Time System Performance). Fix any that may be causing the problem.
  1. Clean the ATR heads to improve the read rate.
  1. Cycle power on the active (green) ATR controller to cause a failover to the secondary controller.
All Bags or All Unknown Bags Sorted to One Location

Bag tag stuck to a conveyor belt:

  • Check conveyor belt to ensure no bag tags are stuck to a belt.
