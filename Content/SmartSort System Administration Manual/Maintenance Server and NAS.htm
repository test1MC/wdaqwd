Maintenance Server and NAS

The maintenance server is a separate physical server outside of the vSphere environment and is used for troubleshooting and system maintenance. It does not provide any mission-critical functions for the BHS.

All relevant software for the operation of the BHS is installed for use:

  • vSphere Windows and Web Client
  • SQL Server Management Studio (Database management)
  • FactoryTalk View Studio (HMI development)
  • RSLogix5000 (PLC program access)
  • RDT Software (ATR diagnostics)

Similar to the maintenance server, the NAS does not provide any mission-critical features. Instead, it is used for long-term storage of data and other information relevant to the operation. It can be accessed from any device on the network at DFW-D-NAS.smartsuite.dfwd. There are four shared directories:

  • BHS files: device configuration, PLC programs, drawings, etc.
  • Software – third-party software installs files
  • TL6015 – Brock Solution's storage while on site
  • Info – NAS information (built-in share)
