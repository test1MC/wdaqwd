Incorrect Active Flight Schedule Data

Web Client flight schedule template is not correct:

  1. Make the necessary changes to the template and reactivate the schedule.
  1. Move flights to the proper makeup units or update their information from the Gantt or Grid screens.

Note: this will only help with the current schedule. The template must also be fixed.

For information about editing flight schedule templates, refer to SmartSort User Guide Chapter 4: Sortation Control.

Flight template import process was unsuccessful or incomplete:

  1. Re-import all valid flight schedule data from a known-good source.
  1. Confirm that the new data is correct using the Web Client.
  1. Copy the updated schedule or reactivate to merge the new data with the old.

For information about importing flight schedule templates, refer to SmartSort User Guide Chapter 4: Sortation Control.

FIDS updates are changing the flight schedule data in unexpected ways:

  1. Turn off the FIDS connection
  1. Contact the FIDS provider to ensure that the data is correct.
