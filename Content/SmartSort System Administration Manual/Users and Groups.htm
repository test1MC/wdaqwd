Users and Groups

The User Administration functions in the SmartSuite Web Client are accessed via the General and User Settings sections of the Settings menu. User settings allow System Administrators to add users and define user access privileges.

System Administrators can add users and assign access privileges based on membership in user groups. To perform this process, access the Users and Groups screen via the Settings pop-out menu.

Access to SmartSuite is controlled by a security architecture based on users and groups. Security groups are added and the access privileges for that group are defined by the System Administrator. When the System Administrator adds new users, each user is assigned to one or more of these security groups based on the required access privileges.

Figure 2‑1: Users and Groups Screen

Note: Members assigned to a given group are limited to the permissions configured in the parent group. Additionally, users can only see/access groups to which they belong.
