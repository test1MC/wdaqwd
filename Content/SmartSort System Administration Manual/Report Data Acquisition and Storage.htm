Report Data Acquisition and Storage

The Counter Collection Services (CoCoS) on the DHS accesses statistical data being accumulated and stored on the PLC and write this data to the database for use in reports. Different types of information are updated at different rates. Static information (e.g. screening machine status) is recorded at 15-minute intervals. All operational and throughput data is collected on a 1-minute interval.

This information is stored according to the retention period using the intervals mentioned above. However, as part of the nightly cleanup process, the CoCoS data is summarized into daily totals. This daily summary data is retained for longer so that certain reports can be run over a wider time range.
