Managing Automatic Report Jobs

The Automatic Reports setup screen is used to manage existing report jobs. The icons in the leftmost column are as follows:

Icon

Function

Edit the selected report job (see section 2.3.2.2.1 Editing an Automatic Report Job).

Click the checkbox in the job's row and the Delete Selected Report Jobs button at the top of the screen (see section 2.3.2.2.2 Deleting an Automatic Report Job).

 

In addition, the list of report jobs can be filtered using the Filter field. Typing in this field causes the list to be dynamically filtered to match the desired results.
