Remote HMI Workstation Architecture

One operator workstation is configured to run the HMI application, a virtual machine containing the FSC and other BHS user interfaces. Two of the operator workstations run the HMI application on dual monitors. The other two workstations are used for maintenance and BHS user interfaces.

Each of the two new workstations is configured as follows:

  • Dell precision Tower 3420
  • Intel i5-6500 3.6 GHz processor
  • 8 GB RAM
  • DVD+/-RW Drive
  • Dual Video Adapter
  • Dell Ultrasharp 24" Monitor
  • Windows 10 Professional (64-bit)
  • Adobe Reader
  • FactoryTalk View SE Client
