Operator Workstation Architecture

Three Operator Workstations are included in the control room. The HMI has been installed on all of these Workstations. The workstations are also able to access the Web Client.

All the workstations are connected to displays with a resolution of 1920x1080. The Operator can navigate to any screen within the HMI application. All workstations in the control room are powered from the server rack UPSs, although not all the monitors have UPS power.

Each of the two new workstations is configured as follows:

  • Dell Precision Tower 3420
  • Intel i5-6500 3.6 GHz Processor
  • 8GB of RAM
  • 1TB Hard Drive
  • DVD/RW Drive
  • Gigabit Ethernet
  • USB Keyboard & Mouse
  • Dell Ultrasharp 24" Monitors
  • Windows 10 Professional (64-bit)
  • Adobe Reader
  • FactoryTalk View SE Client
  • Internet Explorer 11
