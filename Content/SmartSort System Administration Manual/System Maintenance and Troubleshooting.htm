System Maintenance and Troubleshooting

Periodic maintenance is required to ensure the system can function at peak performance. Automatic maintenance procedures exist to clean up log files, etc. (see chapter 3 for details), but manual maintenance is also required. This chapter outlines some of those basic maintenance procedures and practices (see section 4.1 Maintenance). It also provides information about keeping the virus definitions up-to-date (see section 4.2 Anti-Virus Software).

Lastly, this chapter contains a troubleshooting guide which is intended to guide an Administrator through many common problems that may be encountered. For details, see section 4.3 Troubleshooting Guide.
