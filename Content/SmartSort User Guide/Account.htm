Managing User Preferences Account

Selecting User from the Main menu (see Figure 2‑5: Signing Out of the Web Client) displays the Account screen. From this screen, the user can change his/her password, configure his/her email address (where email reports are sent), set the default page that will load when the user logs into the Web Client, and set the display language for that account, if applicable.

Figure 2‑6: Account Screen

To change a user's password:

  1. Enter the old password in the Current Password field.
  1. Enter the desired new password in the New Password field.
  1. Enter the new password again in the Confirm New Password field.
  1. Click the Save New Password button to change the password.

To change the email address associated with the user's account:

  1. Enter the desired email address in the Email field.
  1. Click Save Email to change the email.

To change a user's default page:

  1. Select a page from the Default Page drop-down list.
  1. Click Save Default Page to change the default page.
