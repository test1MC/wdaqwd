Current System Activity

Real-time activity for the system can be seen from the Activity Log. This log contains detailed information about the automatic operations of the BHS. It can often be used by system operators or administrators to monitor for signs of issues.

To view the Activity Log, click the Main Menu icon (), select Dashboards, then click Current System Activity.

When viewing the Activity Log screen, the entries on the screen scroll by in near-real-time. The scrolling can be paused by clicking Pause. Use the filter trees on the left side to reduce the information that is displayed to troubleshoot a specific problem.

Figure 7‑37: Current System Activity

The following filter criteria are available:

  • Levels
  • Error – Displays only error messages or activity related to errors.
  • Warning – Displays only warnings or activity related to warnings.
  • Information – Displays status messages and other entries related to regular operation of the system.
  • Applications – Tree that contains various software components of the system. Some applications also have subsystems, which can be further filtered when one of those applications is selected.

Clicking on a parent item in the Applications tree will select or clear all of the child items.
