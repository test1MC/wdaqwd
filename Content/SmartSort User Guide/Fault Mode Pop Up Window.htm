Fault Mode Pop-Up Window

The Fault Mode Pop-Up window can be accessed by clicking the FM button in the Title/Navigation bar. When a Fault Mode is active, the FM button will turn red.

Figure 7‑24: Fault Mode Pop-Up Window

An operator with sufficient privileges can adjust the Fault Mode countdown times using this pop-up window.
