Shifting Flight Times in the Active Flight Schedule

Clicking Time Shift displays the Time Shift dialog. The Time Shift function allows the user to shift flight times for any selected rows. If multiple flights are selected, the specific flights that are affected can be displayed/hidden by clicking on Show All Affected Flights/Hide All Affected Flights. There are two options for performing a time shift operation:

  • Shift All Times – This will shift the Open, Late, Close, and Estimated Departure each by the same number of minutes specified.
  • Shift Individual Times – This allows shifting of the Open, Late, Close and Estimated Departure times independently by different numbers of minutes specified.

When OK is clicked, the grid will reflect the changes and the status of the flight(s) will become MODIFIED, but changes are not saved to the server until Save All is clicked.
