Equipment Operational Summary Report

The Equipment Operational Summary Report is a single report with sub-sections for each type of equipment that was selected (e.g. Photo Eyes, Diverters, etc.). By default, the report will show details for all devices; however, the operator can select which types of devices should be included in the report. For certain devices, the operator can also select the subsystems for which statistics will be displayed. The operator can also configure thresholds for certain data (to exclude zero counts and/or low counts for certain devices.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Include Photo eyes with Zero Jams – If selected, the statistics for all jam photo eyes will be displayed even if no jams were detected.
  • Threshold Values – Entering a number in the Threshold field will set the maximum statistics that will be displayed for the selected equipment. By default, no thresholds are selected (so that there are no limits on the results).
  • Equipment Type and Equipment Subsystem Filters – Breaks up the equipment in the report by type of equipment and by zone/conveyor line.

The following information is included in this report:

Diverters (HCD)

Column

Description

Diverter

The name of the Diverter for this row of data

Cycle Count

Number of times the Diverter cycled to divert a bag or return to home

Verified Diverts

The number of bags that diverted successfully.

Proximity Failure

The number of times a proximity sensor fault was detected for the diverter.

Failed to Extend

Number of bags that could not be diverted because the diverter failed to extend

Failed to Retract

Number of bags that could not be diverted because the diverter did not properly retract

Failed at Home

Number of bags that could not be diverted because the diverter did not properly extend

Overlength

The number of bags that could not be diverted because they were too long (e.g. two bags were pushed together or the bag was oversized).

Close to Bag

Number of bags that could not be diverted because the bags were too close

Makeup Unit Full

The number of bags that could not be diverted because the destination Make-Up unit was full.

Switched Off

The number of bags that could not be diverted because the diverter was switched off.

Manual Mode

Number of bags that could not be diverted because the selector switch on the HCD was not in the Auto position

Out of Service

Number of bags that could not be diverted because the diverter was not in service (set in the HMI)

Motor Overload

Number of bags that could not be diverted because the diverter motor was overloaded

Failed to Divert

The number of bags that failed to divert for a reason not covered by the divert fault categories in this report.

E-Stopped

Number of times a bag could not be diverted due to an E-Stop

Divert All

The number of bags that diverted because the diverted was in Divert All mode.

Unknown

The number of unknown bags that were diverted.
