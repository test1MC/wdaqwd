Flight Schedule Report

The Flight Schedule Report provides lists of flights and their associated Make-Up Unit assignments in a convenient format for printing. This report can also be used to display a list of flights that have not yet been assigned to Make-Up Units.

This report can be generated in one of three ways:

  • Active Flight List – Displays all flights currently in the schedule.
  • Active Unassigned Flight List – Displays all flights currently in the schedule that are missing On-Time Make-Up Unit information.
  • Flight Schedule Template – Allows the user to select a Flight Schedule Template from the "Schedule" drop-down list in order to display the flights in that template.

The following selection criteria are available:

  • Carrier
  • Flight Number
  • On Time Pier
