Configuring Default Carrier Piers

The Airlines screen is used to define the sortation rules for each carrier. In a system that does not have a flight plan, the configured carrier default piers can be used to sort bags to the correct sort destinations.

The following can be configured for each carrier:

  • Identification – If the Is Self-Taggingcheckbox is selected, the carrier permits passengers to tag their own bags, and BSMs sent by that carrier will also indicate if the bag is Active (verified by an agent) or Inactive (not verified by an agent). This section also displays the IATA/ICAO Code, and Accounting Code for the carrier.
  • Default Piers – The Default Piers section allows the user to configure the default virtual pier to which all various types of bag tags should be assigned. The selected Virtual Piers will be used for sortation if no sort destination has been assigned for a flight in the flight schedule or no BSM information is available.
  • Sort Periods – The default Open, Close, and Late time periods for the carrier must be defined in this section. For each time period, a value can be entered in minutes. This value defines when the specified time period starts relative to the scheduled departure time. For example, if the Default Open Period is configured to be 180 minutes, the default Open Period will begin 180 minutes before the flight's scheduled departure time. For more information about time-based sort piers, refer to section 4.3.1 Flight Times.

To access the Airlines screen, click the Main Menu icon (), select Sortation, then click Airlines.

Adding a New Airline

  1. Click the Add New Airline button. The Airline Details portion of the screen will be displayed with default values for every field. The default Open Period, Late Period, and Close Period are filled in automatically because these fields must be provided.
  1. Select an Airline from the drop-down list. Change the settings as desired.
  1. Click the Add New button. The airline will be added to the Airline List.

Deleting an Airline

  1. Select the checkbox(es) beside the airline(s) that should be deleted.
  1. Click the Delete Selected Airlines button. A confirmation dialog will be displayed.
  1. Click the OK button to delete the selected airline(s).

Carriers that are in use somewhere in the system (e.g. flight schedules or other configurations) cannot be deleted.

Editing a Carrier

  1. Click on the desired carrier in the Airline List. The Airline Details for that airline will be displayed.
  1. Edit the details as required.
  1. Click Save Changes. A confirmation dialog will be displayed.
  1. Click OK to save the changes.
