Using a BRP

When a bag enters CBRA, the bag is conveyed to the furthest downstream BRP station available on the alarm line. When no bag is present at the BRP station, it will indicate a "Waiting for Bag" state using an on-screen message.

Figure 65: BRP - Waiting for Bag

When a bag arrives, and stops at the associated queue photo eye, the bag data along with the Bag Security Status displays. The bag information that the BRP can display includes:

  • Bag Pseudo ID - The PLC-generated value passed to the ISD to associate with the image (used when no other ID is available).
  • IATA Bag Tag Number - If the bag had a readable barcode, it will be passed to the ISD to associate with the image (if no ATRs are installed upstream of screening, this field will be hidden).
  • RFID Tag Number - If the bag had a readable barcode, it will be passed to the ISD to associate with the image (if no readers are installed upstream of screening, this field will be hidden).
  • ISD Machine - The device that screened the bag.
  • BHS ID for the Bag - The PLC ID used for tracking the bag in the system (can be used for bag research from the SmartSuite Enterprise Web Client).
  • Photo Eye - Certain bag statuses displays the photo eye where the bag began to be tracked by the PLC.

The Bag Security Status is indicated with a color associated with the security status. The colors and statuses are defined in PGDS. These colors and statuses are:

  • Cleared (Green) - The bag was cleared by the ISD or through OSR. Bags should rarely arrive in CBRA with a Cleared status.
  • Alarmed (Red) - The bag was not cleared by the ISD or through OSR.
  • ISD Unknown (Yellow) - The bag either received a decision of "Error" from the ISD or did not receive any response at all.
  • BHS Unknown (Blue) - The bag was lost in tracking.
  • Out of Gauge (Orange) - The bag in question was small enough to be conveyed by the conveyor equipment but was too large for the ISD.
  • Reintroduce (Black) - A user at an upstream station has requested that this bag be reintroduced into the system for screening. It should not be removed at this station.

When a bag is present at a BRP station, the user needs to transfer the bag to one of the available BIT stations. The availability of each BIT station displays on the screen. All BIT stations that are not available will have a gray background. All BIT stations that are available and ready to process bags will be green. The primary BIT station (the one located nearest to BRP stations) will be larger and will be indicated as the default BIT station.

Figure 66: BRP - Bag Present

The SmartSuite Enterprise Web Client is used to configure the BIT stations that are associated with each BRP station (including which BIT station will be the default). For more information, refer to the SmartSort System Administration Manual Chapter 2: SmartSuite Enterprise Web Client Administration.

Once a BIT station is selected, the user has a certain amount of time to remove the bag from the associated queue photo eye and physically transfer it to the BIT station. A timer displays on the BRP workstation to let the user know how much time is remaining to transfer the bag. If the bag is not transferred in time, the transfer will automatically be cancelled. The user can also manually cancel the transfer at any time by clicking Cancel.

Figure 67: BRP - Transfer Bag Countdown

Once the bag is removed from the BRP queue photo eye, the bag's information will be displayed on the selected BIT workstation and the BRP will return to the "Waiting for Bag" state.

BRPs can be enabled or disabled by pressing the button in the top right corner of the screen. When the BRP is enabled, the button will read Disable. When disabled, a BRP queue position is not available to process bags, and the following screen displays:

Figure 68: BRP - Workstation Disabled

No bags will stop at this queue for CBRA processing. A bag may still stop for normal bag flow conveyor control. Clicking Enable will re-enable the BRP terminal.

If the user removes a bag from the associated queue photo eye without selecting a BIT workstation, an error screen will be displayed to indicate that the bag was removed improperly.

Figure 69: BRP - Bag Removed Improperly

In normal operations, the user should return the bag to the conveyor, ensuring that the bag is blocking the queue photo eye, and follow the procedure described above.

Purging the bag will remove its data from the system entirely (and cannot be recovered for use at a BIT).
