Quick Release

The user can choose to send a bag directly to a specified Quick Release location (any virtual pier destination may be configured as a Quick Release, but typical Quick Release includes locations such as Reject or Problem) via the corresponding button. The specific Quick Release stations that are available is configurable for each MEC station using the SmartSuite Enterprise Web Client (see the SmartSort System Administration Manual for details). Only Quick Release destinations that have been configured will be visible.

Figure 52: Manual Encoding Method - Quick Release Mode

To encode a bag using a Quick Release Key:

Tap the desired Quick Release button. The bag will leave the Manual Encoding station automatically (the user will not have to select the bag's destination or release the bag from the MEC station). Once released, the MEC will send the bag's destination to the PLC which will route the bag and the bag transaction data will be updated on the SAC.
