Equipment Malfunction Report

The information included in this report only includes all HMI alarms generated as a result of some system malfunction. These include physical device malfunctions, such as bag jams, as well as software malfunctions, such as the host connections becoming disconnected.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Duration Threshold
  • Alarm Type (e.g. Jam, E-Stop, etc.)

Figure 9‑19: Sample Equipment Malfunction Report

The following information is included in this report:

Column

Description

Equipment

The equipment the alarm is associated with.

Alarm

The malfunction that occurred at that device.

Time Set

When the malfunction was first detected.

Time Clear

When the malfunction was resolved. If the malfunction is not resolved as of the time the report was run then this value will always match the time that the report was run.

Duration

The length of the active malfunction (i.e. the difference between Time Set and Time Clear).
