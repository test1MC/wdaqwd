CBIS Bag Volume Report

The CBIS Bag Volume Report provides an overview of the distribution of bags across the various input points. The report also provides information pertaining to the sizing (i.e. In Gauge, Out of Gauge statistics) of the bags entering the CBIS system.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑26: Sample CBIS Bag Volume Report

The following information is included in this report:

Conveyor System Section

Column

Description

Input Conveyor

Identifier of the input conveyor that will ultimately feed into the CBIS system (e.g. Ticket Counter, Curbside, Re-Introduction Lines)

Bag Count

Total number of bags that were seen at the photo eye near the start of the input conveyor (typically near the fire/security door). This count is incremented and maintained in the PLC.

 

Note also that this count may not equal the Total Bags count seen on the CBIS Executive Summary Report (see section 9.23 CBIS Executive Summary Report). This can be caused by several factors including re-induction, tracking issues, jams, etc. The Total Bags on the CBIS Executive Summary Report does not include Out of Gauge bags or other bags which would never be sent through any of the EDS machines.

Percentage

The percentage bags the input conveyor had over the sum of the bags of all input conveyors.

Bag Sizing Section

Column

Description

In Gauge

Bag Count

The number of bags through the dimensioner that were deemed in gauge. This count is incremented and maintained in the PLC based on information received from the dimensioning system.

Percentage

The percentage of all bags through the dimensioner that were in gauge.

Out of Gauge

Bag Count

The number of bags through the dimensioner that were deemed out of gauge. This count is incremented and maintained in the PLC based on information received from the dimensioning system.

Percentage

The percentage of all bags through the dimensioner that were out of gauge.

No Decision

Bag Count

The number of bags through the dimensioner for which no status of the bag sizing was received from the dimensioning system. This count is incremented and maintained in the PLC.

Percentage

The percentage of all bags through the dimensioner that received no decision.

Total

Bag Count

The number of bags through the dimensioner.

 

Note that this count may not equal the Total Bags count seen on the CBIS Executive Summary Report (see section 9.23 CBIS Executive Summary Report). This can be caused by several factors including re-induction, tracking issues, jams, etc. The Total Bags on the CBIS Executive Summary Report does not include Out of Gauge bags or other bags which would never be sent through any of the EDS machines.
