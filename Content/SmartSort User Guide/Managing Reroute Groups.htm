Managing Reroute Groups

Reroute groups can be added, deleted, and modified. Instructions for managing reroute groups are below:

To add a new reroute group:

  1. Click Add New Reroute Group. The Group Details portion of the screen displays default values for all fields.
  1. Enter the description, select the reroute mode (see section 4.4.1 Reroute Modes), configure the Search Window settings (see section 4.4.2 Reroute Search Window), configure the Flight Options (see section 4.4.3 Flight Options), and the Reroute Type Settings (see section 4.4.4 Additional Reroute Settings).
  1. Click Add Reroute Group.

To Edit an existing reroute group:

  1. Select the desired reroute group from the Reroute Group List table. The reroute group details display.
  1. Configure the reroute settings as desired.
  1. Click Save Changes. A confirmation dialog displays.
  2. Click OK to save the changes. Click Cancel to abort the save.

To delete an existing reroute group:

  1. Select the checkbox beside the desired reroute group in the Reroute Group List table. Any flights using this reroute group will have their reroute group set back to default (i.e. no reroute).
  1. Click Delete Selected Reroute Groups. A confirmation dialog displays.
  1. Click OK to delete the reroute group. Click Cancel to abort the delete operation.
