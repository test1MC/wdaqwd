CBRA Baggage Processing Timer Report

The Baggage Processing Timer (BPT) Report provides summary baggage processing statistics related to the CBRA portion of the system. The report is split into two sections: bag type statistics and user-based statistics.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 194: CBRA Bag Processing Timer Report

The following information is included in this report:

Bag Type BPT Statistics

Column

Description
Bag Type The type of bag that was processed in CBRA (Alarmed, Cleared, Unknown, Error, OOG).
Total Bags The total number of bags that were processed in CBRA for each bag type.
CBRA Bag Total % The percentage of bags that were directed to CBRA for each bag type.
Average Time (sec) The average time to process each bag type.
% 0-60 sec The percentage of bags for each bag type that was processed within the allotted time duration.
% 60-120 sec The percentage of bags for each bag type that was processed within the allotted time duration.
% 120-180 sec The percentage of bags for each bag type that was processed within the allotted time duration.
% 180-240 sec The percentage of bags for each bag type that was processed within the allotted time duration.
% 240-300 sec The percentage of bags for each bag type that was processed within the allotted time duration.
% 300+ sec The percentage of bags for each bag type that was processed within the allotted time duration.

BSD BPT Statistics

Row

Description
BSD Location The location of the BSD.

CBRA Bag Total % The percentage of all bags through CBRA that each user processed.
Average Time (sec) The average time spent by each user to process a bag.
% 0-60 sec The percentage of bags processed by each user within the allotted time duration.
% 60-120 sec The percentage of bags processed by each user within the allotted time duration.

% 180-240 sec The percentage of bags processed by each user within the allotted time duration.

% 300+ sec The percentage of bags processed by each user within the allotted time duration.
