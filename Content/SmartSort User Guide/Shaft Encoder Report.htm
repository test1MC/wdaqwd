Shaft Encoder Report

The Shaft Encoder Report includes data on the shaft encoders mounted on conveyors and used for bag tracking by the PLC.

The following selection criteria are available:

  • Subsystem Filters

    •  

    Figure 9‑12: Sample Shaft Encoder Report

    The following information is included in this report:

    Column

    Description
    Encoder Location Location of the Shaft Encoder in the system.
    Current Value

    The total pulses counted in the Pulse Rate Time (See Below for the Pulse Rate Time Constant).
    Expected Value

    The expected total pulses counted in the Pulse Rate Time (Usually set to 1000).
    Deviation  
    Calculated Speed (fpm)
