Alarm Searches and Reports

The Alarm History Search allows operators to search the alarms database. Alarms can be searched by alarm type or by device type. For more information about the Alarm History Search, see section 10.3.3 Alarm History Search.

The Equipment Malfunction Report shows all HMI alarms that were generated as a result of a system malfunction. The data available from this report is also summarized in the Equipment Malfunction Summary Report. For more information about these reports, see sections 9.17 Equipment Malfunction Report and 9.18 Equipment Malfunction Summary Report.
