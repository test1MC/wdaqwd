Alarms Pop-Up

The Alarms pop-up is available by selecting the Alarm icon at the far-right end of the HMI Navigation Bar ( ). In some systems, clicking the Alarm icon toggles the Alarm Banner on/off.

Figure 114: HMI Navigation Bar - Alarm Icon

When the Alarm icon is selected, the Alarm Pop Up displays. The following controls are available on the Alarms Pop Up screen:

Control

Result

Ack Current

Acknowledges the currently selected alarm.

Ack Page

Acknowledges all alarms currently displayed in the Alarm Summary window (the filtered list of alarms).

Ack All

Acknowledges all alarms in the system.
