Template Properties

The following are the properties of a template:

  • Name – Unique name of the template. This field is required.
  • Effective – The date the template becomes effective (the first day when the contents of the flight schedule template will be copied into the active flight schedule). If an effective date is not specified, it is assumed that it is effective until the Discontinue date (if specified). If neither the Effective nor Discontinue date is specified, the template is effective indefinitely.
  • Discontinue – The date the template will be copied into the active flight schedule for the last time (after the specified date, the template will no longer be copied to the active flight schedule). If a discontinue date is not specified, the template is effective indefinitely. Note that flights within templates can each have their own effective and discontinue dates as well.
  • Priority – When the Active Flight Schedule is automatically activated at the day rollover time, only unique flights can be loaded into the schedule. If the same flight number exists in multiple templates that are effective, the priority determines which of these flights is used. Priority 1 is the highest priority. A numeric value that is at least 1 must be used.
  • Enabled – When enabled, the templates flights will be examined as part of the daily automatic activation of the active flight schedule and flights will be copied to the active flight schedule at the specified rollover time.

The following actions can be performed from the Template Management screen:

  • Add Flight Schedule Templates
  • Edit Flight Schedule Templates
  • Duplicate Flight Schedule Templates
  • Delete Flight Schedule Templates
  • Open Flight Schedule Templates to edit flights

Note that duplicating a template also duplicates all the flights contained within the template.

A separate Download button will be available to create a new template and populate it with the current AA flights from FlightHub. The user will be prompted to enter the template details and the configuration of the schedule downloaded when this button is pressed.
