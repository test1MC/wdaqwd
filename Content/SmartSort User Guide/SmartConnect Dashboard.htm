SmartConnect Dashboard

The SmartConnect Dashboard provides an overview of all SmartConnect activity. Status information for each connection (i.e. between BHSs and airline hosts) is displayed, allowing users to view the health of each connection in real-time. The number of messages sent per second is represented graphically on the dashboard.

To access this dashboard, click the Main Menu icon (), select Dashboards, then select Connection Status. Selecting a pipeline from the top of the SmartConnect Dashboard displays each type of connection agent in that pipeline.

Sample Message Broker Dashboard

The dashboard is automatically refreshed every 30 seconds. To manually refresh the dashboard, click the Refresh button (). Users can also hover the cursor over an agent to highlight its message flow, or click and drag agents to move them around on the screen.

Sample Agent Message Flow

Clicking on an agent in the dashboard displays an agent status card at the right-hand side of the screen (see Figure 6 1). Multiple agent cards can be displayed simultaneously, to a maximum of 100 agents. To close an agent status card, click the X in the top-right corner of the card. To close all agent status cards, click Clear.

The following types of information are included in the agent status cards. This information may vary, depending on if the agent is a queueing or processing agent:

  • Connection – The connection status.
  • Messages Received – The number of total messages and rejected messages received.
  • Messages Delivered – The total number of messages delivered successfully.
  • Delivery Time – The average and the maximum time to deliver a single message within the Statistics Interval.
  • Messages Processed – The number of total messages and rejected messages processed within the Statistics Interval.
  • Processing Time – The average and the maximum time to process a single message within the Statistics Interval.
  • Queue Size – The number of messages currently in the queue. To manage the queue size, click Manage Queue (see section 6.2.1).
  • Statistics Interval – The period over which the statistics were gathered.
