Monitoring Communication Status with the HMI

The status of the system PLCs, as well as communications to the various devices in the system, is monitored via the Connection Status screen on the HMI. Clicking the Connection Status button on the Title/Navigation Bar will display this screen.

Figure 7‑26: Connection Status Screen

The Connection Status screen displays internal applications heartbeats, airline host connection status, and other such status indicators. The following topics sections explain the various elements of this screen.
