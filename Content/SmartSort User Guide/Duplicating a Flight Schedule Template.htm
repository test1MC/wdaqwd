Duplicating a Flight Schedule Template

To duplicate (create a copy of) a template, select the Duplicate button for the desired template (located on the far-right side of the row). The Duplicate Template dialog displays. The name of the schedule will be identical to the one that was selected for duplication.

Figure 4‑27: Flight Schedule Template - Duplicate Template Dialog

A new unique template name will be required in the Template Name field. The Save button will only be enabled if all the data is valid and all required fields have data. Once the Save button is clicked, the new Flight Schedule Template will be saved to the server and added to the list. Note that duplicating a template also duplicates all the flights contained within the template.
