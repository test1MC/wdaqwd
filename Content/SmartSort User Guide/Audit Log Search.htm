Audit Log Search

The Audit Log Search screen allows the operator to search user actions in the system. This includes actions such as users logging in and out of applications, users modifying system configurations, and executing searches or reports. Note that the time zone associated with the timestamp returned in the search results is indicated in the top right corner of the screen.

The following search criteria is available if the applicable data is received from the data interfaces:

  • Audit Message (Is In / Not In) – Indicates the message associated with the returned audit log entry.
  • Machine (Is In / Not In) – Indicates the machine associated with the returned audit log entry.
  • Source (Is In / Not In) – Indicates the service associated with the returned audit log entry.
  • Timestamp (Current / Fixed Time Range) – The timeframe used to locate a specific audit log entry.
  • User Name (Is In / Not In) – The user name associated with the returned audit log.
