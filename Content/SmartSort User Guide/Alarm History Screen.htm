Alarm History Screen

All alarms are archived in a history file and may be viewed by selecting Alarm History from the Alarm Summary screen. The Alarm History screen displays all alarms that have been recorded for the last 90 days. This screen (used primarily for troubleshooting purposes) is supplemented by the HMI Alarm Search in the SmartSuite Enterprise Web Client (see section 10.3.3 SmartSort Alarm History Search). Alarm messages are only removed from this screen after the 90-day retention period. Alarm activation events, alarm release events and alarm acknowledge events are all displayed.

The alarm screen allows for the acknowledgement of all current alarms or individual acknowledgement of selected alarms. The main graphic screens are also equipped with an alarm acknowledgement button that can be used acknowledge the most recent unacknowledged alarm.

Figure 116: Alarm History Screen

Users can change the time-period for which the historical alarms display by selecting the appropriate dates from the Start Date and End Date drop-down menus, and by adjusting the time specified in the Start Time and End Time fields.

Right-clicking anywhere on the alarm history screen displays a pop-up with more selections like for printing or viewing alarm history data.
