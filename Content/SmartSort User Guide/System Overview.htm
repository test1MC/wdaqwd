System Overview

This project consists of an upper level controls replacement in Terminal D at Dallas/Forth Worth International Airport (DFW). The upper level controls provide the primary user interfaces to the Baggage Handling System (BHS).

The existing SmartSort software and upper level fault monitoring system was updated to include the new CBIS conveyors and equipment.

The upper level control system is controlled by Brock Solutions' SmartSort system, which consists of redundant sort allocation controllers, data historian servers, and user work stations.

The lower level control system is controlled by Allen-Bradley PLCs programmed with Brock Solutions' standard PLC code. There are four (4) new PLC redundant pairs and fourteen (14) new Motor Control Panels (MCPs). For full details about operating and maintaining the lower-level control systems, refer to the Controls Operations and Maintenance Manual.>
