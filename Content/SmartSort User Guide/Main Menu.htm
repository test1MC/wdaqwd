Main Menu

The Main Menu contains a number of menus that grant the user access to various functions within the SmartSuite Enterprise Web Client.

The items available in the Main Menu are listed below:

  • Settings – Displays the Settings menu when clicked, allowing access to the various configuration screens in the Web Client (see the SmartSort System Administration Manual for details about using the Web Client settings screens to configure the system).
  • Sortation – Displays the Sortation menu when clicked, allowing access to the various flight management screens available in the Web Client. For information about managing flight schedules and configuring virtual to physical Make-Up Unit mapping, refer to section 4 Sortation Control.
  • Dashboards – Displays the Dashboards menu when clicked, allowing access to the various dashboard screens. For information about dashboards, see section 7.4.2 Monitoring Statistical Information with the Web Client.
  • Reports – Displays the Reports menu when clicked, allowing access to the various reports available in the Web Client, as well as the SmartSuite Business Analytics Report Editor. For information about generating reports, see section 9.1 Generating Web Client Reports. For information about the specific reports that are available, refer to sections 9.3 Flight Schedule Report to 9.27 CBRA Performance Report. For information about Business Analytics, refer to section 9.2 SmartSuite Business Analytics Report Editor.
  • Search – Displays the Search menu when clicked, allowing access to the various searches available in the Web Client. For information about performing searches, see section 10.1 Using Global Search. For information about the specific searches available, refer to section 10.3 Using Advanced Searches.
  • Help – Displays the System Help Links screen when clicked, allowing access to links for help with specific areas of the system.
  • User – Displays the Account screen when clicked, allowing the user to change various account settings. For more information, refer to section 2.2.1 Managing User PreferencesAccount.
  • System – Displays the System menu when clicked, allowing access to various parts of the system.
