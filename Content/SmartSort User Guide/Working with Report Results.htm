Working with Report Results

There are several functions available once a report has been generated. If the report has multiple pages, there are controls to go forward, backward, or to a particular page. Note that some of these buttons will be greyed out if they do not apply (e.g. if there is only one page in a report, all of these buttons will be unavailable).

For some reports, the report can also be sorted by a particular column (ascending or descending). To sort the results, click the up/down arrow icon next to a column heading. When the results are being sorted by column, the direction of the sort (ascending or descending) is indicated by an arrow icon.

Figure 9‑1: Report Sorted by a Column Heading

A text search can be performed on the report results by typing search parameters into the Find text box and clicking the Find link. Additional search results can be viewed by clicking the Next button.

Report results can also be refreshed by clicking on the Refresh icon (). Refreshing the report will retrieve any new records that have been added to the database since the report was originally generated. Note: If any report criteria are changed, clicking the refresh icon will not regenerate the report – the Generate button must be pressed to regenerate the report in this case.
