Business Analytics Report Editor

The SmartSuite Business Analytics Report Editor tools allow a user to create custom reports and dashboards. Once created in the Business Analytics Report Editor, any saved reports and dashboards can be accessed on the Web Client. This allows users who are not able to, are not permitted to, or do not want to use the editor, to view the components created in the editor by themselves or by others.

The Report Editor can be accessed by clicking Report Editor in the top-right corner of the Business Analytics screen on the SmartSuite Enterprise Web Client. For information about using the Report Editor, please refer to the SmartSuite Business Analytics Report Editor User Guide.
