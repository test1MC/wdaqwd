Adding a Flight to the Flight Schedule Template

Click Add Flight to display the Add Flight dialog, which allows a flight to be added to the schedule (for information about the flight properties, see section 4.3.4 Flight Properties). Initially the required fields are highlighted in red. If invalid data is entered in any field, the field is also highlighted red.

This dialog is similar to the one available from the Gantt view (see section 4.3.2.4 Gantt View for details). Note that the OK button will only be enabled if all the data on the two tabs is valid. Once the OK button is clicked, the new flight will be added to the grid with a status of NEW, but changes are not saved to the server until Save All is clicked.
