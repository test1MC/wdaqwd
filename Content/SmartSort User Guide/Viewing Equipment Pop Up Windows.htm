Viewing Equipment Pop-Up Windows

The system includes several pop-up windows that are associated with various devices on the graphical HMI screens. Each device automatically changes color depending on its status. Users left click on a particular device to see the device’s pop-up window. The equipment pop-up windows display status information for the associated device.

Note that a Communication Error status typically indicates that the PLC is not communicating with the HMI.

If a user with adequate privileges is signed in, there are also control functions available for certain devices (e.g. Override and Out of Service). For more information on HMI control, refer to section 8 Controlling the System Using the HMI.
