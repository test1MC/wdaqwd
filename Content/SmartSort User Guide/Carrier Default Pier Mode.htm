Carrier Default Pier Mode

The user can choose to send a bag to the default pier for a carrier. The default piers are configured using the SmartSuite Enterprise Web Client (see SmartSort System Administration Manual Chapter 2: SmartSuite Enterprise Web Client Administration). A list of the carriers that have been configured with default piers will be displayed on the screen. All carriers may not be displayed on the screen at once. However, the list of carriers can be filtered by carrier name and carrier code. The user can also browse through the list of carriers. This table can be sorted similar to that of flight mode (see section 5.2.3 Flight Mode).

Figure 5‑7: Manual Encoding Method - Carrier Mode

To encode a bag using Carrier Default Mode:

  1. Tap Carrier. When Carrier mode is selected, the button will be light blue.
  1. Select the desired carrier from the carrier list. Note that the list will only display carriers for which carrier default piers have been assigned. The carriers that display can be filtered by carrier code or carrier name using the on-screen keyboard.
  1. Tap Release. Once released, the MEC will send the bag's destination to the PLC which will route the bag and the bag transaction data will be updated on the SAC.
