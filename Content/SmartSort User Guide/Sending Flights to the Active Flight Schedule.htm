Sending Flights to the Active Flight Schedule

The Send to Active function will copy any of the selected flights in the template to the active flight schedule. Important notes:

  • If the same flight already exists in the active flight schedule, it will be overwritten by the new flight.
  • Flights will be copied to the active flight schedule regardless of the effective and discontinue dates as well as the days of the week the flight operates on.
  • If there are multiple flights with the same flight number selected, only the last flight in the selection will be copied to the active flight schedule.

Clicking Send to Active will display a confirmation dialog. If multiple flights were selected, the flight numbers can be displayed or hidden by clicking Show All Affected Flights/Hide All Affected Flights. Once the Yes button is selected, the flights will be copied to the active flight schedule and saved to the server immediately. Note that flights cannot be send to the active schedule if there are pending changes that have not been saved on the Flight Schedule Template screen.
