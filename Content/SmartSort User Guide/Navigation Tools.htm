Navigation Tools

The BHS Map offers perspective in three dimensions and allows the user to control the angle and depth of the view. Navigation tools are in the lower right corner of the screen.

 

Navigation Tools

Description

Compass

Allows users to rotate the system counter-clockwise by 90-degree increments.

Home

Allows users to return the system to its original orientation.

Zoom In/Out

 Allows users to zoom in/out.
 

Right-Click and Drag

Allows users to rotate the system on a 3-D level.
 

Left-Click and Drag

Allows users to pan in all directions on a 2-D level.
 

Mouse Scroll

 Allows users to zoom in/out.
