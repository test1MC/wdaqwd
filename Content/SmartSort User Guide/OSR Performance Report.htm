OSR Performance Report

The OSR Performance Report provides a summary of statistics related to the CBIS portion of the system. The faults, failures, and usage statistics related to the Level 2 Screening System (also referred to as On-Screen Resolution, or OSR) are included in this report.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Interval (minutes, hours, days, weeks, months, or years)

The following information is included in this report:

Note that there is a summary section containing the totals for each ISD, followed by a table showing the results for each individual ISD at the specified interval.

Column

Description

Clear Bags

 Total number of OSR (Level 1) clear decisions made by a user for the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

Clear %

 The percentage of OSR Total Decisions that received an OSR (Level 2) clear decision from that ISD machine.
Alarm Bags Total number of OSR (Level 1) alarm decisions made by a user for the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).
Alarm % The percentage of OSR Total Decisions that received an OSR (Level 2) alarm decision from that ISD machine.
Error Bags Total number of machine (Level 1) error decisions made by a user for the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).
Error % The percentage of OSR Total Decisions that received an OSR (Level 2) error decision from that ISD machine.
No Decision Bags Total number of OSR (Level 2) bags for which no decision was received. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).
No Decision % The percentage of OSR total bags that did not receive an OSR (Level 1) decision.
Total Bags The total number of OSR (Level 2) decisions made by a user for that ISD machine.
Average Processing Time The amount of time required on average for the user to reach a level 2 decision. Measured in seconds.
