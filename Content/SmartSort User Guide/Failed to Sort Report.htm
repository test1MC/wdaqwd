Failed to Sort Report

The Failed to Sort Report provides statistics for bags that could not be diverted to their sort destinations.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Mainline

Figure 9‑13: Sample Failed to Sort Report

The following information is included in this report:

The information included in this report only includes bags which failed on a mainline. Possible scenarios are Lost Tracking, Failed to Push, and Make-Up Unit not Available.

Column

Description

Timestamp

When the event happened.

Tag Number

The tracked tag value or BHS ID.

Mainline

The mainline where the event occurred.

Device

The device where the event occurred.

Failure

The type of failure (i.e. Lost Tracking, Failed to Push, Make-Up Unit not Available, etc.)

Reason

The reason the PLC gave for the failure.
