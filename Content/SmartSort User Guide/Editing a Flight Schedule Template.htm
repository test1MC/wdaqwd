Editing a Flight Schedule Template

To edit a template, click Edit for the desired template (located on the far-right side of the row). The Edit Template dialog displays.

Figure 4‑26: Flight Schedule Template - Edit Template Dialog

The Save button will only be enabled if all the data is valid and all required fields have data. Once the Save button is clicked, the flight schedule template will be saved to the server and updated in the list.
