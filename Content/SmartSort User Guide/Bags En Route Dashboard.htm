Bags En Route Dashboard

Real-time baggage delivery statistics can be monitored from the Bags En Route Dashboard. This dashboard displays how many bags have been delivered and are currently en route in the BHS for each flight.

The Bags En Route Dashboard is accessed by selecting Dashboards from the Main Menu.

Figure 7‑35: Bags En Route Dashboard

The dashboard refreshes automatically every 15 seconds. The Make-Up Unit that is currently active for each flight is highlighted in green.

The following information is displayed on the dashboard for each flight:

Airline Logo

  • Airline – The associated airline.
  • Flight # – The flight number.
  • Route – The destination city airport code, or multiple destination airports if the flight is multi-leg.
  • Gate – The gate the flight is departing from (if available).
  • Scheduled Departure – The date and time at which the flight was originally scheduled to depart in local time.
  • Estimated Departure – The date and time at which the flight is currently expected to depart in local time.
  • On Time Pier – The pier that on time bags are sent to.
  • Late Pier – The pier that late bags are sent to.
  • Bags Seen – The number of bags for the flight that have been scanned at an ATR or MEC in the BHS.
  • Bags Sorted – The number of bags that are no longer in the BHS because they have been sorted to a Make-Up Unit.
  • Bags in System – The number of bags for the flight that are still in the BHS. A bag is considered to be en route if it has been seen at an ATR or MEC, but has not yet been sorted to a pier. If bags are scanned in the system, but not sorted by tag number, they may not be removed from this count.
