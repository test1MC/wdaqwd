Day End Report

The Day End Report provides an operational summary for the system. The data in the report is a summary of the information provided in the Throughput Report, Laser Array Report, and the Tracking Photo Eyes section of the Equipment Operational Summary Report.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Subsystem Filters

The following information is included in this report:

The information included in this report only includes specifically configured counters (generally throughput counters but others are possible). It is meant to give a quick at-a-glance overview of the system performance.

The counters shown on this report are also found on other reports such as the Throughput and Laser Array Reports.

Row

Description

Device

The device the stats are associated with.

Stat(s)
