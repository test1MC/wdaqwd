Saving Changes to the Active Flight Schedule

Clicking Save All saves all changes to the server. None of the actions performed in the grid or commands performed will be saved to the server until Save All is clicked. Note that any rows with a Status of INVALID will not be saved until the row is corrected. Once the save is complete, the Status of all the saved rows will change to SAVED.
