SmartSort Default Make-Up Report

The SmartSort Default Make-Up Report provides detailed bag counts for the various reason why bags have gone to the default make-up unit.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑11: SmartSort Default Make-Up Report

The following information is included in this report:

Column

Description

Verified Diverts

Number of bags that were assigned and diverted the default MU.

Close to Bag

Number of bags that were routed to the MU because they were too close to the bag to an adjacent bag with the MU set as the sort destination.

Failed to Divert

Number of bags that failed to divert to the MU.

E-Stopped

Number of bags that were sent to the MU because the destination MU was E-Stopped.

Divert All

Number of bags sent to the MU because Divert All was enabled.

Unknown

Number of Unknown bags that were sent to the MU.
