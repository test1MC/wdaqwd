Understanding Web Client Reports

The report screens in the Web Client allow an operator to generate reports with baggage, flight, and alarm records. The data in these reports can be used to identify fault trends and assess system performance.

Available Reports

The following reports are available in the Web Client:

General Reports

  • Business Analytics (see section 9.2 SmartSuite Business Analytics Report Editor )

SmartSort Sortation Reports

  • Flight Schedule Report (see section 9.3 Flight Schedule Report )
  • Flight Sort Summary Report (see section 9.4 Flight Sort Summary Report )
  • Individual Flight Summary Report (see section 9.5 Individual Flight Summary Report )
