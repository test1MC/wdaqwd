Device Status

The following type of status and control pop-up windows can be accessed via the Device Status menu:

All Devices

Selecting a device type (i.e. ATR, MU, HSD, EDS, etc.) from the Device Status menu displays a pop-up window with the status, in both color and textual format, of all applicable system devices for the specified device type. For more information on device status, see section 7.1.2 Viewing Equipment Pop-Up Windows.
