Duplicating a Flight in the Flight Schedule Template

Clicking Duplicate will duplicate (copy) any selected rows and insert new rows into the grid. All of the flight details, including the flight number, are copied when a flight is duplicated. The status will be NEW, but changes are not saved to the server until Save All is clicked.
