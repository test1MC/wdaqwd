Alarm History Search

The Alarm History Search screen allows the operator to search the alarm history recorded by the HMI.

The following search criteria is available if the applicable data is received from the data interfaces:

  • Alarm (Is In / Not In) – Indicates the alarm name associated with the returned alarm history entry.
  • Alarm Description (Contains / Is / Is Not / Begins With / Ends With) – Indicates which keywords should be used/omitted in order to refine the search results.
  • Conveyor (Is In / Not In) – Indicates which conveyor is associated with the returned alarm history entry.
  • Line (Is In / Not In) – Indicates which line is associated with the returned alarm history entry.
  • PLC (Is In / Not In) – Indicates which PLC is associated with the returned alarm history entry.
  • Server (Is In / Not In) – Indicates which server is associated with the returned alarm history entry.
  • Severity (Is In / Not In) – Indicates the severity of the alarm associated with the returned alarm history entry.
  • Tag Name (Is In / Not In) – Indicates the system tag name associated with the returned alarm history entry.
  • Timestamp (Current / Fixed) – The time frame used to locate a specific activity log entry.
  • Transaction (Is In / Not In) – Indicates how the transaction associated with the returned alarm history entry is logged.
