CBRA Performance Report

The CBRA Performance Report provides a summary of statistics related to the Checked Bag Resolution Area (CBRA) portion of the system. The faults, failures, and usage related to the CBRA are included in this report.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Interval (Minutes, Hours, Days, Weeks, Months, or Years)

CBIS Throughput Section

Column

Description

Clear

Total number of clear bags arriving in CBRA on that alarm line. Possible causes for this could be bags that failed to divert on a clear bag line.

Alarm

Total number of alarm bags arriving in CBRA on that alarm line.

EDS Error

Total number of error bags arriving in CBRA on that alarm line.
EDS Flush Total number of flushed bags arriving in CBRA on that alarm line. Flushed bags are bags that were processed by an EDS machine in bypass mode.

Timeout

Total number of no decision bags arriving in CBRA on that alarm line. No decision means that no decision from the EDS machine was received.

Lost in Tracking

Total number of unknown bags arriving in CBRA on that alarm line. Unknown bags are those bags which were lost in tracking between the exit of the EDS machine and entering CBRA.

Out of Gauge

Total number of out of gauge bags arriving in CBRA on that alarm line.

BHS Purged

Total number of purged bags arriving in CBRA on that alarm line.
Comm Error Total number of communication error bags arriving in CBRA on that alarm line. Communication error bags are ones that did not receive any decision from the EDS machine.
Stopped in Scan Total number of stopped in scan bags arriving in CBRA on that alarm line. Stopped in scan bags are those which were inside an EDS machine when the line was stopped, and a full scan of the bag was not completed.

Total

The total number of bags arriving in CBRA on that alarm line.
