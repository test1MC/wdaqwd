Carrier Summary Report

The Carrier Report provides a summary of bags and BSMs seen in the sort system grouped by carrier.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑10: Sample Carrier Summary Report

The following information is included in this report:

The information included in this report only includes outbound bags for which a BSM has been received within the search time. The report is sorted by Carrier Code.

Clicking a Carrier Code causes the report to drill down into transactions for that carrier during the searched timeframe.

Column

Description

Carrier Code

The IATA carrier code.

Airline

The name of the airline.
Default Sort Destination(s) The default sort destination(s) associated with the airline.

BSMs Received

The number of BSMs received for the carrier, for outbound bags, within the search time frame.
Bags Seen The number of bag tags that were seen somewhere in the system.
Bags Sorted The number of bag tags that reported arriving at their sort destination.
Bags With No BSMs The number of bag tags for which no BSM was received.
