List View

One of three possible views for the Active Flight Schedule screen, the List view allows users to view the flights in a table format and provides no editing capabilities. The data can be sorted (for example, by Flight Number, On Time Make-Up Units, etc.) and also filtered on flight properties (for information about how to use the display filters, see section 4.3.2.2 Modifying Display Settings).

Figure 4‑11: Active Flight Schedule - List View

All the flight properties as described in section 4.3.4 Flight Properties are displayed in the list. The contents of the list can be sorted by any of the properties by clicking on the column header. The following can be noted about the contents of the list:

  • An icon of the Airline operating the flight will be displayed on the far left of the flight row if the Airline has been configured with an icon. Icons can be added through the Airline Codes setting screen. See The SmartSort System Administration Manual for details.
  • The Make-Up Unit to which the bags will be sorted at the current time is displayed in bold. For example, if a bag is in the Early time period and an Early Make-Up Unit is configured, the Make-Up Unit(s) configured in the Early Make-Up Units column will be displayed in bold. If no early time period is configured, the On Time Make-Up Unit(s) configured in the On Time Make-Up Units column will be displayed in bold.
  • The time period for each flight will be displayed in bold when the current time is within that time period. For example, if the open time for a flight is configured for 10:30 and the current time is 10:48, the Open Time will be displayed in bold.
  • The list may not show all flights by default. Scroll to the bottom of the screen to the number of pages displaying flights (i.e. "Showing X of Y records"). To the right of that, if more records are available, a button is displayed that can be selected to show more records. Note that any filtering or sorting takes into account all records and not just those being displayed.
