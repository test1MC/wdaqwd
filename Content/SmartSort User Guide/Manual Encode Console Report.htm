Manual Encode Console Report

The Manual Encode Console Report provides a summary of bags processed at each Manual Encode Station.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑23: Sample Manual Encode Console Report

The following information is included in this report:

Note that there are two sections to this report. The first deals with Bag Dispatch Statistics, which includes counts of how things were sorted at the MEC. The second breaks down encodings by carrier, assuming that the bag was sorted via a method that associates it with a carrier.

The report allows the user to drill down into individual Bag Transactions by clicking on certain values in the Tag Statistics section (such as Flight, Virtual Make-Up Unit and Carrier).

Bag Dispatch Statistics section:

Row

Description

Bag Tag (BSM)

The number of bags sorted where the operator scanned a valid sortable IATA tag which the system had a BSM for.

Keyed Barcode

The number of bags sorted where the operator used the keyboard to type the barcode data.

Other Tag Types

The number of bags sorted using specific use tags such as IATA Fallback and pier tags.

Flight

The number of bags manually sorted using the flight sort option.

Virtual Pier

The number of bags manually sorted using the virtual pier sort option.

Carrier

The number of bags manually sorted using the carrier sort option.

Carrier Statistics section:

Row

Description

Configured Carrier

The number of bags sorted via a method that associated it with the configured carrier.

Only configured carriers will appear in this list, and only if they have seen bags at an MEC.
