Gantt View

One of three possible views for the Active Flight Schedule screen, the Gantt view provides a graphical representation (by Make-Up Unit/make-up) of the flights allocated to those Make-Up Units by time. Editing of individual flights is available on the Gantt view.

The following is a sample of what the Gantt view looks like:

Figure 4‑12: Active Flight Schedule - Gantt View

The Gantt display has one horizontal segment to represent each sort destination (Make-Up Unit). The Gantt Flight Schedule screen displays several Flight Bars. Each Flight Bar represents one flight Make-Up Unit assignment. The length of the bar is determined by the open and departure times of the flight.

There is a Time axis displayed along the top of the Gantt display. Flights are placed on their associated virtual Make-Up Units in relation to the Time axis. There is also a vertical black line that represents the current time.

Any flights that are not assigned to a sort destination will be displayed at the top of the Gantt display in the None row.

Figure 4‑13: Active Flight Schedule - Gantt View Unassigned Flights

Flight Bars

Basic flight details are available on each flight bar including:

  • Airline icon (if configured for the Airline)
  • Airline code
  • Flight Number
  • Flight Route
  • Status (Cancelled or On Hold)
  • Early, Late, and Missed Make-Up Unit Assignments (if specified)
  • Estimated Departure (represented by the plane icon)

Flight bars can have one of the following colors:

  • Gray – The flight has been cancelled
  • Yellow – The flight is on hold
  • Orange – The flight does not have the required On Time Make-Up Unit assignment
  • Blue – None of the above are true (i.e. the flight is "normal")

Sort Destination Rows

Each sort destination row is initially collapsed and has bars with shades of green throughout the time range. The green bars represent when flights are open. The darker the shade of green and the taller the green bar, the more flights there are that are open during that time. The following shows a sample initial state of the Gantt view when loaded.

Figure 4‑15: Active Flight Schedule - Gantt View Sort Destination Rows

It can be seen that SP2 is expected to have more flights between 14:00 and 15:00 than any other time of the day.

Each row (Make-Up Unit) can be expanded or collapsed individually by clicking on the arrow beside the row name. All the rows can be expanded or collapsed together by clicking on Expand All/Collapse All in the upper left corner.

Additional Flight Details

More flight details can be seen by hovering over a flight bar. A tooltip will appear containing the following information:

  • Flight (Airline Code and Flight Number)
  • Route
  • Gate
  • Aircraft Type
  • On Time Make-Up Unit(s)
  • Early Make-Up Unit(s)
  • Late Make-Up Unit(s)
  • Missed Make-Up Unit(s)
  • Open Time
  • Late Time
  • Close Time
  • Estimated Departure Date/Time
  • Scheduled Departure Date/Time

Editing a Flight

Individual flights can be edited by clicking on a flight bar. The Flight Edit dialog is displayed when a flight bar is clicked. The flight properties that can be edited are described in section 4.3.4 Flight Properties. There are two tabs on the window; one displays general properties of the flight and the other displays the Make-Up Unit assignments for the flight.

Figure 4‑16: Active Flight Schedule – Flight Edit Dialog: General Properties Tab

Figure 4‑17: Active Flight Schedule - Flight Edit Dialog: Make-Up Unit Assignments Tab

Make-Up Unit assignments can be added or deleted by clicking the Add () or Delete () button respectively.

The OK button will only be enabled if all the data on the two tabs is valid. Once the OK button is clicked, the changes are automatically saved.
