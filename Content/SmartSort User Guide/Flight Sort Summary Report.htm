Flight Sort Summary Report

The Flight Summary Report provides a summary of the bags processed by the system for each flight.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑6: Sample Flight Sort Summary Report

The following information is included in this report:

Column

Description

Flight

The flight identifier (i.e. DL1234).

Early Bags

The number of bags destined to the flight that were seen in the system early.

On Time Bags

The number of bags destined to the flight that were seen in the system on time.

Late Bags

The number of bags destined to the flight that were seen in the system late.

Missed Bags

The number of bags destined to the flight that were seen in the system after the flight has departed.

ATR Sorted

The number of bags which were automatically sorted by the system (without any operator input).

MEC Sorted

The number of bags which were manually sorted at the Manual Encode Console (with operator input).

Total Bags

The total number of bags for the flight that were seen in the system.
