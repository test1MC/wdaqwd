Network Status Report

This report provides a list of IP addresses as well as the current status of servers and PLC equipment used in the BHS.

There are no selection criteria for this report. It is generated from current system data.

Figure 180: Sample Network Status Report

The following data is available on the report:

Column

Description

Computer or Device Name

Name of the device from which the data is being reported.

Status

Status of the device (Offline, Active, Primary, Secondary).

IP Address or Network Path

IP Address or Network Path of the device.
