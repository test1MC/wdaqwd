Adding a New Flight Schedule Template

To add a new template, click New on the upper left corner of the screen. The New Template dialog displays.

All required or invalid fields are highlighted in red. The Save button will only be enabled if all the data is valid and all required fields have data. Once the Save button is clicked, the new flight schedule template will be saved to the server and added to the list.
