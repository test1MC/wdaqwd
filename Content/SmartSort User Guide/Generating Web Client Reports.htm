Generating Web Client Reports

The report screens allow an operator to query the DHS database for baggage, flight, and alarm records that meet a given set of criteria. The data is returned as a pre-formatted report.

Each of the Reports screens behaves in the same manner: report criteria are selected, and the report is generated. A time and date range is required for most reports. Some reports have additional report criteria that can be selected.

Once a report is generated, the report results can be printed, or exported to file (CSV, PDF, Excel). The report criteria can also be modified and a new report can be generated at any time.

Each Report in the Web Client has a Report Criteria screen, from which the various criteria defining the search are selected. For most reports, only a time/date range is required. However, some reports have additional criteria such as intervals, equipment type, etc.

Once the Report Criteria have been selected, click Generate to generate the report. The report will be displayed on the right side of the screen.

Reports can be generated in report viewer format, or in PDF format. By default, the report will be displayed in the Report Viewer format. Results from the Report Viewer can be sorted (see section 9.1.1 Working with Report Results) or printed/exported (see section 9.1.2 Printing/Exporting Reports).

Clicking the PDF radio button will generate the report results in a PDF format and open the results in Acrobat Reader. The results will still be displayed in the same Web Client window, but the options to print, save, and navigate are controlled through the version of Acrobat installed on the computer where the report was executed.

To generate a report:

  1. Select the desired report from the Reports menu.
  1. Enter the report criteria.
  2. Most reports only require a time and date range; however, some reports require other criteria. Each report screen will indicate which criteria are optional. Attempting to generate a report without one of mandatory criteria will return an error message.
  3. Note: When generating multiple reports for a shift or time period, it can be helpful to use the same time and date criteria that was used for the most recently generated report. To use the previous report's time and date criteria, click the Use Previous time and date range link below the other date criteria fields. The Use Previous link displays for any report that uses a date range or a date time range.
  1. Click Generate. The report will be displayed in the report viewer.
  2. The navigation bar at the top of the report allows the user to navigate through the report, search for specific terms, export the results (CSV, PDF, and Excel) and refresh the report with new records (if available).
  3. If the report is generated as a PDF, the report can be printed, or exported to file (PDF).
  4. The report criteria can be modified and a new report can be generated at any time.

The sections later in this chapter describe each of the reports provided with the system. Each section lists the parameters that can be used to filter the reports and the results with details of each column or section.
