CBRA Executive Summary Report

This report provides summary statistics related to the CBRA portion of the system.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 192: Sample CBRA Executive Summary Report

The following data is available on the report (data is available for each CBRA):

Summary

Column

Description

CBRA Location

 Location of the CBRA in the system.

Bag Type

Rows

Description
Cleared Total number of clear bags arriving in CBRA on that alarm line.
Alarmed Total number of alarm bags arriving in CBRA in that alarm line.
EDS Error Total number of EDS error bags arriving in CBRA on that alarm line.
EDS Flush Total number of flushed bags arriving in CBRA on that alarm line.
No Decision Total number of no decision bags arriving in CBRA on that alarm line.
Lost in Tracking Total number of unknown bags arriving in CBRA on that alarm line.
Out of Gauge Total number of out of gauge bags arriving in CBRA on that alarm line.
BHS Purged Total number of purged bags arriving in CBRA on that alarm line.
Comm Error Total number of communication error bags arriving in CBRA on that alarm line.
Stopped in Scan Total number of stopped in scan bags arriving in CBRA on that alarm line.
Length Change Purge A BHS purge that occurs when the length of the bag changes from the last time it was mentioned.
Unknown Proximity Purge A BHS purge that occurs when an unknown bag is discovered too close beside a tracked bag such that it is uncertain which bag was being tracked.
Total The total number of tags arriving in CBRA on that alarm line.
