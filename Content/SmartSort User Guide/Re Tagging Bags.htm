Re-Tagging Bags

Re-tagging bags reduces the number of bags that are sorted to the correct make-up unit with outdated human readable information on the bag tag. It is meant to reduce the number of bags where the baggage handler may question whether the bag is at the correct location.

Bags will be sent to re-tagging stations in the following scenarios:

  • Flag is set in BSM indicating the bag needs to be re-tagged (.E/RUSH element in a SABRE BSM)
  • Flight has changed since the initial BSM (either flight number or destination)
  • Bag is being rerouted by Dynamic Reroute (see section 4.4.1 Reroute Modes)

Bags requiring a re-tag will be sent to a manual re-tagging station consisting of an MEC with a DYMO Printer.

To re-tag a bag using a re-tagging station:

  1. Scan the bag tag using the MEC hand scanner. Instead of the bag being released to its destination, it will remain at the MEC so that a new overlay can be applied.
  1. Once scanned, a new overlay will print and a pop-up message with instructions will appear on the MEC screen.
  1. Follow the instructions on the MEC screen to apply the new overlay on the bag tag. If the tag did not print or was damaged, the Reprint button can be used to make another tag.
  1. Once the new overlay has been applied, press the Done button on the pop-up. This will release the bag and it will sort to its destination.
