BIT Hand Scanners

Each BIT station is equipped with a hand scanner mounted at the inspection table. This hand scanner is used to display bag images on the SVS computer. There are two modes for the hand scanner to be in:

  • Prompt Mode – The user is prompted to scan the bag tag, and the system checks its information for the tag number.
  • Ad Hoc Mode – At any other time, the hand scanner data just passes through to the SVS, with no lookup being performed.

When a BHS unknown bag arrives in CBRA and is transferred to the BIT, the user will be prompted to scan the IATA tag before tapping BAIT.

If security information is available for that bag tag, the display will update to show the correct status (clear, alarm, etc.) and it can be BAITed as usual. If no information is found, the status will remain BHS Unknown and the scan will have to be skipped as noted above.

When the hand scanner finds information about the bag, the data that is sent to the SVS is the pseudo ID (eight digits). If this does not retrieve the correct image, the IATA tag can be scanned again (no longer in prompt mode), and the 10-digit tag will be passed to the SVS.
