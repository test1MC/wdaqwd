Editing a Flight in the Flight Schedule Template

Clicking Edit Flight opens the Edit Flight dialog will be open for whichever flight is associated with the currently selected cell. This is almost identical to the Add Flight dialog described above. Once OK is clicked, the grid will reflect the changes and the status of the flight will become MODIFIED, but changes are not saved to the server until the Save All button is clicked.
