Line Switching

Some systems have multiple MEC stations in the same vicinity. In places where two stations are physically close to each other, a single MEC station will be configured to operate multiple MEC positions. The current location of the station is shown at the top of the Action Log. The key-switch button lets the user switch between the two different positions.
