Dimensioner Pop-Up Window

The Dimensioner pop-up window contains:

  • Current Status of the Dimensioner
  • Number of In Gauge Bags
  • Number of Out of Gauge Bags
  • Number of No Decision Bags
  • Number of Oversize Bags
  • Number of Errored Bags
  • Total Number of Bags
