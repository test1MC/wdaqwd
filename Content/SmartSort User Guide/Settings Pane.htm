Settings Pane
  • System Status – Allows users to turn enable or disable the System Status function. When System Status is 'enabled, users can access either Mechanical or Operational System Status.
  • Mechanical System Status – Displays the Running/Faulted State of each segment (similar to what is seen in the SmartVision HMI application). Mechanical Status shows segment status according to the following legend:

Figure 131: BHS Map - Mechanical System Status Legend

  • Operation System Status – Displays an indication of how well bags are flowing through segments based on averaged data from the last few minutes of Mechanical System Status (like traffic maps). Typically, the data is sampled (for run/stop status) every 30 seconds over the last 15 minutes. Note that an idle status is considered running since when in an Idle state, the system should be capable of running and processing bags. Operational Status shows segment status according to the following legend:

Figure 132: BHS Map - Operational System Status Legend

  • Live Bags – Allows users to enable or disable Live Bags to. By default, this setting is enabled. Note that since this system runs in real time, no bags will display if this setting is disabled.
  • Bag Modes – Allows users to choose how to filter bags in the system.
  • Normal – Displays bags as Tracked, Untracked, or Uncategorizable. Uncategorizable bags are bags where information about them is unknown, e.g. No Read, a bag that was previously lost in tracking and re-acquired, etc. Normal Legend displays:

Figure 133: BHS Map - Normal Bag Mode Legend

  • Time to Departure – Displays bags as Early, On-Time, Late, Missed, Uncategorizable, and Untracked. These statuses are based on flight times as listed in the flight schedule. Time to departure Legend displays:

Figure 134: BHS Map - Time to Departure Legend

  • Time in System – Displays bags as Good, Warning, Bad, Uncategorizable, and Untracked. These statuses are based on system settings. By default, the system settings are set to 10 - 15 minutes in system. Users can change these settings by dragging the yellow and red bag to the desired time frames (see Figure 136). Time in System Legend displays:

Figure 135: BHS Map - Time in System Legend

Figure 136: BHS - Time in System Settings

  • Security Status – Displays bags as Alarmed, Cleared, EDS Error, Pending Decision, Timeout, Out of Gauge, BHS Unknown, Communication Error, Uncategorizable, and Unknown. Uncategorizable are bags where the security status of the bag is not known. All bags prior to the ISD machines would be considered Uncategorizable. Note: only authorized users will have access. Security Status Legend displays:

Figure 137: BHS Map - Security Status Legend

  • View Exception Bags – Allows user to enable or disable pins for Hot Bags, Rejected Bags, Goal Missed Bags, and Cold Bags on the map. When enabled, only one pin is displayed for each exception bag.
  • Key Performance Indicators – Allows user to show or hide KPIs in the system. By enabling KPIs, KPI icons displays near devices, allowing users to view statistics headlines about system performance. Clicking on a headline displays detail cards for the selected KPI.

Figure 138: KPI Headlines

  • Labels – Allows users to turn enable labels in the system. These will typically identify devices in the system such as piers, make-up units, ATRs, ISD machines, etc.

Figure 139: BHS Map - Device Labels On
