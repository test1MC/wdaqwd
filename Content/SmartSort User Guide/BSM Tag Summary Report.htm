BSM Tag Summary Report

The BSM Tag Summary Report provides a listing of all bag tags seen in the system during a given operational period.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Flight
  • Bag Tag

Figure 9‑9: Sample BSM Summary Report

The following information is included in this report:

Column

Description

Bag Tag

The bag tag number.

Type

The bag type: (L)ocal or (T)ransfer.

Flight

The bag's flight number.

Dest.

The bag's flight destination.

BSM Received

The time and date the BSM was received.

Passenger

The passenger's name.

Time Sts.

The time status based on when the BSM was received (Early, On Time, Late, etc.).

Trans. Time

The time the transaction was logged to the database.

Device

The device that interacted with the bag (scanner, photo eye, etc.).

Event

The event that occurred (e.g. bag tag scanned, bag sorted, bag entered CBIS, etc.).
