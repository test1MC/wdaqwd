Bags Pane

When a user clicks on a bag in the system, the Bags pane expands. Each selected bag has a unique Bag Details Card, which includes the Tag Number, Security Status, Flight Number, Time in System, and Assigned Pier.

Note: The values included on the Bag Details Card will only display if the information exists an is known. Security Status will only display if the user has adequate security privileges. The appearance of the Bag Details Card may change as the bag progresses through the system, depending on if it is located on a tracked or untracked segment.

The user can navigate to the bag by clicking the Locate button on the Bag Details Card (see Figure 141). This feature zooms in on the segment where the bag is located.

When a user clicks on the Tag Number, the Bag Details screen displays in a new tab (see section 10.2 Viewing Details Screens). From there, when a user clicks View Detailed Transactions, the Bag Transaction Details screen displays (see section 10.3.5 Bag Transaction Search).
