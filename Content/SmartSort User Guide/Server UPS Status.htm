Server UPS Status

The UPS status section displays the status of each UPS in the system.

Figure 7‑30: Connection Status Screen - UPS Section

The following information is available for each UPS:

  • Power Failure – Indicates whether or not the UPS is functioning properly. If this indicator is red, there has been a fault and the UPS is on battery power.
  • Load % – The load percent indicates the percentage use of the UPS capacity.
  • Battery Remaining (min) – The remaining time before the battery will run out of power (in minutes).
