CBIS Executive Summary Report

The CBIS Executive Summary Report provides an overview of the performance of the CBIS system based on messages from bags being screened.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑25: Sample CBIS Executive Summary Report

The following information is included in this report:

Column

Description

EDS Machine

Identifier of the ISD machine to which the data in that row applies.

Total Bags

Total number of bags that were seen at the photo eye prior to handing the bag over to the ISD machine. This count is incremented and maintained in the PLC.

 

Note that this total may not equal the sum of Machine Total Decisions. For example, if an ISD machine loses a bag, two decisions may result; flushed bags do not result in a decision; and the entrance PE may get flagged more than the number of decisions.

 

Also note that this count may not equal the Total Bags count seen on the Input Conveyors section of the CBIS Bag Volume Report (see section 9.24 CBIS Bag Volume Report). This is caused by a few factors including bags that are re-inducted, tracking issues, jams, bags manually moved from one conveyor to another, etc. Additionally, the Total Bags on the Bag Volume Report would include all Out of Gauge or other bags which would never be sent through any of the ISD machines.

Machine Decisions – Total

Total number of machine (Level 1) decisions made by that ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

 

Note that this total may not equal the sum of Machine Cleared and Machine Alarmed decisions. This is because this report does not include every type of decision (i.e. Other and Unknown decisions are not included). For more detailed information, refer to the ISD Performance Report (see section 9.26 ISD Performance Report), which includes columns for each type of decision.

 

Also note that any interruptions in the communication between the ISD machine and the PLC, or, the PLC and the upper level may cause this count to appear different than those reports produced by the ISD.

Machine Decisions – Cleared

Total number of machine (Level 1) clear decisions made by the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

Machine Decisions – % Clear

The percentage of Machine Total Decisions that received a machine (Level 1) clear decision from that ISD machine.

 

Note that the percentage of Machine Clear Decisions and Machine Alarm Decisions may not sum to 100%. This is because this report does not include every type of decision (i.e. Other and Unknown decisions are not included). For more detailed information, refer to the ISD Performance Report (see section 9.26 ISD Performance Report) which includes columns for each type of decision.

Machine Decisions – Alarmed

Total number of machine (Level 1) alarm decisions made by the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

Machine Decisions – % Alarm

The percentage of Machine Total Decisions that received a machine (Level 1) alarm decision from that ISD machine.

 

Note that the percentage of Machine Clear Decisions and Machine Alarm Decisions may not sum to 100%. This is because this report does not include every type of decision (i.e. Other and Unknown decisions are not included). For more detailed information, refer to the ISD Performance Report (see section 9.26 ISD Performance Report) which includes columns for each type of decision.

OSR Decisions – Total

Total number of OSR (Level 2) decisions made by an operator for the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

 

Note that this total may not equal the sum of OSR Cleared and OSR Alarmed decisions. This is because this report does not include every type of decision (i.e. Other and Unknown decisions are not included).

 

Note also that any interruptions in the communication between the ISD machine and the PLC, or, the PLC and the upper level may cause this count to appear different than those reports produced by the ISD.

OSR Decisions – Cleared

Total number of OSR (Level 2) clear decisions made by the operator for the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

OSR Decisions –
% Clear

The percentage of OSR Total Decisions that received an OSR (Level 2) clear decision from that ISD machine.

 

Note that the percentage of OSR Clear Decisions and OSR Alarm Decisions may not sum to 100%. This is because this report does not include every type of decision (i.e. Other and Unknown decisions are not included).

OSR Decisions – Alarmed

Total number of OSR (Level 2) alarm decisions made by the operator for the ISD machine. This count is incremented and maintained in the upper level database based on messages received from the ISD (relayed through the PLC).

OSR Decisions –
% Alarm

The percentage of OSR Total Decisions that received an OSR (Level 2) alarm decision from that ISD machine.

 

Note that the percentage of OSR Clear Decisions and OSR Alarm Decisions may not sum to 100%. This is because this report does not include every type of decision (i.e. Other and Unknown decisions are not included).

Bags Tracked to CBRA

Total number of bags that were correctly tracked from the exit of the ISD machine to CBRA.

Tracking Accuracy %

Percentage of the total number of bags that exited the ISD machine and were not lost in tracking before entering CBRA.
