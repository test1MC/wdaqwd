Bag Tag Printer Report

The Bag Tag Printer Report provides statistics on the Bag Tag Printers used to send BSM data and print bag tags at the ticket counters for the BHS. This report requires the BSM hosts to send printer information (the .T field) or it will not be populated.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑8: Sample Bag Tag Printer Report

The following information is included in this report:

Tags that are seen only at an MEC, but not an ATR are assumed to have poor print quality. Printers that are particularly bad are highlighted in red on the report.

Row

Description

Bag Type

(L)ocal or (T)ransfer

Printer

The ID (as given by the Airline in the .T element of the BSM) of the printer that produced the tag.

Carrier

The Airline associated with the bag tag that needed to be printed.

Station

The station the tag was printed at. Note that for tags printed at other stations other than the local station, the Inbound flight information is used to determine the station. This may not be 100% accurate as it is possible that a bag tag was printed at another station prior to the direct Inbound station.

ATR Count

The total number of tags from the printer successfully scanned by any ATR.

MEC Count

The total number of tags from the printer scanned by any MEC.

Error Ratio

The number of tags from the printer scanned by any MEC over the number of tags successfully scanned by any ATR.
