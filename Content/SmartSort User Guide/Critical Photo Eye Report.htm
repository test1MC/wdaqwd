Critical Photo Eye Report

The Critical Photo Eye Report provides a time-based account of all bags that cross a critical photo eye. Critical photo eyes are located immediately upstream and downstream of the ISD machines and immediately upstream of the first BRP station in CBRA.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • BHS ID
  • Bag Tag

Figure 188: Sample Critical Photo Eye Report

The following information is included in this report:

Column

Description

Time

 Time the transaction occurred.

BHS ID

 Baggage Handling System identification number.
Bag Tag Baggage tag number.
Transaction Type Type of photo eye transaction event.
Device Device name where the transaction occurred.
Security Status Security status of the bag.
