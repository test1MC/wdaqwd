Pier Mode

The user can choose to encode a bag by pier (which will send a bag directly to the selected virtual pier).

A list of all virtual piers will be displayed on the screen. All virtual piers may not be displayed on the screen at once. However, the list of virtual piers can be filtered by pier name. The user can also browse through the list of piers. This table can be sorted similar to that of flight mode (see the description in section 5.2.3 Flight Mode).

Figure 5‑6: Manual Encoding Method - Pier Mode

To encode a bag using Pier Mode:

  1. Tap Pier. When pier mode is selected, the button will be light blue.
  1. Select the desired pier from the pier list. The piers that display can be filtered by name using the on-screen keyboard and keypad.
  1. Tap Release. Once released, the MEC will send the bag's destination to the PLC which will route the bag and the bag transaction data will be updated on the SAC.
