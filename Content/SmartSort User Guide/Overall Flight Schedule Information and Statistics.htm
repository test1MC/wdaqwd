Overall Flight Schedule Information and Statistics

Information about the flights currently displayed on the Active Flight Schedule displays beneath the Display Filters. The information available varies depending on which Flight Schedule view is displayed. For example, the number of pending flights, invalid flights, and deleted flights are only available in the Editor View and the Updated Time is only available in the List and Gantt view.

Figure 4‑23: Overall Flight Schedule Information and Statistics

Note that the information and statistics displayed is only for flights currently displayed on the Active Flight Schedule (the flight's Estimated Departure time is within the Time Range and the flight matches all of the filters that are applied).

The following statistical information is available:

  • Updated – Displays on the List and Gantt views only. Updated indicates the last time the flights being shown were refreshed from the server.
  • Total – Total number of flights.
  • Open – Total number of flights that are currently in "Open Time" (the time between a flight's open time and a flight's late time, see section 4.3.1 Flight Times for details).
  • Unassigned – Total number of flights that do not have an On Time Pier assignment. In order for a flight to be considered assigned, there must be at least one On Time Pier assignment that does not have any class or city criteria associated with it. For example, if a flight has a pier assignment but it is only for bags for one particular class, some bags for the flight will not have a pier assignment, so the flight is considered unassigned.
  • Held – Number of flights that have been individually placed On Hold. On Hold means that all bags for the flight will sort to the On Time Pier assignment regardless of time. For example, if a flight is On Hold and has a Missed Pier assignment and the bag is seen after the Missed Time, it will still sort to the On Time Pier. Note that this number does not reflect any flights placed on hold by clicking Hold All Flights.
  • Cancelled – Number of flights that have been cancelled.
  • Pending – Displays on the Editor view only. Indicates the number of flights that have been changed in the Editor view and not yet saved to the server. Pending changes will be saved to the server when Save All is clicked.
  • Invalid – Displays on the Editor view only. Indicates the number of flights that have been changed in the Editor view and need to be corrected before they will be saved to the server (these flights have an "Invalid" status). Note that if other changes are saved, any flights with an "Invalid" status will not be saved to the server.
  • Deleted – Displays on the Editor view only. Indicates the number of flights that have been marked for deletion and not yet deleted from the server. Flights will be deleted from the server when Save All is clicked.
