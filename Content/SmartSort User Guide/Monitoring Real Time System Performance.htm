Monitoring Real-Time System Performance

The HMI and the Web Client provide various methods for monitoring the real-time status of the BHS. These methods can be divided into three categories:

  • Monitoring Equipment Status – The HMI allows operators to monitor the status of various system devices in real-time, both in graphical and textual format. For more information, see section 7.1 Monitoring Equipment Status.
  • Monitoring Alarms – Many of the faults monitored by the system produce alarms. The HMI is the primary means of monitoring fault alarms. The Web Client also provides a number of methods of monitoring alarms. For more information, see section 7.2 Monitoring Alarms and Fault Modes.
  • Monitoring Communication Status – The Web Client and the HMI both provide means of monitoring the status of communications for the various services used in the SmartSort system. For more information, see section 7.3 Monitoring Communication Status.
