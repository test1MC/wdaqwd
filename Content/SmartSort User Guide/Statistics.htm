Statistics

The following statistics are available above the flights being displayed. Note that the statistics apply to all flights being returned and not to any additional filters applied.

  • Total – Total number of flights.
  • Unassigned – Total number of flights that do not have an On Time Pier assignment. In order for a flight to be considered assigned, there must be at least one On Time Pier assignment that does not have any class or city criteria associated with it.
  • Pending – Indicates the number of pending flights that have been changed and not yet saved to the server.
  • Invalid – Indicates the number of invalid flights that have been changed and need to be corrected before they will be saved to the server.
  • Deleted – Indicates the number of flights that have been marked for delete and not yet deleted from the server.
