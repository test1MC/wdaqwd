Pusher Pop-Up Window

The Pusher pop-up window contains:

  • Current Status of the Diverter
  • Cycle Counts and other statistics for the Diverter

Control functions (Out of Service/Push All/Push Half/Keyswitch Position Auto) are only available when a user with adequate security clearance is signed in. For more information, refer to section 8 Controlling the System Using the HMI. Note that for the standard User HMI account, the control functions will not be displayed.
