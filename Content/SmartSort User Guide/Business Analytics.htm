Business Analytics

The Business Analytics screen provides a list of the items created in the Report Editor. It allows users who are not able to, are not permitted to, or do not want to use the editor to view the components created in the editor by themselves or others.

To access this screen, click the Main Menu icon (), select Dashboards, then click Business Analytics .

Figure 9‑3: Business Analytics - List of Saved Dashboards and Reports

A list of accessible Interactive Reports, Analysis Reports, and Dashboards is displayed. This screen is divided into two main sections:

  • My Content – This section of the screen provides quick access to the Default item (if selected), favorited items, and recently viewed items.
  • Available Content – This section of the screen lists all available items. To add an item from this list to the Favorites list, select the Star icon beside the desired item.

Selecting an item from any list on the Business Analytics screen displays the item in its default view (Figure 9‑4: Business Analytics: Sample Item). The user can interact with the item to change the visualizations and filters to find information. The next time the item is accessed, it will return to its default view. Clicking List from this screen returns the user to the Business Analytics screen.

Figure 9‑4: Business Analytics: Sample Item

If a default item has been selected (report or dashboard), that item will be displayed automatically when Business Analytics is selected from the Dashboards menu. If a default item has not been selected, the Business Analytics screen (Figure 9‑3: Business Analytics - List of Saved Dashboards and Reports) will be displayed.

To set a default item, open the item from the Business Analytics screen and select the Default checkbox at the top of the screen.

Users can also set Business Analytics as their default home page instead of the Welcome screen by selecting Default Business Analytics View from the Default View drop-down list on the Account screen (see section 2.2.1 Managing User Preferences). If a default item has not been selected, but Business Analytics has been selected as the default page from the Account screen, then the list of saved dashboards and reports on the Business Analytics screen is displayed as the default page. If a default item is selected, and the user preferences are to view Business Analytics as the default page, then the default item is displayed as the default page.
