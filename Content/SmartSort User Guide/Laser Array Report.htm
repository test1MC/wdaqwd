Laser Array Report

The Laser Array Report provides a summary of bags that are seen at the ATR. Statistics are provided for both Heads (barcode counts) and for the Scanner (bag counts).

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

The following information is included in this report:

Note that there is a summary section containing the totals for each ATR following the specific head read rates for each individual ATR. Some values also allow for additional drill-down information (i.e. clicking on the Total Bags value takes you to the Bag Transaction Search).

Individual Scanner Array Heads section:

The individual scanner array section breaks down the following by read head:

Row

Description

Barcodes Read

The number of barcodes read by the particular read head.

Total Barcodes

The total number of barcodes read by all read heads.

% of Total

The % of total barcodes read by the particular read head. (Barcodes Read / Total Barcodes * 100%)

% Nominal Rate

The configured nominal read rate for the particular read head. This provides the operator something to compare the actual % percent read against.

Differential

The difference between the % Nominal Rate and the actual % of Total read.

Overall Statistics section:

The overall statistics section breaks down the following by individual ATR. It is divided into different sections for readability. Note that each row is broken down by actual value and percent of total. There is also a combined total which shows all ATR values together.

Read Rate section:

Row

Description

Total Bags

The number of bags which passed under the ATR.

Bags Read

The number of bags where at least one valid barcode was successfully read at the ATR.

Timeouts

The number of bags where no barcode information was received within the window of opportunity.

No Reads

The number of bags where the ATR was unable to successfully read any valid barcodes.

Read Rate (%)

The percent of total bags which were successfully read. (Bags Read / Total Bags * 100%)

Tag Status section:

Row

Description

Rejects

The total number of tags which were logically rejected by the system.

Conflict Tags

The number of bags where multiple conflicting tags was read. For instance two different valid and sortable IATA tags on the same bag.

Unassigned Flight

The number of tags read where the flight that the bag is destined for is not currently configured within the active sort schedule.

No Matching BSMs

The number of tags read where no BSM information could be found. This could indicate an issue with the host BSM connection.

Late Bags

The number of tags read from bags which are now considered late.

Valid Tags

The total number of sortable barcodes read.

Overall Tag Counts section:

Row

Description

Configured Tag Type

The total number of tags read for the configured tag type (i.e. IATA Tags, IATA Fallback Tags, 2-Digit Pier Tags, etc.).

Tag Counts by Bag section:

Row

Description

Configured Tag Type Category

The total number of tags read for the configured tag type category (i.e. Only IATA Tags, Only 2-Digit Pier Tags, IATA & 2-Digit, etc.).
