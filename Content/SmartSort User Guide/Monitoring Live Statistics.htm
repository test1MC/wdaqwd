Monitoring Live Statistics

The system monitors certain statistics in near-real-time such as the number of jams, number of faults, ATR read rates, etc. so that trends can be identified and any problems can be addressed. The primary methods of monitoring live system statistics are via the Business Analytics (see section 9.2 SmartSuite Business Analytics Report Editor) and the HMI Statistics screens.
