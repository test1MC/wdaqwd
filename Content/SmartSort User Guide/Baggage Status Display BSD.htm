Baggage Status Display (BSD)

Each Inspection Table has a BSD station. These stations allow users to select a bag that has stopped at the associated BRP queue and move it to the BIT. The user uses the BSD to transfer the bag data to the Secondary Viewing Screen (SVS) to see the x-ray image of the bag captured by the screening machines. The BSD screen displays prompts for required action from the user and/or messages providing information.

BSD stations also use Baggage Process Timers (BPTs) to record the processing time for every bag at each station. The BPT information is available from the SmartSuite Enterprise Web Client in the Baggage Process Timer Report (see section Bag Transit Time Report).
