Title/Navigation Bar

The Title/Navigation Bar contains:

  • User Name of the Currently Logged-In User
  • Workstation Name
  • HMI Server Status
  • Date/Time
  • Navigation Buttons/Menus

The following navigation buttons and menus are available in the Title/Navigation Bar:

  • Previous – Displays the previously-viewed screen.
  • Next – Displays the next screen (the screen viewed before the Previous button was clicked).
  • Home – Displays the HMI Screen Overviewshown in Figure 3‑1: HMI Screen Overview.
  • Graphics – Displays the Graphics menu(see section 3.1.2 Navigating HMI Graphical Screens).
  • Device Status – Displays the Device Status menu (see section 8.1 Device Status ).
  • Line Control – Allows the user to set lines in latch bypass and, in some cases, out of service.
  • Statistics – Displays the Statistics menu (see section 7.4.1 Monitoring Statistical Information in the HMI).
  • Alarm Summary – Displays the Alarms Summary screen (see section 7.2.2 Alarm Monitoring and History).
  • Connection Status – Displays the Connection Status screen (see section 7.3.1 Monitoring Communication Status with the HMI).
  • Utilities – Displays the Utilities menu.
  • FM – Displays the Fault Mode pop-up window (see section 7.2.3 Fault Mode Pop-Up Window).
  • SmartSuite Enterprise Web Client – Opens the SmartSuite Enterprise Web Client (see section 2 SmartSuite Enterprise Web Client Overview).

The Alarm Banner displays at the bottom of the HMI graphical display. In some systems, the Alarm Banner can be toggled on/off by clicking the Alarm icon () located in the Title/Navigation bar. This banner contains a list of currently active alarms with the following columns:

  • Alarm Date and Time
  • Acknowledgment Time
  • Alarm Severity
  • Tag Name in the HMI
  • Tag Description

Some typical alarms include:

  • Motor Overload
  • Jam
  • Motor Disconnect Off
  • Out of Service
  • E-Stop
  • VFD Fault
  • PLC Faulted
  • Encoder Fault

The Alarm Banner also contains several control buttons. These buttons operate as follows:

Control

Function
Ack Current Acknowledges the currently selected alarms. The alarm(s) will stop flashing but will remain in the list until the alarm is cleared by the PLC (i.e. it needs to be reset in the field, as in the case of a jam).
Ack Page Acknowledges the currently displayed page of alarms.
Ack All Acknowledges all alarms in the system.
Pr. Resolution Displays the Problem Resolution Text window (see section Problem Resolution Text).
