CBIS ISD (Level 1) Performance Report

The ISD Performance Report provides a summary of statistics related to the CBIS portion of the system. The faults, failures, and usage statistics related to the Inline Screening Devices (ISDs) are included in this report.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Interval (Minutes, Hours, Days, Weeks, Months, or Years)

Figure 9‑28: Sample CBIS ISD (Level 1) Performance Report

The following information is included in this report:

Note that there is a summary section containing the totals for each ISD, followed by a table showing the results for each individual ISD at the specified interval.

Column

Description

Clear Bags

Total number of machine (Level 1) clear decisions made by the EDS machine. This count is incremented and maintained in the upper level database based on messages received from the EDS (relayed through the PLC).

Cleared %

The percentage of Machine Total Bags that received a machine (Level 1) clear decision from that EDS machine.

Alarmed Bags

Total number of machine (Level 1) alarm decisions made by the EDS machine. This count is incremented and maintained in the upper level database based on messages received from the EDS (relayed through the PLC).

Alarmed %

The percentage of Machine Total Bags that received a machine (Level 1) alarm decision from that EDS machine.

Error Bags

Total number of machine (Level 1) error decisions made by the EDS machine. This count is incremented and maintained in the upper level database based on messages received from the EDS (relayed through the PLC).

Error %

The percentage of Machine Total Bags that received a machine (Level 1) error decision from that EDS machine.
Comm Error Bags Total number of Machine (Level 1) no decisions made by the EDS machine.
Comm Error % The percentage of Machine Total Decisions that received a machine (Level 1) no decision from that EDS machine.

OSR Decisions

Total number of bags that passed through that EDS machine that received their final security status from an OSR (Level 2) decision.

OSR Decisions %

The percentage of Machine Total Decisions that received their final security status from an OSR (Level 2 ) decision.

Total

The total number of machine (Level 1) decisions from that ISD machine.
ISD ID Unusable Total number of security decisions that were unusable because the bag was lost inside the ISD machine.
Late Decision Total number of security decisions that were unusable because they arrived after the final decision diverter.

Average Processing Time

The amount of time required on average for the ISD to process a bag and reach a level 1 decision. Measured in seconds.
