Problem BSM Report

The Problem BSM Report provides a list of all bags that could not be sorted automatically by the BHS, but whose bag tags were read at the ATR. Bags would be included in this report, for example, if there was a problem with the information in the BSM, if the flight was not in the schedule, or if there was no BSM for the bag.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 9‑14: Sample Problem BSM Report

The following information is included in this report:

The information included in this report comes from bag transactions and can potentially contain multiple records for the same bag.

Column

Description

Carrier

The name of the airline.

Barcode

The tag value.

Scan Time

When the bag was scanned at the ATR.

Location

The device that the bag was scanned at.

Reject Reason

Why the bag was rejected (i.e. No BSM Data, Bag City does not match Flight City, Too Many Sort Attempts, etc.).
