VMU and VSU Pop-Up Window

The VMU/VSU pop-up window contains:

  • Current Status of the VMU/VSU
  • Cycle Counts and other statistics for the VMU/VSU

Control functions (Override Up/Override Down/All Devices) are only available when a user with adequate security clearance is signed in. For more information, refer to section 8 Controlling the System Using the HMI. Note that for the standard User HMI account, the control functions will not be displayed.

Figure 99: VSU Pop-up Window

Clicking the VSU button from the Device Status menu opens a pop-up window that displays all the VSUs in the system and their statuses.
