Monitoring Communication Status

The statuses of communications for the various SmartSort services can be monitored from the HMI using the Connection Status screen (see section 7.3.1 Monitoring Communication Status with the HMI) or from the Web Client using the Current System Activity screen (see section 7.4.2 Monitoring Statistical Information with the Web Client).
