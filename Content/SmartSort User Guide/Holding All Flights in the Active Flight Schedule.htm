Holding All Flights in the Active Flight Schedule

The Hold All Flights function will place the entire Active Flight Schedule on hold. When a flight is on hold, all bags will be sorted to their flight's On Time Pier regardless of the time when the bags are seen in the system.

Clicking Hold All Flights will display the Hold All Flights confirmation dialog before flights are placed on hold. Note that even after flights are placed on hold, flights are immediately placed on hold in the Active Flight Schedule without having to click Save All. However, if there are any unsaved changes to the Active Flight Schedule on the Editor View those changes will still be unsaved when the flight hold is activated; the operation puts on hold the flights that have already been saved to the server.

Figure 4‑20: Hold All Flights Confirmation Dialog

When the schedule is on hold, an indicator in the area above the flight details will be displayed in all three views.

Figure 4‑21: Active Flight Schedule - Editor View All Flights On Hold

When the schedule is on hold, the Hold All Flights button in the Editor View changes to Remove Hold. Clicking Remove Hold will display a dialog which allows the flight hold to be removed for all bags. Once the hold is removed, normal time-based sortation is resumed.
