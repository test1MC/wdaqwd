Understanding Search Results

There are two types of searches available in SmartSort: Global Search and Advanced Searches. The Global Search functionality allows a user to search for any combination of flights, passengers and baggage (for more information, see section 10.1 Using Global Search). Advanced Searches involve entering specific criteria and then generating an ad-hoc report.

The following searches are available in the Web Client:

  • Bag Tag Search (see section 10.3.4 Bag Tag Search )
  • Bag Transactions Search (see section 10.3.5 Bag Transactions Search )
  • Raw Message Search (see section 10.3.6 Raw Message Search )
  • Activity Log Search (see section 10.3.7 Activity Log Search )
  • Audit Log Search (see section 10.3.8 Audit Log Search )
  • Alarm History Search (see section 10.3.3 Alarm History Search )
