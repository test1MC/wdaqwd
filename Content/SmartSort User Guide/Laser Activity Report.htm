Laser Array Activity Report

The Laser Array Report provides the read rates of bag and no read activity for the ATRs at configurable intervals.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Interval (minutes, hours, days, weeks, months, and years)

Figure 179: Sample Laser Array Activity Report

The following data is available on the report for each ATR:

Row

Description

Total Bags

The number of bags that passed under the ATR.

Bags Read

The number of bags where at least one valid barcode was successfully read at the ATR.

Read Rate (%)

The percent of total bags which were successfully read. (Bags Read /Total Bags * 100%).
