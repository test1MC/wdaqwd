Flights Pane

To search for a bag by flight, a user can use the Flights pane. Users can search by Carrier and Flight Number. When searching by Carrier, all bags associated with the selected Carrier will appear in the search results. To narrow the search results, users can add a Flight Number (numbers only) to locate all bags associated with that specific Flight Number.

Note: Bags that do not fit the search parameters will also be filtered out on the BHS Map through "Ghosting" (i.e. making irrelevant bags transparent). This allows users to easily locate all relevant bags within the system by making their appearance more prominent on the BHS Map.
