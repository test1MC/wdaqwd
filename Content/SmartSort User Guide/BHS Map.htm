BHS Map Dashboard

The SmartSuite Enterprise Web Client includes a BHS Map that provides users with a conceptual, three-dimensional view of the overall system. This representation offers both visual and contextual-based information designed to facilitate operational management of the system. Users can zoom in and out of the map and view multiple different layers and filters.

The BHS Map is not intended to replace the SmartVision HMI application, but rather, to complement it. The BHS Map Dashboard provides a high-level overview of the system's performance and how bags are flowing through the system.

The BHS Map is made up of segments. Each segment is made up of one or more conveyors/devices. Typically, segments start and end between divert and merge points as well as other devices such as ATRs, MECs, ISD, etc.

The conveyors and associated features are coded as follows:

  • Gray – Tracked
  • Hatched – Untracked
  • White – Informational
  • Dot – Decision Point (Point between two segments)

Because some segments are untracked, the BHS Map shows an approximation of where bags are in the system. The BHS Map Dashboard is approximately five seconds behind real-time. For more accurate information about BHS statistics and status, use the SmartVision HMI.
