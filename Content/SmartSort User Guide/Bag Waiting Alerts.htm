Bag Waiting Alerts

When a bag that requires manual intervention is inactive at a BRP for too long, the following alarms will occur in sequence:

  1. If no action is taken within 15 seconds , the associated BSD screen displays a "Bag waiting too long" message in red text.
  2. If no action is taken within 60 seconds , the associated BSD sounds an audible alarm until an action is taken for the bag.
