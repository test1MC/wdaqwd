Virtual to Physical Mapping Report

The Virtual to Physical Mapping Report provides lists of virtual to physical Make-Up Unit mappings in a convenient format for printing. Any special status conditions associated with Make-Up Units (Round-Robin, Re-Assigned) will be shown.

Figure 9‑24: Sample Virtual to Physical Mapping Report

The following information is included in this report:

This configuration can be modified via the Web Client (see section 4.1 Virtual to Physical Mapping).

Row

Description

Physical Pier

The configured physical destination.

Virtual Zone

The logical zone under which the virtual pier falls.

Virtual Pier

The configured virtual destination.

Status

If the virtual pier doesn't equal the physical pier the status will display as Reassigned.
