Bag Transaction Search

The Baggage Transaction Search screen allows the operator to search for records in the Bag Transaction table on the DHS. This table contains multiple records per bag, one record for every recorded event. Also note that it may contain many sets of records with the same bag tag (because of the roll-over of IATA bag tag numbers). Note that the time zone associated with the timestamp returned in the search results is indicated in the top right corner of the screen. Station-specific data is filtered to only display information the user has permission to view. For example, a user that does not have permission to access information related to any of the stations associated with a bag's itinerary will not be able to successfully query associated baggage transaction records.

The following search criteria is available if the applicable data is received from the data interfaces:

  • BHS ID (Is In / Not In) – Indicates the BHS ID associated with the returned bag.
  • Comment (Contains / Is / Is Not / Begins With / Ends With) – Indicates any system generated comments that pertain to the associated baggage transaction.
  • Destination (Is In / Not In) – Indicates the associated bag's final destination.
  • Device (Is In / Not In) – Indicates the device used to record the associated baggage transaction.
  • Device Type (Is In / Not In) – Indicates the device type of the device used to record the associated baggage transaction.
  • Event (Is In / Not In) – Indicates the event recorded for the baggage transaction.
  • Flight List (Is In / Not In) – Indicates the flight number associated with the returned baggage transaction.
  • Physical Pier (Is In / Not In) – Indicates the physical pier the associated bag was diverted to.
  • Reason (Is In / Not In) – Indicates the reasoning being a sort decision (e.g. Pier Full).
  • Reject Reason (Is In / Not In) – Indicates the reason for which the associated bag was rejected from sortation.
  • Security Bag ID (Is In / Not In) – Indicates the ID number given to the associated bag by the screening machine to track the bag as it is being screened.
  • Sort Method (Is In / Not In) – Indicates which method was used to sort the associated bag.
  • Tag Number (Is In / Not In) – Indicates the associated bag's tag number.
  • Timestamp (Current / Fixed Time Range) – Specifies the time range during which the baggage transaction was recorded.
  • Transaction Type (Is In / Not In) – Indicates the transaction type of the returned baggage transaction record (e.g. Arrived in CBRA, Bag Measured, Bag Sorted, etc.).
  • Virtual Pier (Is In / Not In) – Indicates the virtual pier destination associated with the returned baggage transaction record.
