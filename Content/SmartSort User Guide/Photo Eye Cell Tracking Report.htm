Photo Eye Cell Tracking Report

The Photo Eye Cell Tracking Report provides an overview of the performance of the CBIS photocells based on messages from bags being screened.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Subsystem Filters

Figure 187: Sample Photo Eye Cell Tracking Report

The following information is included in this report for Tracking Photo Eyes:

Column

Description

Unknown Bags

The number of bags that were tracked as Unknown.

Missing Bags

The number of bags that did not appear at a tracking photo eye that were expected to.

Deleted Bags

The number of deleted bags.
Missing Bag Jams The number of missing bag jams.
Total Bags The total number of bags.
