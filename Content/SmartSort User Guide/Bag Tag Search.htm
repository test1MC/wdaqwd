Bag Tag Search

The Bag Tag Search screen allows the operator to search for baggage records. This table contains one record per bag. However, it may contain many records with the same bag tag (because of the roll-over of IATA numbers). Multiple records of the same bag tag number display if BSMs were received for the associated bag at multiple stations. Note that the time zone associated with the timestamp returned in the search results is indicated in the top right corner of the screen. Station-specific data is filtered to only display information the user has permission to view. For example, a user that does not have permission to access information related to any of the stations associated with a bag's itinerary will not be able to successfully query associated baggage records. Bag tag search results are station-specific based on BSMs received; if a BSM is received from other stations for a specific tag number, these results will also display, provided the user has the appropriate permission.

The following search criteria is available if the applicable data is received from the data interfaces:

  • BHS ID (Is In / Not In) – Indicates the BHS ID associated with the returned bag tag.
  • BSM Received OR First Seen Timestamp OR Last Seen Timestamp (Current / Fixed Time Range) – Specifies the time range the system received a BSM for the returned bag tag.
  • BSM Status (Is In / Not In) – Indicates the status of the BSM associated with the returned bag tag (e.g. BSM Arrived Late, BSM Received, BSM Updated After Sort).
  • CBRA (Is In / Not In) – Indicates the CBRA associated with the returned bag tag.
  • CBRA Time (Current / Fixed Time Range) – Specifies the time the associated bag tag entered CBRA.
  • Creation Time (Current / Fixed Time Range) – Specifies the time the associated bag tag record was created.
  • Creation Type (Is In / Not In) – Indicates the bag record creation type associated with the bag tag.
  • Deleted Time (Current / Fixed Time Range) – Indicates the time the system received a BSM Delete from the airline for the associated bag tag.
  • Dimensioner (Is In / Not In) – Indicates the dimensioning device used to measure the associated bag tag entry.
  • Divert Confirmation (Current / Fixed Time Range) – Specifies the time the associated bag was diverted.
  • Exception Data (Is In / Not In) – Indicates the associated bag's exception data as provided in the BSM.
  • Final Destination (Is In / Not In) – Indicates the associated bag's final destination.
  • First Seen Device (Is In / Not In) – Indicates the first device in the system that scanned the associated bag tag.
  • First Seen Device Type (Is In / Not In) – Indicates the first type of device used to scan the associated bag tag.
  • First Seen Time (Current / Fixed Time Range) – Specifies the time the associated bag tag was first scanned.
  • Host Address (Is In / Not In) – Indicates the ID from which the BSM message was received.
  • Inbound (Is In / Not In) – Indicates whether or not the associated bag is inbound.
  • Inbound Date (Current / Fixed Time Range) – Specifies the time the associated bag is expected to arrive.
  • Inbound Flight List (Is In / Not In) – Indicates the inbound flight number associated with the returned bag tag record.
  • Inbound Origin (Is In / Not In) – Indicates the original station from which the associated bag travelled from.
  • Internal Airline Data (Is In / Not In) – Indicates the internal airline data for the associated bag as provided in the BSM.
  • Last Seen Device (Is In / Not In) – Indicates the last device known to have scanned the associated bag tag.
  • Last Seen Device Type (Is In / Not In) – Indicates the type of device associated with the last known scan of the bag tag.
  • Last Seen On (Current / Fixed Time Range) – Specifies the time the associated bag tag was last seen.
  • Last Update Time (Current / Fixed Time Range) – Specifies the time the associated bag tag record was last updated.
  • MEC (Is In / Not In) – Indicates the Manual Encode Station used to process the associated bag.
  • MEC Time (Current / Fixed Time Range) – Specifies the amount of time spent by the associated at a Manual Encode Station.
  • Message Reference (Is In / Not In) – Indicates the carrier or number referenced in the associated BSM.
  • Onward Flight Destination 3 (Is In / Not In) – Indicates the subsequent destination of the associated bag.
  • Onward Flight List 1, 2, and 3 (Is In / Not In) – Indicates the subsequent flight(s) of the associated bag.
  • Onward Transfer (Is In / Not In) – Indicates whether or not the associated bag will be transferred to another flight.
  • Outbound Date (Current / Fixed Time Range) – Specifies the time the associated bag is expected to travel from the current station.
  • Outbound Destination (Is In / Not In) – Indicates the next stop on the associated bag's itinerary.
  • Outbound Flight List (Is In / Not In) – Indicates the associated bag's outbound flight number.
  • PNR (Is In / Not In) – Indicates the associated bag's PNR number.
  • Passenger Name (Is In / Not In) – Indicates the name of the passenger who owns the associated bag.
  • Physical Pier (Is In / Not In) – Indicates the physical pier the associated bag is being diverted to.
  • Recircs (Is In / Not In) – Indicates the number of times the associated bag has been recirculated.
  • Reject Reason (Is In / Not In) – Indicates the reason for which the associated bag was rejected from sortation.
  • Scanner (Is In / Not In) – Indicates which scanner screened the associated bag.
  • Scanned Time (Current / Fixed Time Range) – Specifies the time the associated bag tag was last scanned by an ATR.
  • Screened By (Is In / Not In) – Indicates which EDS machine screened the associated bag.
  • Screening Time (Current / Fixed Time Range) – Specifies the time range during which the associated bag was scanned at an EDS machine.
  • Security Bag ID (Is In / Not In) – Indicates the ID number given to the associated bag by the screening machine to track the bag as it is being screened.
  • Security Level (Is In / Not In) – Indicates the security level associated with the returned bag tag.
  • Security Status (Is In / Not In) – Indicates the security status associated with the returned bag tag.
  • Sort Method (Is In / Not In) – Indicates which method was used to sort the associated bag.
  • Standby (Is In / Not In) – Indicates whether or not the associated bag is in standby.
  • Tag Number (Is In / Not In) – Indicates the associated bag's tag number.
  • Tag Printer (Is In / Not In) – Indicates the printer at which the associated bag tag was printed.
  • Time Status (Is In / Not In) – Indicates the associated bag's time status (i.e. Early, Late, Missed, On Time, or Unknown).
  • Transfer Type (Is In / Not In) – Indicates the transfer type of the associated bag (i.e. Interline, NonTransfer, Online, or Unknown).
  • Virtual Pier (Is In / Not In) – Indicates the virtual pier destination of the associated bag.
