Duplicating a Flight in the Active Flight Schedule

Clicking Duplicate will duplicate (copy) any selected rows and insert new rows into the grid. All of the flight details are copied when a flight is duplicated with the exception of the flight number (the Flight Number cell will be highlighted in red to indicate that a value needs to be entered). The status will be INVALID until the flight number is specified. When the flight number is updated, the status will be NEW, but changes are not saved to the server until Save All is clicked.
