Removing a Bag

Baggage Removal Point (BRP) terminals allow users to remove bags from the system so that they can be manually inspected for explosives at the associated BIT workstation.

Each BRP queue conveyor has a BRP terminal. These terminals allow users to select and hold a bag stopped at the associated queue for transfer to a BIT workstation. The BRP terminal displays messages prompting required action or providing information.
