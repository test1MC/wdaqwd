Editing the Flight Schedule using an Excel Worksheet

In cases where editing through the built-in editor is not sufficient, clicking Worksheet will download an Excel template that can be used to copy and paste data between Excel and the editor. It is important that the Worksheet be used as it has the correct cell formatting for copying and pasting between the two applications.

When copying data from Excel to the Editor view, select the whole row including the blank column A.

Note that the Worksheet button is displayed on all three views, but it will typically only pertain to a user working in the Editor view.
