Shifting Flight Times in the Flight Schedule Template

Clicking Time Shift displays the Time Shift dialog. The Time Shift function allows the user to shift flight departure times for any selected rows. If multiple flights are selected, the specific flights that are affected can be displayed/hidden by clicking Show All Affected Flights/Hide All Affected Flights.

Figure 4‑33: Time Shift Dialog

When OK is clicked, the grid will reflect the changes and the status of the flight(s) will become MODIFIED, but changes are not saved to the server until Save All is clicked.
