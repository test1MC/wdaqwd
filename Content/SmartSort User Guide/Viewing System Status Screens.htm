Viewing System Status Screens

The status of various system devices can also be viewed from the screens accessible in the Device Status menu. Each of these screens displays the status, in both color and textual format, of the applicable system devices.

If the user has sufficient privileges, then the screens on this menu will also contain control functions. For more information, see section 7.4.2.2 Current System Activity. A sample Status screen is shown below.
