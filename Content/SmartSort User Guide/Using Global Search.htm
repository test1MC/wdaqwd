Using Global Search

The Global Search allows the operator to search the SmartSuite system for flights, passengers, baggage, or any combination of those items from the same criteria screen. The operator can choose to search for data for today or a custom date range can be entered.

To access Global Search, click the Main Menu icon (), select Search, then select Global Search.

Figure 10‑1: SmartSort Menu - Search

To perform a Global Search in the SmartSuite Enterprise Web Client:

  1. Select Global Search from the Search menu. The Global Search screen displays.
  1. If searching for results for a range of dates, click Range to enter the appropriate start and end dates.
  1. Enter the search criteria in one of the following formats:
  • Bag:
  • 10-digit baggage tag number, or Carrier Code and six-digit bag tag number (e.g. 0220811053, LH811053)
  • Flight:
  • Carrier code and flight number (e.g. AC1234, QK5)
  • City Pair - two three-letter airport codes separated by a dash (e.g. YYZ-LAX)
  • Passenger:
  • Passenger's Last Name/First Name or Last Name (e.g. SMITH/JOHN, SMITH)
  • PNR Number - six-character PNR code (e.g. DLH8G8, MLYCBD, 324543)
  • Search Combinations:
  • Flight # and Passenger (Passenger Name or PNR) (e.g. ZZ123 SMITH)
  • Passenger (Passenger Name or PNR) and Bag (e.g. SMITH/JOHN 0123123456)
    1. A set of matches for each of the data types is returned to the user. From the set of results, the user has the ability to view the details for each search result by clicking the associated link. These links will open the associated Details screen (see section 10.2 Viewing Details Screens).
