Manually Encoding Bags

The Manual Encoding Console (MEC) is composed of a touch screen computer and a handheld scanner.

The MEC allows for manual sortation of bags which are not read at the ATR and for bags which are read properly by the ATR but must be manually handled for some other reason (e.g. Tag Conflict, No BSM, or Flight not in Schedule, lost in tracking, etc.).

The MEC provides a method to manually assign bags to sort destinations, and acts as a backup sorting machine if an ATR Scanner fails and/or if the servers fail.

The recent activity of the MEC displays in the on-screen action log. The action log will include details about when a bag is released to a sort destination. It also displays error messages, as appropriate.

The statistics related to the MEC can be viewed on the Manual Encode Console Report. For more information, please refer to section 9.21 Manual Encode Console Report.

Manual Encode Consoles have a specific location defined in the Baggage Handling System - usually a Line and Belt position - configurable in the Workstation Settings screen of the SmartSuite Enterprise Web Client. Modifying this setting will change the conveyors being controlled. In this way, Manual Encode Consoles are inherently interchangeable with other Manual Encode Consoles in the system. For more information on location settings, please refer to SmartSort System Administration Manual Chapter 2: SmartSuite Enterprise Web Client Administration.
