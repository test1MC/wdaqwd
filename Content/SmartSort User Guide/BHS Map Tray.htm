BHS Map Tray

The BHS Map Tray expands from the right side of the screen by clicking the button. The tray can be collapsed by click the button. The following sections are available:

  • Settings – Displays the Settings pane, allowing users to access settings for System Status, Bag Modes, KPIs, and Labels. For more information, see section 8.5.1.2.1.
  • KPIs – Displays the Key Performance Indicator pane, allowing users to access operational statistics. For more information, see section 8.5.1.2.2.
  • Bags – Displays the Bags pane, allowing users to view bag detail cards for pinned bags. For more information, see section 8.5.1.2.3.
  • Flights – Displays the Flights pane, allowing users to search for bags by airline and flight number. For more information, see section 8.5.1.2.4.
