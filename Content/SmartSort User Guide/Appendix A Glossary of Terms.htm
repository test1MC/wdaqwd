Appendix A: Glossary of Terms

Name

Description

AB

Allen-Bradley – An industrial parts manufacturer.

 

ACK

In an acknowledgement protocol, this refers to a positive response.

 

Airport Code

Each Airport has a unique 3-letter code.

 

Area

Area refers to a location or bag room within the Baggage System.

 

ASCII

American Standard Code for Information Interchange

 

ATR

Automatic Tag Reader – An electrical device that can read barcodes on a label and transmit the barcode text to a computer (also known as a scanner).

 

BAIT

Bag Automatic ID Transfer

 

Barcode

A series of printed lines that can be read by a scanner automatically.

 

BHS

Baggage Handling System.

 

BPM

Baggage Processed Message - Sent to the airlines by the BHS, containing information about bag movements and locations.

 

BSD

Baggage Status Display: a terminal/computer used by operators in CBRA.

 

BSM

Baggage Source Message - Sent to the BHS by the airlines, containing information about a bag suitable for sortation.

 

Carrier

Airlines are often referred to as carriers.

 

CBIS

Checked Baggage Inspection System.

 

CBRA

Checked Baggage Resolution Area. This is the screening room where TSA performs hand searches on unclear or unknown baggage. Earlier systems called this the ETD Area.

 

CIP

Common Industrial Protocol.

 

Controller

When used in conjunction with a conveyor system, this is the electrical device that is programmed to control the conveyors in a transport system.

 

Conveyor

General term for a mechanical device to transport products: can be roller, gravity, or belt.

 

CPU

Central Processing Unit.

 

CSV

Comma-Separate Values

 

Database

A tool used to store and access data in an efficient manner.

 

DHS

Data Historian Server.

 

Dimensioner

Baggage Dimensioning System that uses laser scanning heads to determine the size of a piece of baggage. Sometimes known as a BDS, BDD, or BMA.

 

Diverter

A mechanical device on a sort conveyor that shifts bags to chutes or adjacent conveyors.

 

Domain Controller

Server that responds to security authentication requests on a computer network (and manages that security configuration).

 

E-Stop

Emergency Stop

 

Ethernet

A high-speed communication link between computers.

 

FIDS

Flight Information Display System

 

Heartbeat

Part of a communication protocol that is used by both sides of a connection to detect loss of communication

 

HMI

Human Machine Interface – Used to display a graphical and textual overview of the mechanical system and allow operator control of certain equipment.

 

IATA

International Air Transport Association – An international trade-group representing airlines and airports.

 

IP

Internet Protocol. A scheme for assigning addresses to devices so that they can be discovered and accessed on an Ethernet network.

 

IPC

Inter-Process Communication

 

ISD

Inline Screening Device.

 

I/O

Input/Output.

 

LAN

Local Area Network.

 

LED

Light Emitting Diode.

 

Lower Level Control System

A term that collectively refers to the portion of the control system and is responsible for the direct handling of baggage. The Lower Level components typically include PLCs, MCPs, ATR, BDD, sensors, control stations and similar devices.

MCP

Motor Control Panel.

 

MEC

Manual Encode Console.

 

MIS

Management Information System.

 

NAK

In an acknowledgement protocol, this refers to a negative response.

 

OOG

Out of Gauge

 

OPC

OLE for Process Control. A communication protocol to allow computers to talk with industrial devices such as PLCs.

 

OSR

On-Screen Resolution. The second level of security screening where a human remotely inspects a machine-generated baggage image.

 

PAX

Industry term for Passenger or Passengers

 

PDF

Portable Document Format

 

PE

Photo Eye. A device that detects when a beam of light has been obscured. It is used to indicate the presence of an object on a conveying system.

 

Make-Up Unit

The physical location to which bags will be sorted.

 

PLC

Programmable Logic Controller.

 

Pusher

A mechanical/electrical device that physically redirects baggage from a mainline a different conveyor line.

 

RAID

Redundant Array of Independent Disks – A system of multiple hard drives for data storage that provides redundancy in case of failure and/or increased performance compared to single disk systems

 

RAM

Random Access Memory - efficient computer data storage used by active programs for operations, with minimal access time.

 

Round Robin

A method of distributing load evenly between two or more devices. In Round-Robin, the load is distributed to all of the available resources by cycling through them in sequence.

 

SAC

Sort Allocation Controller.

 

Scanner

An electrical device that can read barcodes on a label and transmit the barcode text to e.g. a computer.

 

SmartSort

The Sortation Management tool used to control sortation within the system and communication with host systems.

 

Socket

For TCP/IP communications, a socket is established between two computers. It is essentially a virtual channel which is opened for data to flow between devices.

 

TCP/IP

Transmission Control Protocol/Internet Protocol – A method of communication between computers over a network.

 

Ticket Counter

The location in the BHS where passengers check in and drop off their baggage, inside the terminal building.

 

TSA

Transportation Security Administration.

 

UPS

Uninterruptible Power Supply. Used to provide power to the computers during brief periods of power failure.

 

Upper Level Control System

A term that collectively refers to the portion of the control system and is primarily responsible for BHS data management and does not directly control mechanical equipment. The Upper Level components include HMI workstations, rack computers and associated software (SmartSort, Reports, etc.) and similar devices.

 

VFD

Variable Frequency Drive. An electrical control device that can finely vary the speed of a motor by controlling the electrical power supplied to it.

 

VM

Virtual Machine

 

VPN

Virtual Private Network.

 

XML

Extensible Markup Language
