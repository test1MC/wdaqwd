PLC Status

The PLC pairs communicate over ControlNet, and with the HMI Servers over Ethernet. Information about the PLC communications is shown in the PLC section.

Figure 7‑27: Connection Status Screen - PLC Status

The following lists the indicators and statistics shown above and gives a short description about them:

  • Name – Shows the name of the PLC pair.
  • PLC-A Status – Indicates the current status of each PLC in the PLC pair. A green indicator is displayed beside the active PLC. The inactive PLC will be yellow if it is in standby mode and red if an error is present.
  • PLC-B Status – Indicates the current status of each PLC in the PLC pair. A green indicator is displayed beside the active PLC. The inactive PLC will be yellow if it is in standby mode and red if an error is present.
