Carrier Default Pier Auto-Release

If a bag tag is scanned or manually keyed and a BSM does not exist for that bag tag, the system is normally configured to automatically release it to the carrier default pier for the bag, but it can be configured to prompt the user before automatically releasing the bag.

If the operator prompt is disabled, the bag will automatically release when a bag tag is scanned or when a bag tag is keyed and the bag is released by the operator.

If the operator prompt is enabled, the operator will be able to stop the bag from being released to the default pier. The bag can be encoded using one of the other encoding methods.

If the BSM does not exist, the bag is processed according to the Pseudo BSM settings (see section 5.2 Pseudo BSMs).
