SmartSuite Enterprise Web Client Overview

The SmartSuite Enterprise Web Client is a fully web-based application that provides the interface to the Management Information System (MIS). The Web Client provides the data management and reporting facilities required to operate and maintain the system. This application is available from any workstation that has IP connectivity to the Data Historian Server (DHS) via a web browser. The Web Client should be accessible from any modern web browser. Login security is used to control access to the Web Client and its various functions (see section 2.2 Login and Security).

The URL for the Web Client is http://DFW-D-DHSdhs/WebClient/.
