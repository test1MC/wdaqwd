Viewing Details Screens

Detail Screens provide additional information about bags, passengers, flights, and transactions.

The SmartSuite Enterprise Web Client provides access to the various details screens via Global Search, or by clicking links in various other screens (i.e. Flight Details Screen). Each type of Details screen is described below.

Each of the details screens contain links to other details screens (e.g. clicking a flight number on a Bag Details screen will display the Flight Details screen).

Bag Details Screen

The Bag Details screen displays detailed information about one specific bag tag, including information about the passenger as well as the flight itinerary. A time stamped history of changes to bag information and handling of the bag – including times and locations of various bag scans –is also displayed.

Figure 10‑3: Bag Details Screen

Flight Details Screen

The Flight Details screen displays details about one specific flight leg, including the status of the flight's departure and arrival. The history of changes to the flight information is also displayed (as received in FIDS messages).

Figure 10‑4: Flight Details Screen

Detailed Transactions Screen

The Detailed Transactions screen displays transactions for bag tags or flights. These detailed transactions screens are accessible via the Bag Details screen and the Flight Details screen.

Every history item on the details screen coincides with a detailed transaction. There are also often additional transactions shown on the Detailed Transaction screens.
