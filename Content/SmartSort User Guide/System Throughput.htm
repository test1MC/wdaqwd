System Throughput

Selecting Throughput from the Statistics menu of the navigation bar displays the Throughput Counts screen, which summarizes throughput information for some of the devices in the system.

Statistics are automatically reset every day at 02:00. Historical statistics can be accessed via the Web Client (see section 9.13 Bag Transit Time Report).
