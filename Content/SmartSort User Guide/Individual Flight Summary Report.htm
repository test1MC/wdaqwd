Individual Flight Summary Report

The Individual Flight Report displays information for specific flights in the system. The information is divided into two sections. The first section provides a summary of the flight information and counts of bags in the system. The second section provides information about each of the bags that was sorted to this flight.

The following selection criteria are available:

  • Flight Carrier and Number
  • Depart Date

Figure 9‑7: Sample Individual Flight Report

The following information is included in this report:

Flight Summary section:

Column

Description

Airline

The name of the flight airline.

Schedule

What schedule template the flight belongs to.

Depart Time

The scheduled departure time of the flight.

Route

The airport code destination(s) for the flight.

Gate

Which gate the flight departs from.

Aircraft

The equipment that the aircraft is flying.

Early Bags

The number of bags destined to the flight that were seen in the system early.

On-Time Bags

The number of bags destined to the flight that were seen in the system on time.

Late Bags

The number of bags destined to the flight that were seen in the system late.

Missed Bags

The number of bags destined to the flight that were seen in the system after the flight has departed.

Bags not Seen

Bags for the flight which BSMs were received for but were not seen by the system.

Bags in System

The number of bags for the flight that are still working their way through the system. This only includes bags which have not yet been sorted or handled by the MEC.

BSMs Received

The number of BSMs received for the flight.

Bag Details section:

Column

Description

Bag Tag

The bag barcode data.

Passenger Name

The name of the passenger that the bag belongs to.

Scanner

The last ATR or MEC associated with the bag.

Sort Pier

Where the bag was sorted to.

Time Status

Whether the bag was early, on time, late or missed.

BSM Received

The time when the BSM for this bag was received.

Scan Time

When the bag was last scanned at an ATR or MEC.

Sort Time

When the bag was sorted to a Make-Up Unit.
