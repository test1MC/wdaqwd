Host Interfaces

The Host Interfaces section displays the communication status of BSM, BPM, and FIDS messages being sent through the system. If the indicator is red, the connection with the airline host system is not available.

If no messages are flowing from the host, the connection status may turn Inactive. This does not necessarily mean there is a problem. If no messages are received for too long a time, the system will disconnect and reconnect from the host to try to wake the connection up.
