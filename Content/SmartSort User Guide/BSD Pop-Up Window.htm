BSD Pop-Up Window

The BSD pop-up window contains:

  • Current Status of the BSD
  • Print Enable Control (if applicable)

Control functions (Print Enable) are only available when a user with adequate security clearance is signed in. For more information, refer to section 8 Controlling the System Using the HMI. Note that for the standard User HMI account, the control functions will not be displayed.
