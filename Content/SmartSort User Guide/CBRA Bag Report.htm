CBRA Bag Report

The CBRA Bag Report provides details about each bag processed at a BSD in CBRA.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time

Figure 193: CBRA Bag Report

The following information is included in this report:

Column

Description
Start Time The bag processing start time.
End Time The bag processing end time.
Duration The amount of time it took to process the bag (in seconds).
IATA or Pseudo ID The bag ID.
BHS ID  
Bag Type The security status of the bag.
BSD Location The BSD Station ID.
