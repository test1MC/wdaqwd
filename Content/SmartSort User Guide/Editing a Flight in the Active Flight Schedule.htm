Editing a Flight in the Active Flight Schedule

Clicking Edit Flight will open the Edit Flight dialog, which will be open for whichever flight is associated with the currently selected cell. This is the same dialog that is available from the Gantt view (see section 4.3.2.4 Gantt View for details on using this dialog). Once the OK button is clicked, the grid will reflect the changes and the status of the flight will become MODIFIED, but changes are not saved to the server until the Save All button is clicked.
