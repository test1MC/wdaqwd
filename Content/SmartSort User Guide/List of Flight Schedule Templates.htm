List of Flight Schedule Templates

Multiple templates can be stored by the system to account for different scenarios and schedule changes. The following Template Management view will be displayed when the Flight Schedule Template loads initially:

Figure 4‑24: Flight Schedule Template - Template Management View

When first loaded, all templates in the system are displayed in the list.

Filter

The list can be filtered by typing text in the Filter field located in the top-left corner of the screen. The list will filter as text is being typed and will filter based on all content in the list that contains the value being typed.

Sorting

The list can be sorted by each of the columns by clicking on the column header.
