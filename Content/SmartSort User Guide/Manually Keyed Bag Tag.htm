Manually Keyed Bag Tag

If the bag tag barcode cannot be scanned using the hand scanner, the bag tag number can also be entered using the on-screen keypad. Any tag types used by the ATR are also supported at the MEC. The process used when keying in the data is the same as when a tag is scanned.

Due to the likelihood of mistakes when entering the numeric data, this sort method is not recommended for general use.

To manually key a bag tag:

  1. Tap Keyed Tag. When Keyed Tag Mode is selected, the button will be light blue.
  1. Use the on-screen key pad to enter the bag tag number.
  1. Tap Release. The bag is processed according to the flow described in Figure 5‑2: MEC Flow Diagram. The bag is processed according to the flow described in this diagram.
