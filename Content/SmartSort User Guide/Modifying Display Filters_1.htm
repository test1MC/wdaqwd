Modifying Display Filters

The flights that display on the Active Flight Schedule screen can be filtered based on several flight properties (which are described in section 4.3.4 Flight Properties).

There are three types of filters:

  • Drop-Down List – A single value can be selected from a drop-down list. The filters include Reroute and Pier Time.
  • Free Form Text – Allows the user to type any value (e.g. Flight #). The display will show any flights that contain the value entered for that property. The filters include Flight, Route, and Gate. For example, entering a "2" in the Flight# filter field will filter the flights so that only flight numbers containing a 2 are displayed.
  • Multi Value – Allows the user to select multiple values from a drop-down list (e.g. the user can specify to filter for Airlines ZZ and YY). Once a value is selected from the list, it is displayed in the filter box. Selecting an additional item from the list will add the value to the filter box. Each value in the filter box can be removed by clicking the "x" icon for the filter or by deleting the item using the keyboard. Multi-Value filters include Airline, Aircraft, Market, Zone, Pier, and Status.

When multiple filters are added, only flights that match all of the filters will be displayed. Note that for multi value filters, flights only need to match one value from the list of filters for that flight property. For example, specifying ZZ and YY for Airline and A1 for Gate will find all flights that have an Airline of either ZZ or YY and the flight's Gate contains A1. Note that the filters remain when switching between the three views.

To remove all filters specified, click Clear Filters.
