Upper-Level Alarms

The Upper- Level Alarms screen of the Web Client can be used to monitor computer system-level alarms.

Figure 7‑25: Sample Upper Level Alarms Screen

Alarms listed on the screen include a timestamp and a description of the fault that occurred.
