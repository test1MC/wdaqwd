Inquiry Mode

Inquiry mode allows the user to investigate where an encoding method would send the bag without releasing the bag from the MEC. In this mode, any encoding method can be used. The user encodes a bag as normal (releasing the bag, if required). However, instead of physically releasing the bag to a sort destination, the system displays a message showing the sort destination to which the bag would be sent.

For IATA barcodes with BSM data, the Inquiry action displays a dialog box containing any information that is known about the bag.

Enter Inquiry mode by tapping the Inquiry button. It will turn yellow to indicate an unusual mode is active.

Figure 5‑9: Manual Encoding Console - Inquiry Mode

To exit Inquiry mode, tap the Inquiry button again and the button will turn back to its normal blue color.
