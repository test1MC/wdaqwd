System Throughput Report

The System Throughput Report includes throughput statistics for conveyor system subsections as collected by the DHS.

This report can be generated in a tabular format. In the tabular format, there is one row for each interval during the time period selected. For example, if a report time period of 24 hours is selected using the Start Time and End Time parameters and a Report Interval of 30 minutes is selected, there will be 48 entries for throughput (one for each 30-minute period for the duration specified – 24 hours).

In graphical format, each interval is represented as a data point on the graph. Using the example above, there would be 48 data points on the graph (one for each 30-minute period during the 24-hour time period).

The time periods are restricted to keep the reports short enough for proper system operation. Reports with too many time periods on them will not generate in a timely manner. The restrictions prevent more than 250 time items to be shown on a single report. For example, a one-minute interval report can run for four hours, but a two-minute interval can be run for eight hours.

The following selection criteria are available:

  • Start Date and Time
  • End Date and Time
  • Interval (Minutes, Hours, Days, Weeks, Months, or Years)
  • Report Type
  • Subsystem Filters

The following information is included in this report:

Column

Description

Time Interval

A snapshot in time that the values were recorded within.

Device

The throughput count for the device in that time interval.

Total

A summation of all device counts for that time interval.
