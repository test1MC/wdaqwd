Bag Tag Search

To search for a bag by bag tag number, users can key in 8-character (e.g. XX123456 where XX is the airline code) or 10-digit bag tag numbers into the search bar. After keying in four digits, the search bar will attempt to auto-complete all matching tag numbers (see Figure 143). If the corresponding bag is on the map, users can click on the tag number which will open a Bag Details Card in the Bags pane (see 8.5.1.2.3 Bags Pane). If the bag Tag Number does not correspond to a tracked bag in the system, a link to Global Search displays (see Figure 144). For more information on Global Search, see section 10.1 Using Global Search.
